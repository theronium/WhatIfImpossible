# WhatIfImpossible

> 現代科学では実現不可能な思考実験を集めるサイト

「もし不可能なことが起こったら？」という問いを軸に、物理学・宇宙論・量子力学・哲学の限界を超えた思考実験を考察します。記事はMarkdownで管理し、GitHubで公開しています。

- 各記事のアイデアは著者が発案し、Claude（AI）との対話を通じて執筆・考察を深めています。

- 主な用途：SF設定の考証——独自の世界観に登場する概念や用語について、物理的・論理的な根拠をClaudeとの対話で検証・整理しています。
