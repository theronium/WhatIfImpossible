## 散逸構造——不均衡が局所均衡を生む

[wiim_109](../philosophy/wiim_109.md) では「僅かの偏りが増幅されて創発が起きる」という構造を論じた。しかしそれだけでは片面しか見えていない。不均衡は単に増幅されるだけでなく、ある閾値で**局所的な閉じた均衡構造**を生み出す。

これをプリゴジンは「散逸構造（dissipative structure）」と呼んだ。大域的な不均衡（エネルギー勾配・物質流）の中で、系が自発的に秩序だった局所構造を形成する現象である。

| 大域的不均衡 | 生まれる局所均衡構造 |
|------------|-------------------|
| 熱勾配（上下の温度差） | ベナール対流セル（六角形の安定した渦） |
| 化学反応物の濃度差 | ベロウソフ＝ジャボチンスキー反応の自発振動 |
| 太陽エネルギー勾配 | 細胞（ホメオスタシスによる内部均衡） |
| 重力崩壊（不均衡の極限） | 安定した惑星軌道 |

散逸構造はエネルギーを絶えず外部に捨てることで内部の秩序を維持する。均衡に「達する」のではなく、不均衡を消費しながら均衡を「演じ続ける」構造だ。

---

## フラクタル的な入れ子

散逸構造の最も深い特性は、ある階層の局所均衡が次の階層の「素材（粒子）」になることだ。

```
大域的不均衡
　↓ 散逸構造の形成
局所均衡（閉じた構造）
　↓ それ自体が次の階層の構成要素に
より高次の不均衡
　↓
さらに大きな局所均衡……
```

具体的には：原子（量子的均衡）→ 分子（化学的均衡）→ 細胞（生物的均衡）→ 個体 → 生態系 → 惑星系 — この各階層で同じパターンが繰り返される。

物理学の繰り込み群はこの構造を数学的に記述する手法のひとつだ。小さなスケールの揺らぎを「積分して消す」と、より大きなスケールの有効理論が現れる。階層ごとに均衡の形が変わりながら、同じ入れ子が続く。

## 均衡と不均衡——どちらが先か

この入れ子構造の源流を問うと、「均衡が先か、不均衡が先か」という問いに行き着く。

現代物理学の標準的な立場では宇宙は**低エントロピー（不均衡に近い状態）**から始まったとされる。しかし「以前」を問うと：

- **ペンローズの共形サイクリック宇宙論（CCC）**：宇宙の熱的死（最大エントロピー・均衡の極致）が共形変換によって次の宇宙のビッグバン（最小エントロピー・不均衡の始まり）と等価になる
- **ループ量子宇宙論のビッグバウンス**：収縮（均衡へ向かう運動）が反転して膨張（不均衡）になる。宇宙はこれを繰り返す
- **量子真空**：「何もない均衡」は量子力学的に存在しない。ゼロ点エネルギーにより真空自体が永続的に揺らいでおり、完全な静止は原理的に禁じられている

---

## 始まりのない波

これらを総合すると、均衡と不均衡は**どちらが先かを問えない、交互に現れる波**として捉えることが最も整合的かもしれない。

山と谷のどちらが先かを波に問うことが無意味であるように、均衡と不均衡は一つの永続する過程の二つの位相だ。均衡は次の不均衡の種（量子揺らぎ・熱揺らぎ）を内包し、不均衡は局所均衡を生み出すことで次の均衡の素材になる。

この見方において、wiim_109が論じた「完全な対称性からは創発が起きない」という命題は補完される：

> 完全な対称性は現実には存在できない——なぜなら均衡そのものが次の不均衡を内包しているからだ。創発は一度限りの出来事ではなく、均衡と不均衡が互いを生み出し続ける永続する構造の名前かもしれない。

## 均衡は山の頂上にある

ここで「均衡」の意味をもう一段深く見直す必要がある。

通常、均衡は「安定した終着点」として語られる。しかし物理学はこれを覆す。ヒッグス場のポテンシャルは「メキシカンハット型」であり、完全に対称な中心点（φ=0）は**局所的な極大**——つまり山の頂上だ。宇宙はこの不安定な頂上から転がり落ち、対称性の破れた縁（安定な真空）に落ち着いた。

```
        ●  ← 完全な均衡（山頂・不安定）
       /|\
      / | \
─────/  |  \─────
   谷   |   谷
   ↑         ↑
 対称性の破れた安定状態
```

**頂上にいる限り何も起きない。転がり落ちた瞬間に構造が生まれる。**

これは自発的対称性の破れ（spontaneous symmetry breaking）として素粒子物理学で定式化されている。超伝導・磁性・宇宙初期の相転移はすべてこの「山頂からの転落」として理解できる。

### 安定なものは全て「転がり落ちた先」にある

この視点で見ると、自然界に存在する「安定な状態」はすべて、より完全な均衡からの転落点だ。

| 状態 | 転落元の「均衡」 | 安定した「破れた状態」 |
|------|---------------|-------------------|
| 宇宙の物質 | 物質・反物質の完全対称 | 10億分の1の物質優位 |
| 原子の電子軌道 | 核からの均一な距離 | 離散的な軌道（量子化） |
| 生命のキラリティ | L型・D型アミノ酸の均等 | L型アミノ酸への完全偏り |
| 強磁性体 | 全スピン方向が等確率 | 特定方向への自発磁化 |

均衡を「安定」と呼ぶ習慣は実は逆だったのかもしれない。真の均衡（完全対称）は最も不安定であり、私たちが「安定」と呼ぶ状態はすべて破れた非対称性の上に立っている。

> 存在するとは、転がり落ちることだ。あらゆる構造・生命・意識は、完全な均衡という山頂から転げ落ちた先に生まれる。創発とはその転落の別名かもしれない。
