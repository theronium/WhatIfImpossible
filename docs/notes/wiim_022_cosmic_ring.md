## 概要

アンキロン（wiim_022）の「空間固着」は時空計量の絶対的な基準点を提供する。しかしアンキロンが点的な基準点であるのに対し、**慣性系の回転そのものを連続的に計測する**には別の仕組みが必要だ。

コズミックマイス（wiim_008）の菌糸ネットワークが太陽系規模に広がったとき、その菌糸は光を伝播できるほどの光学的均質性を持つ可能性がある。この菌糸を光路として利用したリングレーザージャイロスコープ（g237）は、サニャック効果（g236）によって恒星系規模の回転計測を可能にする——**アンキロン暦の「錨」に対し、慣性回転の「目盛り」を与える技術**だ。

---

## 構造

### 光路としての菌糸

コズミックマイス菌糸誘導通信（wiim_033）では、菌糸内部を化学・電気信号が伝播することが論じられている。光の伝播にはさらに高い透明性・均質性が必要だが、長期進化（クロノスフィア実験炉）を経た菌糸が光ファイバー的な構造を自己組織化する可能性は否定できない。

恒星系規模のリング光路を形成する場合、菌糸は惑星軌道に沿った巨大なループを構成する。光路長が天文単位（AU）スケールに達すると、リングレーザーの感度（サニャック効果による位相差）は地上装置の約10¹⁰倍以上に向上する。

### アンキロンとの役割分担

| 技術 | 役割 |
|------|------|
| アンキロン（wiim_022） | 空間内の固定点——2点間の距離を計量変化に抗して保持する |
| 菌糸リングレーザー | 慣性系の回転を連続的に計測——銀河公転・太陽系の固有運動を検出する |

アンキロンが「何が固定されているか」を定義し、リングレーザーが「どれだけ回転しているか」を計測する。2つを組み合わせることで計量測量ブランチの最終目標——宇宙航法座標系の絶対基準——が初めて完全に定義される。

## L4・L5実証型——既存インフラによる最初の恒星系規模リングレーザー

菌糸光路もアンキロン固定も必要としない、**最も早期に実現可能な恒星系規模リングレーザー**が、ラグランジュ点を利用した実証型だ。

クロノスフィア年表（chronosphere_timeline）では外部年70年以降にL5点へシード・チェンバー1号が設置されている。ラグランジュ点は重力と遠心力が釣り合う安定点であり、一度配置した機器が軌道維持コストなしに留まれる——これをリングレーザーの反射点として転用できる。

### 構成案

| 頂点 | 位置 | 備考 |
|------|------|------|
| L4 | 地球軌道上・60°先行 | 安定ラグランジュ点 |
| L5 | 地球軌道上・60°後行 | シード・チェンバー設置済み |
| 地球・月系 | L4-L5の中間 | 中継・検出器配置 |
| L3 | 太陽の反対側 | 不安定点だが中継点として利用可能 |

L4-L5間距離は約1AU（約1.5×10¹¹m）。これだけでリングレーザーの感度は地上装置の10¹⁰倍以上に達し、太陽系の銀河公転（約220 km/s）が生むサニャック偏差を十分検出できる。

### シード・チェンバーとの共用

L5点のシード・チェンバーはクロノスフィア実験炉として運用されており、外部への通信・補給インフラがすでに整備されている。この設備に高精度ミラーと光検出器を追加するだけでリングレーザーとして機能させられる——クロノスフィア研究の副産物として恒星系規模の慣性計測が手に入る。

逆に言えば、クロノスフィア実験中に**サニャック信号が意図せず観測される可能性**もある。「クロノスフィア効果の確認実験中に内部時間の方向依存性を発見した」という偶然の発見シナリオ（補遺wiim_002_rotation_principle）と同様の経緯で、慣性計測網の構想が生まれたとする世界観設定も成立する。

### 技術段階の位置づけ

```
L4・L5実証型（人工構造物）← 最初の実現段階
    ↓
菌糸リングレーザー（コズミックマイス光路）← 生命系ブランチが成熟後
    ↓
アンキロン固定型（絶対静止系基準）← 計量測量ブランチの最終段階
    ↓
複合型（菌糸光路＋アンキロン端点固定）← 究極形
```

## 計測できるもの

光路長を惑星軌道スケール（数AU）とした場合：

- **太陽系の銀河公転**（公転速度約220 km/s）による慣性系のサニャック偏差
- **太陽系固有運動**（局所銀河群に対して約370 km/s）の方向・速度
- **重力波通過時の計量歪み**——アンキロンで固定した光路に対し、重力波が計量を一時的に変動させるときのサニャック信号変化

特に重力波との組み合わせは、アンキロン固着と重力波の干渉を「計量の乱れ」として直接検出する手段になる。

## アンキロン固定型リングレーザー

菌糸リングレーザーの根本的な限界は、**光路を支える鏡・菌糸が物理的構造体に固定されている**点だ。物理構造体は惑星重力・太陽風・銀河公転の影響を受けて動く。どれだけ光路が長くても、鏡ごと回転している系の回転は検出できない。

これを解決するのが「アンキロン固定型リングレーザー」だ——**鏡をアンキロンに付着させ、光路を時空計量そのものに固定する**。アンキロンは物質の属性ではなく計量テンソルに結合しているため、銀河公転・宇宙背景放射に対する固有運動・慣性系の絶対回転をすべて検出できる。いわば「宇宙の絶対静止系」を基準にしたリングレーザーだ。

### 鏡のアンキロン付着問題

ここで核心的な設計課題が生まれる。アンキロンは計量テンソルに結合する架空粒子だが、**通常の物質（鏡）との結合機構が定義されていない**。

考えられる解決策は2つだ：

**案A：アンキロン−物質結合の定義拡張**
アンキロンが計量だけでなく物質の特定属性（たとえば結晶格子の格子定数、または特定の分子結合角）とも弱く結合できるとする設定を追加する。この場合、鏡の表面に「アンキロン親和性材料」（アンキロン結合層）を蒸着することで付着が実現する。ただしこれはアンキロンの物理設定の拡張であり、計量テンソルへの選択的結合という定義と整合する形で論じる必要がある。

**案B：アンキロン配置による空間固定点を鏡の「錨」として利用**
アンキロンを鏡の近傍に2点以上配置し、それらの固有距離を保持する性質を利用して鏡を間接的に空間固定する。鏡にアンキロンを直接付着させるのではなく、アンキロンが形成する固有距離場の中に鏡を拘束する構造だ。鏡は物理的にはアンキロンに触れていないが、アンキロン場の計量固定効果に囲まれることで実質的に空間固定される。

案Bは既存のアンキロン設定を拡張せずに実現できる点で設定整合性が高い。

### 菌糸型との比較

| | 菌糸リングレーザー | アンキロン固定型 |
|---|---|---|
| 光路の固定基準 | 物理構造体（菌糸） | 時空計量（アンキロン場） |
| 検出できる回転 | 慣性系に対する相対回転 | 絶対慣性系に対する回転 |
| 銀河公転の検出 | 可能（菌糸が銀河公転していなければ） | 原理上確実 |
| 構築の難易度 | 菌糸の光学特化進化が前提 | アンキロン付着機構が前提 |
| 計測精度 | 光路長に依存 | アンキロン場の安定性に依存 |

究極形は**菌糸をアンキロン固定した鏡の間に張り渡す複合型**だ——菌糸が光路の「中継・修復」を担い、アンキロンが光路端点の絶対固定を担う。光学的均質性と絶対基準の両方を同時に実現する。

## 限界と課題

**菌糸の光学特性**
菌糸が光を伝播するには細胞壁・細胞液の光学均質性が必要だ。現在知られる菌糸は光ファイバーとしての機能を持たず、これは長期進化でのみ実現できる特殊適応になる。

**光路の安定性**
惑星軌道スケールの光路は、惑星の重力摂動・太陽風・小天体衝突で常に乱される。ハイヴマインド（wiim_059）として機能するコズミックマイスが菌糸を能動的に再配置・修復しなければ、光路の幾何学的整合性は維持できない。

**サニャック効果の読み取り**
巨大リングでの干渉縞の読み取りには、光路全体にわたる位相の同期が必要だ。分散知性としてのコズミックマイスが位相整合した信号処理（wiim_059）を行えるなら、これは同じ菌糸体の別機能として自然に実現される——センサーと知性が同一の構造体という一体型設計だ。

