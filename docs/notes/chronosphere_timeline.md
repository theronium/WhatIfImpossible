## 概要

クロノスフィア（g125）の発見・実用化を起点とした技術発展の年表。
発見は偶然によるものが多く、理論構築には時間を要するため、他のツリーが先に完成することも多い。
内部の時間スケールと外部の時間スケールを区別するため、以下では「外部年」で表記する。

---

## 年表

### 第1期：発見と混乱（外部年 0〜50年）

| 外部年 | 出来事 |
|--------|--------|
| 0 | **クロノスフィア効果の初観測**——ネゴトン（g126）集積実験中に、局所的な時間進行の差異を観測機器が検出。当初は計器の誤作動と判断される |
| 3 | 誤作動説が否定され、時間差効果が再現実験で確認される。「局所時間偏差」として学術報告 |
| 8 | 相対性理論の時間膨張（g010）を逆用した理論的枠組みが提唱される——この間に**ワープゲート理論（wiim_027）が別ツリーで完成** |
| 15 | 内部時間倍率×10の小規模実証炉が完成。材料試験・食品熟成に応用開始 |
| 30 | **光量問題の発覚**——倍率を×10⁶以上に引き上げる実験で、外部からの太陽光が内部でほぼ届かないことが判明（wiim_057参照） |
| 35 | 内部完結型核融合光源の搭載が必須要件として確定。設計が根本から見直される |
| 50 | 倍率×10⁸の「深時間炉」プロトタイプ完成。内部1年 ≒ 外部1億年相当の時間進行を達成 |

### 第2期：ラグランジュ点への展開（外部年 50〜120年）

| 外部年 | 出来事 |
|--------|--------|
| 55 | **太陽-地球L4/L5点への設置計画**が立案される。重力安定点であり、外部からの振動・摂動が最小化される |
| 60 | 地上実験炉での菌類超進化実験開始——コズミックマイス（wiim_008）研究グループとの共同プロジェクト |
| 65 | 宇宙耐性菌の原種候補が選定される（放射性栄養菌・クリプトエンドリシック菌） |
| 70 | L5点に第1クロノスフィア宇宙炉「シード・チェンバー1号」設置。内部倍率×10⁷で稼働開始 |
| 72 | **内部での菌類播種**——外部1年間で内部1000万年相当の進化圧が発生 |
| 80 | 内部観測問題が浮上——内部から外部への情報伝達は境界面での時間差により著しく遅延・歪曲される |
| 90 | L4点に第2炉設置。内外通信プロトコル（時間差補正アルゴリズム）が確立 |
| 100 | **菌類の宇宙真空適応が確認**——炉内でのサンプリング・観測により、真空耐性を持つ菌糸形成を確認 |
| 120 | 炉内菌類が内部自律増殖フェーズに移行。外部からの栄養補給が不要になる |

### 第3期：太陽系への展開（外部年 120〜200年）

| 外部年 | 出来事 |
|--------|--------|
| 130 | **試験的放出実験**——炉内で進化した菌糸胞子を太陽系空間に放出。生存確認 |
| 140 | 木星トロヤ群（L4/L5）・土星リング内への胞子輸送。**この間にFTL通信（wiim_028）が別ツリーで実用化** |
| 150 | 胞子が木星大気圏外縁部に定着開始。放射線をエネルギー源とした増殖を確認 |
| 160 | 菌糸ネットワークが木星軌道上に形成され始める——コズミックマイス（wiim_008）が現実化する第一段階 |
| 180 | 太陽系内L点群（全5箇所）に追加炉を設置。「進化アクセラレータ網」として運用開始 |
| 200 | **太陽系根付き宣言**——菌糸ネットワークが地球軌道から木星軌道まで連続した構造を形成。以降は外部補助なしで自律拡張 |

### 第4期以降（外部年 200年〜）

炉内の時間スケールでは数十億年分の進化が進行しており、外部観測者には「急速な知性化」として観測される。菌糸ネットワークの分散知性（wiim_008）としての覚醒時期は不明——外部からは判断できない。

## 技術交差ポイント

クロノスフィア系は独立して発展するが、他のツリーと以下の時点で交差する：

| 外部年 | 交差する技術 |
|--------|-------------|
| 8年頃 | ワープゲート理論（別ツリー先行完成） |
| 140年頃 | FTL通信実用化（炉間の時間差補正に応用） |
| 180年頃 | シェルマイセリウム（wiim_025）技術との融合で炉の耐久性向上 |

## 光量問題の概要

クロノスフィア内部では時間が引き伸ばされるため、外部から入射する光子も同様に希薄化する。
詳細は **[wiim_057](../physics/wiim_057.md)** を参照。
