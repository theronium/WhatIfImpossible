## 概要

アンキロン（wiim_022）を兵器・防衛設備として運用する場合の設計論。計量固着力を「バリケードの強度」として捉え、突破・移動・除去の三段階で整理する。また「計量に作用する」ことと「物質に作用する」ことの区別が、封鎖兵器としての有効範囲を根本的に制限することを論じる。

---

## 計量への作用と物質への作用——アンキロンは船を止められるか

アンキロンは計量テンソルに結合する。しかし**計量と物質は同じものではない**。

一般相対論において物質の運動は測地線方程式で記述される——物質は曲がった時空の中で「最も直線的な経路（測地線）」を進む。アンキロンが計量を固定すると測地線も固定されるが、物質は固定された測地線に沿って自由に動ける。鉄道のレール（測地線）を固定しても、列車（物質）は止まらない。

通常の推進——ロケットエンジン・イオン推進——は推進剤を噴射して運動量を得る。これは計量を変化させる操作ではなく、物質間の運動量交換だ。アンキロンで固定された空間内でもロケットエンジンは正常に機能すると考えられる。

| 操作 | 計量変化を要するか | アンキロンの影響 |
|------|-----------------|---------------|
| 等速直線運動（慣性） | 不要 | **なし** |
| ロケット推進・機動 | 不要 | **なし** |
| ワープドライブ | 必要（計量操作そのもの） | **封鎖される** |
| 重力波通信・検知 | 必要（計量の振動） | **封鎖される** |
| アルクビエレ型推進全般 | 必要 | **封鎖される** |

アンキロン封鎖は**ワープ技術を持つ艦に対してのみ有効な罠**だ。通常推進しか持たない艦にはほぼ無効になる。

---

## 封鎖兵器としての有効範囲——ワープ艦専用の罠

アンキロンを「敵艦の行動を封じる兵器」として用いる構想（wiim_035）は、ワープ技術の普及度に有効性が依存する。

**ワープ可能な艦にとって：** 致命的だ。計量操作が封じられ脱出手段を失う。通常推進のみで逃げるしかなく、機動性が大幅に低下する。

**ワープを持たない通常艦にとって：** ほぼ無効だ。エンジンは動き、機動も可能で、重力波による通信・検知が使えなくなるだけだ。

この非対称性が生む逆説——**ワープ技術を持つ高度な文明ほど、アンキロン封鎖への脆弱性が高い。** 技術的優位が固有の弱点を生む。またワープに依存した通信インフラを持つ文明では、アンキロン展開だけで情報網を麻痺させることができる。

## バリケードとしての強度設計——突破・移動・除去

### 突破：強度を上回る力を加える

η_Aが有限である以上、十分なエネルギーを投入すれば計量を強制変動させてバリケードを突破できる。必要なエネルギーはη_Aと展開密度に比例する。専用の対粒子や装置は不要——これは純粋に「強度の比較」の問題だ。

| η_A の強度 | 突破に必要なもの |
|-----------|--------------|
| 低 | 通常のエキゾチック物質装備で十分 |
| 中 | カシミールフォージ（wiim_023）級の集中投射 |
| 高 | 複数フォージの同時展開・大規模作戦 |
| 無限大 | 理論上突破不可能——ただし自己矛盾が先に発生（wiim_022§2参照） |

### 移動（巻き上げ）：錨が場所を変えるだけ

突破はバリケードを「押しのける」だけだ。アンキロンは計量に固着する性質を持つため、強制的に計量が変動しても、アンキロン自体は消滅せず**新しい計量状態に再固着する**。

```
突破前：アンキロンが計量Aを固定
    ↓（エキゾチック物質で強制変動）
突破後：アンキロンが計量Bを固定
```

錨を海底から引き剥がしても錨は消えない——別の場所に落ちて再び固着する。突破した領域には「計量の傷跡」が残り、アンキロンが乱れた計量を固定し続ける。ワープ航行への干渉は続き、完全な解決にはならない。

### 除去：反アンキロンによる対消滅

アンキロンを真に消滅させるには、対応する**反アンキロン**との対消滅が最も自然な解だ。

量子場理論ではすべての粒子に反粒子が存在する——アンキロンを粒子として定義した時点で、反アンキロンはその帰結として「存在してよい」ことになる。新たな概念を追加するのではなく、定義の論理的な含意だ。

> アンキロン＋反アンキロン → 対消滅 → 重力波バースト

アンキロンは計量に結合しているため、対消滅エネルギーは計量の振動——すなわち重力波として放出されると考えられる。

**この帰結が生む制約：**
- 除去は静かに行えない。重力波バーストが発生するため敵に検知される
- グラビトーペイク（wiim_010）装甲内でアンキロン除去を行うと、放出された重力波が内部に閉じ込められる（wiim_035の逆説が再発する）
- 反アンキロン単独では「計量を不安定化させる」逆方向の作用を持つ可能性がある——攻撃的な転用の余地だ
