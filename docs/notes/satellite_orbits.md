# 人工衛星の特殊軌道——太陽同期・モルニア・ハロー軌道と軌道選択の論理

LEO・MEO・GEO・HEOの基本分類は [地球周回軌道](../../glossary/terms/g077.md)（g077）を参照。本ノートはその上に「なぜその軌道形状・傾斜角でなければならないか」という設計論理と、特殊な用途のために生まれた名前付き軌道を補足する。

---

## 傾斜角による分類

高度だけでなく**軌道傾斜角**（赤道面に対する軌道面の角度）が軌道の性質を大きく左右する。

| 傾斜角 | 種類 | 特徴 |
|--------|------|------|
| 0° | 赤道軌道 | 常に赤道上空を通過。静止軌道（GEO）は傾斜角0°の特殊ケース |
| 0°超〜90°未満 | 傾斜軌道 | GPS・ほとんどの有人ミッション |
| 約63.4° | 臨界傾斜角 | 地球扁平率による近地点回転がゼロになる特別な角度（モルニア軌道が利用） |
| 90° | 極軌道 | 全球をカバーできる。地球観測・気象衛星 |
| 90°超〜180° | 逆行軌道 | 西向きに周回。太陽同期軌道はわずかに逆行方向 |

---

## 太陽同期軌道（SSO）

地球の扁平率（赤道が膨らんでいること）は、軌道面を少しずつ東向きに回転させる摂動（J2項）を生む。通常これは「ずれ」として補正が必要だが、太陽同期軌道では**この歳差を意図的に利用**し、軌道面の回転速度を地球の公転速度（1日0.9856°）に一致させる（→ [太陽同期軌道](../../glossary/terms/g358.md)）。

$$
\dot{\Omega}_{\rm SSO} \approx -\frac{3}{2} \frac{R_E^2\, J_2}{p^2} n \cos i = +0.9856°/{\rm day}
$$

（$p$：軌道の半直弦、$n$：平均運動、$i$：傾斜角）

この条件を満たす傾斜角は高度によって決まる。

| 高度 | 必要傾斜角 |
|------|-----------|
| 400 km | 約 97.0° |
| 600 km | 約 97.8° |
| 800 km | 約 98.6° |
| 1,000 km | 約 99.5° |

**効果**：軌道面が太陽方向と常に同じ角度を保つため、衛星が特定の地点を通過するときの地方太陽時が一定になる。毎回同じ照射条件で地表を撮影でき、季節・年をまたぐ比較観測に優れる。

**代表例**：ランドサット・SPOT・だいち（ALOS）・Sentinel系、気象衛星NOAA・MetOp。

## モルニア軌道

静止軌道は北緯60°以上では仰角が低くなりすぎて実用にならない。ソ連はシベリア・北極圏をカバーするため、高楕円軌道を採用した（→ [モルニア軌道](../../glossary/terms/g357.md)）。

**設計の核心——臨界傾斜角 63.4°**

楕円軌道では通常、地球の扁平率によって近地点の位置が軌道面内を回転し続ける（近地点歳差）。ところが傾斜角が約63.4°（または116.6°）のとき、この歳差がちょうどゼロになる。これにより近地点を南半球の低高度に固定したまま、遠地点を北高緯度上空に安定して維持できる。

```
        遠地点（北緯60〜70°上空・約4万km）
             ★ 〜8時間滞在
            /
           /  楕円軌道
          /
 近地点（約500km）
```

**3機で北半球をほぼ24時間カバー**できるため、1機が遠地点に近い高滞在ゾーンにいる間に次の機が上昇してくる体制を組める。

| パラメータ | 値 |
|-----------|-----|
| 近地点高度 | 約 500 km |
| 遠地点高度 | 約 39,900 km |
| 傾斜角 | 63.4° |
| 周期 | 約 12時間 |
| 遠地点滞在時間 | 約 8時間 |

## ハロー軌道・リサージュ軌道

[ラグランジュポイント](../../glossary/terms/g049.md)（g049）のL1・L2は不安定な均衡点であり、そのまま静止することはできない。しかし少量の推力で軌道を維持する「ハロー軌道」や「リサージュ軌道」を描かせることができる。

| 軌道タイプ | 特徴 | 代表例 |
|-----------|------|--------|
| ハロー軌道 | L点周りの周期的な3次元ループ | SOHO（L1）、初代JWST候補 |
| リサージュ軌道 | L点周りのX-Y面で非周期的なループ | JWST・プランク・ハーシェル（L2） |
| ディープ・スペース・ゲートウェイ候補 | 月のL4・L5近傍 | 将来の有人拠点構想 |

L2は地球の影に入らず太陽・地球・月から等距離に近い安定した熱環境が得られるため、宇宙望遠鏡の配置場所として好まれる。推力維持コストは少ないが完全にゼロではなく、燃料切れが寿命を決める。

---

## 凍結軌道（フリーズ軌道）

地球は完全な球ではなく、極が若干つぶれ赤道が膨らんでいる（J2項）。さらに南北非対称の高次項（J3）も存在する。これらの摂動を組み合わせると、特定の高度・離心率の組み合わせで**近地点の高度が長期間一定に保たれる軌道**（凍結軌道）が存在する。

地球観測衛星では地表との距離が変化しないことが重要なため、凍結軌道条件を満たす設計が選ばれることが多い。ERS・Envisat・Sentinel-6などが採用している。

---

## アナレンマとグラウンドトラック

静止軌道（GEO）の衛星は地上からほぼ静止して見えるが、**厳密に静止できない**2つのケースでは空に軌跡を描く。

1. **軌道傾斜角が残っている場合**：衛星は1日1回、南北に振れる8の字を描く
2. **軌道が楕円の場合（離心率あり）**：均時差と同じ理由で東西方向に振れる

この2つが重なると、地上から見た軌跡が [アナレンマ](../../glossary/terms/g353.md)（g353）と同じ8の字形になる。太陽のアナレンマが「地軸の傾き＋地球軌道の離心率」で生じるのと構造的に同一であり、衛星設計では傾斜角・離心率の両方を管理してアナレンマの大きさを最小化する（位置保持：station-keeping）。

## 軌道選択の論理まとめ

| 目的 | 適切な軌道 | 理由 |
|------|-----------|------|
| 全球カバレッジ | 極軌道 LEO コンステレーション | 傾斜角90°で全緯度を通過 |
| 特定地域の高仰角・長時間カバー（高緯度） | モルニア軌道 | GEOは高緯度で仰角が低すぎる |
| 安定した照射条件で地球観測 | 太陽同期軌道 | 毎回同じ地方太陽時に通過 |
| 全球通信・放送・気象（低〜中緯度） | GEO（静止軌道） | 地上アンテナ固定で運用可能 |
| 深宇宙観測・安定した熱環境 | L2ハロー/リサージュ軌道 | 太陽・地球・月の遮蔽なし |
| 測位（GPS等） | MEO 傾斜軌道 | 複数機で全球を立体的にカバー |
