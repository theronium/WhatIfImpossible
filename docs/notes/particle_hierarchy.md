## 概要

標準模型における素粒子と、クォークが束縛してできる複合粒子（ハドロン）の分類をまとめた参照ノート。

---

## 1. 標準模型の素粒子（構成要素）

素粒子はスピンによって **フェルミオン**（半整数スピン）と **ボソン**（整数スピン）に二分される（→ g064）。

### フェルミオン — 物質を構成する粒子

#### クォーク（スピン 1/2）
6種類あり、「世代」によって3世代に分かれる。単独では存在できず（クォーク閉じ込め）、常に複数が束縛してハドロンを形成する。

| 世代 | アップ型 | ダウン型 |
|------|---------|---------|
| 第1世代 | アップ（u） | ダウン（d） |
| 第2世代 | チャーム（c） | ストレンジ（s） |
| 第3世代 | トップ（t） | ボトム（b） |

u・d クォークが陽子・中性子の主成分。t クォークは最も重く、発見（1995年）は第3世代クォークの最後だった。

#### レプトン（スピン 1/2）
ハドロンを作らず、電磁力・弱い力のみに感応する。6種類。

| 世代 | 荷電レプトン | ニュートリノ |
|------|------------|------------|
| 第1世代 | 電子（e⁻） | 電子ニュートリノ（νₑ） |
| 第2世代 | ミュオン（μ⁻） | ミュオンニュートリノ（νμ） |
| 第3世代 | タウ（τ⁻） | タウニュートリノ（ντ） |

ニュートリノは電荷ゼロで質量が極めて小さく、ほぼ何にも反応せずに通過する。

---

### ボソン — 力を伝える粒子

#### ゲージボソン（スピン 1）
4種の基本力のうち3つを媒介する。

| 粒子 | 媒介する力 | 質量 |
|------|-----------|------|
| 光子（γ） | 電磁力 | ゼロ |
| グルーオン（g） | 強い力（色力） | ゼロ（8種） |
| W⁺・W⁻ボソン | 弱い力（荷電カレント） | 約80 GeV |
| Zボソン | 弱い力（中性カレント） | 約91 GeV |

重力を媒介する「重力子（グラビトン）」は理論上スピン2のボソンとして予測されているが、未観測（標準模型の外）。

#### スカラーボソン（スピン 0）

| 粒子 | 役割 |
|------|------|
| ヒッグス粒子（H⁰）（→ g070） | 他の粒子に質量を与えるヒッグス機構の担い手。2012年LHCで発見 |

---

## 2. 複合粒子（ハドロン）

クォークが強い力（グルーオン交換）によって束縛した粒子の総称。詳細は g114 を参照。

```
クォーク（u/d/c/s/t/b）
└─ ハドロン
     ├─ バリオン（クォーク × 3）        ← フェルミオン（スピン 1/2 または 3/2）
     │    ├─ 陽子／プロトン（uud）  電荷+1・安定・唯一の安定バリオン
     │    ├─ 中性子／ニュートロン（udd）  電荷0・原子核内で安定、単独では約15分で崩壊
     │    ├─ ハイペロン（Σ・Λ・Ξ・Ω など）  ストレンジクォーク含む・不安定
     │    └─ デルタ粒子（Δ、スピン 3/2）  陽子・中性子の励起状態に相当・不安定
     └─ メソン（クォーク ＋ 反クォーク） ← ボソン（スピン 0 または 1）
          ├─ パイオン（π）  核力を媒介・最も軽いメソン
          ├─ ケーオン（K）  ストレンジクォーク含む
          └─ その他（Dメソン・Bメソンなど、不安定・すぐ崩壊）
```

**陽子（プロトン）** は電荷 +1 を持ち、単独で無限に安定して存在できる唯一のバリオンである。陽子1個のみが原子核となったものが水素イオン（H⁺）であり、「陽子ビーム＝水素の核」として加速器実験でも扱われる。

**中性子（ニュートロン）** は電荷ゼロで、原子核内では陽子との核力（パイオン交換）により安定するが、単独では弱い力によりベータ崩壊（n → p + e⁻ + ν̄ₑ）し、半減期は約10分11秒。

陽子と中性子をまとめて**核子**と呼ぶ。核子が集まって原子核を形成し、周囲を電子が取り巻いたものが原子となる（→ g114）。

## 3. ボソン／フェルミオンの横断整理

「ボソン／フェルミオン」はスピンで決まるため、素粒子だけでなく複合粒子にも適用される（g064）。

| 粒子 | 構成 | スピン | 統計 |
|------|------|--------|------|
| 陽子 | uud（クォーク×3） | 1/2 | **フェルミオン** |
| 中性子 | udd（クォーク×3） | 1/2 | **フェルミオン** |
| デルタ粒子（Δ⁺⁺） | uuu | 3/2 | **フェルミオン** |
| パイオン（π⁰） | uū | 0 | **ボソン** |
| ρメソン | uū | 1 | **ボソン** |
| ヘリウム-4原子核 | 陽子×2＋中性子×2（原子核・ハドロン外） | 0 | **ボソン** |

> ヘリウム-4が超流動を示すのは、**原子全体**（核＋電子2個）のスピンが整数（0）になりボソンとして振る舞うためである。ヘリウム-4原子核自体もスピン0のボソンだが、超流動はあくまで原子単位の現象である。

パウリの排他原理（g060）はフェルミオンにのみ適用され、電子の軌道配置・核子の詰め込みに上限を設ける。ボソンにはその制約がなく、ボース＝アインシュタイン凝縮が起こりうる。

---

## 4. 反粒子について

すべての素粒子には対応する**反粒子**が存在し、質量は同じで電荷等の量子数が反転する。電子の反粒子は陽電子（e⁺）、u クォークの反粒子は反 u クォーク（ū）。光子・Z ボソン・（仮説上の）マヨラナニュートリノは自己反粒子（粒子＝反粒子）。

物質と反物質が衝突すると対消滅し、光子（ガンマ線）などに変換される。宇宙が物質優勢である理由（バリオン非対称性）は未解決問題。
