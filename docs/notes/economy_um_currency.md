## 概要

WhatIfImpossible世界における通貨制度と、エキゾチック物質の流通・単価の設計メモ。連邦発行のエネルギー兌換通貨を基軸とし、地方通貨・現物決済を組み合わせた三層構造を採用する。

---

## UM通貨：設計思想

### エネルギー本位制

通貨の裏付けを**エネルギー生産量**とする。金本位との違いは、裏付け資産（エネルギー）が文明の発展とともに増加し続ける点だ。これにより：

- エネルギー生産が拡大する時期 → インフレ圧力
- 生産が停滞する時期 → 自動的に緊縮
- 大型エネルギー施設（ダイソン球など）の完成 → 急激なデフレリスク

### カルダシェフスケール基準

```
1 F-UM = Type I 文明の1秒分エネルギー出力 ÷ 10¹⁰
       = 10¹⁶ J ÷ 10¹⁰
       = 10⁶ J（≒ 1 MJ）
```

Type I（惑星規模）を基準にすることで、地方通貨との階層的な対応関係が生まれる。

### 二層構造

| 通貨 | 正式名 | 性格 | 主な用途 |
|------|--------|------|---------|
| **E-UM** | Energy UM | 硬貨：エネルギー1:1兌換、発行制限あり | 大型取引・星間決済・エキゾチック物質購入 |
| **F-UM** | Fed UM | 軟貨：中央銀行が量を調整、微インフレ設計 | 日常流通・惑星内取引 |

地方通貨はF-UMまたはE-UMに連動して発行される。辺境コロニーでは事実上のエキゾチック物質現物決済が主流になる。

## エキゾチック物質の単位定義

通常の質量・モルではなく**効果量**を基準とする。

| 物質 | 単位名 | 定義 |
|------|--------|------|
| エキゾチック物質 | グラム（g） | 物質なのでSI単位で管理 |
| コーラ粒子（wiim_013） | **1コル（Cor）** | 1 GJを余剰次元に排出できる量 |
| ネゴトン（wiim_010） | グラム（g） | 格子素材として質量で管理 |
| アンキロン（wiim_022） | **1アンク（Ank）** | 1 km³の計量を1時間固定できる量 |
| 反アンキロン | 1アンク相当 | アンキロンと対消滅できる量 |

| 物質 | 単位 | 市場価格 | 備考 |

| エキゾチック物質 | 1g | 1〜5 MUM | カシミールフォージ（wiim_023）で生産 |

| ネゴトン | 1g | 3〜15 MUM | 精密製造、グラビトーペイク（wiim_010）用途 |
| アンキロン | 1アンク | 50〜300 MUM | 時空操作が必要、高希少 |
| 反アンキロン | 1アンク相当 | 500 MUM〜2 GUM | 軍事管理品、闇市場価格は別 |

### 希少性序列

```
エキゾチック物質（汎用）
    ↓
コーラ粒子（重力勾配付近で収集可能）
    ↓
ネゴトン（精密な負質量制御が必要）
    ↓
アンキロン（時空計量への結合が必要）
    ↓ 最希少
反アンキロン（軍事管理品）
```

## ライセンス制度

民間市場だが取得・保有にライセンスが必要。軍も同じ免許保持者を内部に抱える構造。

```
連邦エキゾチック物質管理局
│
├── 軍事ライセンス（最高位）
│     艦隊・惑星防衛・特務機関
│
├── 産業ライセンス
│     恒星開発・ワープゲート建設・大規模採掘
│
├── 探索・開拓ライセンス（取得要件が比較的緩い）
│     フロンティアの現実に規制が追いついていない
│
└── 研究ライセンス
      大学・研究機関、少量使用のみ
```

| 区分 | 年額 |
|------|------|
| 研究ライセンス | 50〜500 kUM |
| 探索・開拓ライセンス | 1〜10 MUM |
| 産業ライセンス | 100〜1,000 MUM |
| 軍事（予算内包） | — |

## 派生する経済的緊張

- **研究者**：大学予算でアンキロン少量なら買える。ただし1回の実験でほぼ年間研究費が飛ぶ
- **探索者**：ワープ装備は数年分の収入。破損したら廃業水準の損失
- **闇市場**：反アンキロンは正規市場で購入不可のため、軍の横流しや密造品が流通する
- **辺境採集**：コーラ粒子は重力勾配付近で自己採集できるため、ライセンスなしの密採が後を絶たない
- **ライセンスブローカー**：免許自体が売買される。探索者が軍の下請け扱いで免許を「借りる」慣行がある
