## 概要

宇宙規模の文明における政治体制の整理。連続した領土を前提とする地上の国家概念は機能しないため、航路・ゲート・資源点を軸にした別の形態が自然発生する。ステラリス等のSF作品で描かれる枠組みと大きく重なるため、WIIM独自の記事化は行わない。

---

## 基本形態

| 形態 | 説明 |
|------|------|
| ユニオン・同盟 | 共通利益（航路・FTL規格・通貨）で結束した自発的連合 |
| コンソーシアム | ゲート網や資源を管理する企業的連合体 |
| 帝国 | ゲート独占・軍事力による覇権国家。「帝国」と呼称される |
| 思想・宗教連合 | 価値観・法体系の共有による緩やかな統合 |

## ゲート制覇と帝国

ワープゲート網の所有・管理が事実上の覇権を決定する。宙域資産権（g154）の登記機関が中立的条約組織として発足しても、ゲート帝国の官僚機構に変質するリスクがある。

## 関連用語

- 宙域資産権（g154）— 空域の所有権概念。管轄機関の問題と接続する
