## 概要

エキゾチック物理学（g350）が対象とする粒子・物質および派生技術の一覧。「通常物理と符号が逆転した物理法則」を持つもの（符号逆転系）と、通常物理の枠組み自体が適用されないもの（構造的異常系）に分けて整理する。

---

## WIIM独自粒子・物質

### 符号逆転系

物理量（質量・エネルギー・エントロピーなど）の符号が通常と反転している粒子・物質。

| 逆転則 | 粒子・物質 | 概要 |
|--------|-----------|------|
| 負質量 | ネゴトン g126 | 固有の負の実質量を持つ架空粒子。力と逆向きに加速する |
| 負質量（複合） | ネゴトロン g247 | 正負ネゴトンが対をなす電気的中性の負質量複合体 |
| 負質量（凝縮） | ボース・ネゴトン g316 | ネゴトンが超低温でBEC的集積体を形成した状態 |
| 負エネルギー | ホロウゲイザー g344 | ディラックサイフォン相境界から放出される負エネルギー担体 |
| 負エントロピー | レトロン g163 | 高エントロピー系に引き寄せられ局所的な秩序をもたらす |
| 虚数温度 | 虚数温度 g343 | 負エネルギー系の熱平衡を記述するための概念（粒子ではなく状態量） |
| 空間省略 | コーラ粒子 g127 | 時間は経過するが空間を経由せず移動する |
| エネルギー全反転 | パランティ粒子 g161 | 反粒子を超えた第三カテゴリ。E・CPT全反転 |
| エネルギー符号のみ反転 | パラポジ粒子 g209 | 電荷・スピンは通常粒子と同符号、エネルギーのみ反転 |

### 構造的異常系

符号の反転ではなく、時空や伝播の構造そのものが通常物理の枠外にある粒子・場。

| 性質 | 粒子・場 | 概要 |
|------|---------|------|
| 計量テンソル直結 | アンキロン g128 | 質量・電荷を介さず時空の計量テンソルに直接結合する |
| エバネッセント伝播 | エイソン g277 | 遠距離には届かず、内部構造に直接染み込む粒子 |
| 時空接続形式の場 | トポロフィ g293 | 多様体のどの点とどの点がつながるかを決定する架空の場 |
| トポロジー励起 | トポロン g294 | トポロフィの局所励起。接続形式そのものを書き換える事象 |

## 実在仮説（speculative）

実在仮説はあるが未観測の粒子・物質。エキゾチック物質（g068）の枠組みに含まれる。

| 逆転則 | 粒子・物質 | 概要 |
|--------|-----------|------|
| 負質量 | 負の質量 g072 | 力と逆方向に加速・暴走運動を示す |
| 虚数質量 | タキオン g069 | 光速未満に減速できず光速を下限とする |
| 負エネルギー密度 | エキゾチック物質 g068 | 負エネルギー密度等を持つ物質の総称 |
| 安定性逆転 | ストレンジ物質 g071 | 通常核物質より安定な可能性（ウィッテン予想） |

## 派生技術

上記粒子・物質を基礎とする架空の工学・技術システム。

| 技術名 | ID | 基礎粒子・物質 | 概要 |
|--------|-----|--------------|------|
| ピエゾアンキロン効果 | g317 | アンキロン | 計量テンソルの変形を電気エネルギーに変換する効果 |
| 計量測量学 | g263 | アンキロン | アンキロンを標識として空間の絶対座標を確立する測量体系 |
| 計量暦システム | g265 | アンキロン | アンキロンを軌道上の年標識として宇宙規模の統一暦を実現 |
| 反アンキロン除去技術 | g264 | アンキロン | 軌道上に散逸した反アンキロンを誘導・対消滅させる技術 |
| テルモスタシス板 | g194 | コーラ粒子 | コーラ粒子格子が「熱固定場」を発生し温度変化を抑制する板材 |
| テルモクラシス板 | g195 | コーラ粒子 | 周囲の物質を設定温度へ引き込む能動的な均一化板材 |
| トポロフィピン | g315 | トポロフィ・コーラ粒子 | トポロフィ場を局所励起しコーラ粒子の出現領域を閉域に限定する機構 |
| パラドックス粒子通信 | g153 | パラドックス粒子 | パラドックス粒子の放出事実を信号として利用する通信技術 |
| パランティ粒子生成技術 | g261 | パランティ粒子 | エキゾチック物質の精密配置とレトロンのエントロピー操作でパランティ粒子を生成・供給する技術 |
| 余剰次元バンク | g262 | 余剰次元 | 余剰次元をエネルギー・エントロピーの貯蔵機構として利用する概念 |
