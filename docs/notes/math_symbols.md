## 概要

数学・物理で使われる記号のうち、ギリシャ文字以外のものを由来・導入者・意図とともに整理する。ギリシャ文字の命名理由は別補遺「ギリシャ文字の命名理由」を参照。

---

## 微積分・解析系

### ∂ — 偏微分記号（ラウンドd、カーリーd）

| 項目 | 内容 |
|------|------|
| 読み | ラウンドd / パーシャル |
| 導入 | アドリアン＝マリ・ルジャンドル（1786年頃） |
| 由来 | ラテン文字 d の筆記体を変形した「丸みのある d」 |

多変数関数において「ある変数だけで微分し、他を定数扱いにする」操作を表す。通常の微分記号 d と区別するために形を変えた。熱力学・流体力学・量子力学など偏微分方程式が支配的な分野で至る所に登場する。

---

### ∫ — 積分記号

| 項目 | 内容 |
|------|------|
| 読み | インテグラル |
| 導入 | ゴットフリート・ライプニッツ（1675年） |
| 由来 | ラテン語 **summa**（和）の頭文字 s を縦に引き伸ばした形 |

「無限に細かく分割した値の総和」という積分の本質を記号の形で表している。ライプニッツは記号の設計にこだわりがあり、dx・dy などの微分記号も彼が導入した。ニュートンとの微積分の発見論争は有名だが、現代で使われる記号体系はほぼライプニッツ由来である。

---

### ∇ — ナブラ（del）

| 項目 | 内容 |
|------|------|
| 読み | ナブラ / デル |
| 導入 | ウィリアム・ハミルトン、普及はピーター・テイト（1860年代） |
| 由来 | 古代ヘブライ・フェニキアのハープ型弦楽器「ネヴェル（nevel）」の形 |

勾配（grad）・発散（div）・回転（rot/curl）を統一的に表す微分演算子。∇f で勾配、∇·F で発散、∇×F で回転となる。電磁気学のマクスウェル方程式、流体力学のナビエ＝ストークス方程式など、場を扱う物理学で不可欠な記号。

---

### d/dx, dy/dx — ライプニッツ記法

ライプニッツが導入した微分の記法。「y の x に対する変化率」を dy/dx と分数のように書く。ニュートンの ẏ（点記法）より直感的に連鎖律などが書けるため現代では標準となっている。

---

## 定数・特殊値

### ∞ — 無限大

| 項目 | 内容 |
|------|------|
| 読み | インフィニティ |
| 導入 | ジョン・ウォリス（1655年） |
| 由来 | ラテン語 **infinitas**（果てしなさ）の頭文字、またはローマ数字 1000（M）を崩した形、あるいは古代のウロボロスに由来する説など諸説あり |

「限りなく大きくなる」という概念の記号。数学的には「∞ という数」ではなく「発散する極限」を表す。カントールが無限集合の濃度理論を確立してからは、ℵ（アレフ）記号と使い分けられるようになった。

---

### ℏ — エイチバー（換算プランク定数）

| 項目 | 内容 |
|------|------|
| 読み | エイチバー / ディラック定数 |
| 導入 | ポール・ディラック（1926年頃） |
| 由来 | プランク定数 h にバー（横棒）を付けた変形。h/(2π) を表す |

量子力学では角振動数 ω を使うと h より ℏ = h/2π の方が式が単純になる。ディラックがこの省略形を積極的に使ったため「ディラック定数」とも呼ばれる。シュレーディンガー方程式・不確定性原理・交換関係など量子力学の中心式に登場する。

---

## 集合論・論理学

### ℵ — アレフ（無限集合の濃度）

| 項目 | 内容 |
|------|------|
| 読み | アレフ |
| 導入 | ゲオルク・カントール（1883年） |
| 由来 | ヘブライ文字の第1字 **א**（アレフ） |

自然数全体の濃度を ℵ₀（アレフ・ゼロ）、実数全体の濃度を ℵ₁ 以上として表す。ギリシャ文字でも数字でもなくヘブライ文字を選んだのは、カントールが「既存の数学記号と混同されないようにしたい」と考えたため。

---

### ∀ — 全称量化子（for all）

| 項目 | 内容 |
|------|------|
| 読み | フォーオール / 任意の |
| 導入 | ゲルハルト・ゲンツェン（1935年頃）、普及はラッセルらの記号論理 |
| 由来 | "All" の A を逆さにした形 |

「すべての x について〜が成立する」を ∀x と書く。

---

### ∃ — 存在量化子（there exists）

| 項目 | 内容 |
|------|------|
| 読み | ゼアエグジスツ / 存在する |
| 導入 | バートランド・ラッセル（1897年頃） |
| 由来 | "Exists"（または "There exists"）の E を逆さにした形 |

「〜となる x が少なくとも1つ存在する」を ∃x と書く。∀ との対で論理式の骨格を作る。

---

### ∈ — 元素記号（集合の帰属）

| 項目 | 内容 |
|------|------|
| 読み | イン / 属する |
| 導入 | ジュゼッペ・ペアノ（1889年） |
| 由来 | ギリシャ文字 ε（エプシロン）を崩した形。「ἐστί（estin、〜である）」の頭文字 |

「x ∈ A」は「x は集合 A の元（要素）である」を表す。ε と区別するため現在の ∈ の形に変形されている。

---

## 演算・関係

### ! — 階乗

| 項目 | 内容 |
|------|------|
| 読み | ファクトリアル |
| 導入 | クリスティアン・クランプ（1808年） |
| 由来 | 感嘆符を転用。以前は n(n-1)(n-2)… と展開するか「n 階乗」と言葉で書いていた |

n! = n×(n-1)×…×2×1。組み合わせ・順列の計算の核。スターリングの近似 n! ≈ √(2πn)(n/e)^n によって e（g226）・π と深く結びつく。

---

### ≡ — 合同記号

| 項目 | 内容 |
|------|------|
| 読み | コングルエント / 合同 |
| 導入 | カール・フリードリヒ・ガウス（1801年『整数論研究』） |
| 由来 | = に横棒を1本追加した形。「より強い等しさ」を意図 |

a ≡ b (mod n) は「a と b を n で割った余りが等しい」を意味する。オイラーの定理（g230）の a^φ(n) ≡ 1 (mod n) はこの記号で書かれる。

---

## まとめ——記号設計の哲学

数学記号の多くは「既存の文字を変形・反転・拡張する」という方法で作られている。

| 方法 | 例 |
|------|----|
| 文字の伸長 | ∫（s を縦に伸ばす） |
| 文字の丸め | ∂（d を丸める） |
| 文字の反転 | ∀（A を逆さ）、∃（E を逆さ） |
| 文字の変形 | ∈（ε を崩す） |
| 異文化の文字 | ℵ（ヘブライ文字） |
| 既存記号の修飾 | ℏ（h にバー） |

「形が意味を示唆する」という設計思想が一貫しており、∫ が「和（summa）」を、∂ が「普通の微分ではない」を視覚的に伝える。
