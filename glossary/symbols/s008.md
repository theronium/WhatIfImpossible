# ∃ — 存在量化子（existential quantifier）

`\exists`　数学・論理・幾何学 / s008
読み: そんざいりょうかし　別称: ゼアエグジスツ / there exists　

「〜となる x が少なくとも1つ存在する」を ∃x と書く。"Exists" の E を逆さにした形。バートランド・ラッセルが1897年頃に導入。
