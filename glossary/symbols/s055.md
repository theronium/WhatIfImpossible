# ∎ — ハルモス記号（Halmos symbol）

`\blacksquare`　数学・論理・幾何学 / s055
読み: はるもすきごう　別称: tombstone / QED / 証明終 / 墓石記号　

数学の証明の末尾に置く「証明終了」の印。ポール・ハルモスが導入したことからハルモス記号と呼ばれる。QED（Quod Erat Demonstrandum：これが示すべきことであった）の代替として広く使われる。

塗りつぶし版（∎）と中空版（□）があり、どちらも同じ意味で使われる。中空版 □ は「□」（U+25A1）、塗りつぶし版 ∎ は「∎」（U+220E）。飾り文字や区切りとして装飾的に使われることもある。
