# ∵ — なぜならば（because）

`\because`　数学・論理・幾何学 / s054
読み: なぜならば　別称: because / 故に（逆） / 三点リーダ（根拠）　

「なぜならば」を表す論理記号。3つの点を正三角形に配置した形（∵）で、結論の根拠・理由を示すときに使う。「B ∵ A」は「B である。なぜならば A だからだ」を意味する。

対になる記号は ∴（ゆえに / therefore）。証明では ∵ で根拠を提示し、∴ で結論をまとめる順で書くことが多い。
