# ℏ — ディラック定数（reduced Planck constant）

`\hbar`　量子物理学 / s001
読み: えいちばー　別称: エイチバー / h-bar / hbar / ディラック定数　

プランク定数 h を 2π で割った値（ℏ = h/2π ≈ 1.055×10⁻³⁴ J·s）。量子力学の中心式（シュレーディンガー方程式・不確定性原理・交換関係など）に広く登場する。角振動数 ω を使うと h より ℏ の方が式が単純になるため、ディラックが積極的に採用した。
