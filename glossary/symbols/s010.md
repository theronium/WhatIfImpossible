# ≡ — 合同記号（congruent / identical to）

`\equiv`　数学・論理・幾何学 / s010
読み: ごうどう　別称: コングルエント / 三本線　

a ≡ b (mod n) で「a と b を n で割った余りが等しい」を意味する。ガウスが1801年『整数論研究』で導入。= に横棒を1本追加した「より強い等しさ」を表す形。
