# ∀ — 全称量化子（universal quantifier）

`\forall`　数学・論理・幾何学 / s007
読み: ぜんしょうりょうかし　別称: フォーオール / for all　

「すべての x について〜が成立する」を ∀x と書く。"All" の A を逆さにした形。∃（存在量化子）と対になって論理式の骨格を作る。
