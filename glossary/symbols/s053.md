# ∴ — ゆえに（therefore）

`\therefore`　数学・論理・幾何学 / s053
読み: ゆえに　別称: therefore / 故に / 三点リーダ（論理）　

「ゆえに」を表す論理記号。3つの点を逆三角形に配置した形（∴）で、前提から結論を導く推論の区切りに使う。数学の証明でよく登場し、「A ∴ B」は「A であるから B である」を意味する。

対になる記号は ∵（なぜならば / because）。証明の書き方では ∵ で根拠を示し、∴ で結論をまとめる流れが一般的。
