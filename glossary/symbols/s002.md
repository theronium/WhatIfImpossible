# ∇ — ナブラ演算子（nabla / del operator）

`\nabla`　数学・論理・幾何学 / s002
読み: なぶら　別称: デル / del / nabla　

ベクトル微分演算子。スカラー場に作用すると勾配（grad）、ベクトル場に内積をとると発散（div）、外積をとると回転（rot/curl）を与える。形は古代ヘブライ・フェニキアのハープ型楽器「ネヴェル（nevel）」に由来する。
