# ∂ — 偏微分記号（partial derivative）

`\partial`　数学・論理・幾何学 / s003
読み: へんびぶん　別称: round d / curly d / ラウンドd / d rond / パーシャル　

多変数関数において1変数のみで微分し他を定数扱いにする操作を表す。通常の微分記号 d と区別するためラテン文字 d を丸めた形に変形した（ラウンドd）。1786年頃ルジャンドルが導入。
