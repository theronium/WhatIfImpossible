## アルファ凝縮（Alpha Condensation）

**読み**: あるふぁぎょうしゅく
**別名**: α凝縮 / アルファクラスター凝縮
**分野**: 核物理学・量子多体問題
**関連記事**: [wiim_069](../docs/physics/wiim_069.md), [wiim_070](../docs/physics/wiim_070.md), [wiim_071](../docs/physics/wiim_071.md)

[原子核](particle.md)内に存在するアルファ粒子（ヘリウム-4核：陽子2個＋中性子2個の塊）が、ボース＝アインシュタイン凝縮に似た量子コヒーレント状態を形成する現象。

アルファ粒子はスピン0のボソンであるため、多数が同一量子状態に凝縮するアルファ凝縮が理論上可能と考えられる。炭素-12の「ホイル状態」はその代表例で、3個のアルファ粒子が希薄なガス状クラスターとして緩く結合した構造を持つ。この励起状態が[恒星](astronomy.md)内の三重アルファ過程（3He-4→C-12）における核融合の共鳴窓を提供しており、生命必須元素である炭素が宇宙に存在できる理由の一つでもある。

WIIMの思考実験では、[カシミールフォージ](wiim-engineering.md)（[wiim_023](../docs/physics/wiim_023.md)）や[コーラ粒子](wiim-concepts.md)（g013）による量子状態操作がアルファ凝縮状態の核を標的とすることで、通常より低いエネルギーで[核変換](particle.md)（g272）を誘発できるかが問われる（[wiim_069](../docs/physics/wiim_069.md)）。

## アルファ固着問題（Alpha Sticking Problem）

**読み**: あるふぁこちゃくもんだい
**別名**: アルファ固着 / α固着問題
**分野**: 核物理学・素粒子物理学
**関連記事**: [wiim_069](../docs/physics/wiim_069.md), [wiim_070](../docs/physics/wiim_070.md), [wiim_071](../docs/physics/wiim_071.md)

[ミュオン触媒核融合](particle.md)（g273）において、核融合反応で生成されたヘリウム-4核（アルファ粒子）にミュオンが捕獲され、触媒として再利用できなくなる現象。

ミュオン触媒核融合では1個のミュオンが[水素](particle.md)核の周囲を軌道運動して核間距離を縮め、[クーロン障壁](particle.md)（g271）を回避しながら核融合を触媒する。理想的にはミュオンは反応後に解放されてサイクルを繰り返すが、実際には生成されたアルファ粒子（He-4核）の強い電荷に引きつけられ、約0.7〜1%の確率でそのまま捕獲される。これがアルファ固着問題だ。

この確率的捕獲により1個のミュオンが触媒できる反応回数は平均100〜300回程度に限られ、ミュオンの生産コストが得られる核融合エネルギーを上回るためエネルギー収支は赤字のままだ。固着確率を劇的に下げる物理的な解決策は現在も見つかっていない。

WIIMの思考実験では、[パランティ粒子](wiim-concepts.md)（g161）が固着ミュオンをアルファ粒子から「静かに引き剥がす」機構として機能できるかが論点となりうる（[wiim_069](../docs/physics/wiim_069.md)）。

## LHC（Large Hadron Collider）

**読み**: えるえいちしー
**別名**: 大型ハドロン衝突型加速器 / 粒子加速器
**分野**: 素粒子物理学・加速器科学
**関連記事**: [ヒッグス粒子](particle.md#g070)

スイス・フランス国境地下100mに建設されたCERN（欧州原子核研究機構）の円形粒子加速器。円周約27km、陽子を光速の99.9999991%まで加速して正面衝突させる。2008年に稼働開始し、現在も世界最大・最高エネルギーの加速器として運用されている。

最大の成果は2012年の[ヒッグス粒子](particle.md)（g070）の発見で、素粒子物理学の標準模型の最後のピースを埋めた。このほかトップ[クォーク](particle.md)の精密測定、[反物質](speculative.md)の性質研究、クォーク・[グルーオン](quantum.md)・プラズマの生成（初期宇宙の再現）など多岐にわたる成果を上げている。

稼働前にはストレンジレット生成・マイクロブラックホール生成による地球消滅リスクが一部で議論されたが、LHCが達成するエネルギーは宇宙線が日常的に大気に降り注ぐエネルギーより低く、安全性評価委員会が問題なしと結論づけている。

現在は第3稼働期（Run 3）が進行中で、2026年以降にはさらに高輝度化した「HL-LHC（高輝度LHC）」へのアップグレードが計画されている。標準模型を超えた物理（超対称性粒子・[余剰次元](particle.md)など）の探索が続いている。

## 核物理学（Nuclear Physics / Scattering Theory）

**読み**: かくぶつりがく
**別名**: 核物理 / 散乱理論
**分野**: 核物理学・素粒子物理学
**関連記事**: —

[原子核](particle.md)の構造・反応・崩壊を扱う物理学。核分裂・核融合（g254）・放射性崩壊・核力（強い相互作用）を主題とし、散乱理論（粒子の衝突反応の確率を反応断面積として記述する手法）を主要ツールとして用いる。

WIIMでは[ストレンジ物質](speculative.md)（g071）の制御技術とストレンジスター建造の前提科学として技術ツリーに登場する。ストレンジ物質は核物理の延長線上にある概念であり、その制御には散乱断面積レベルの精密な核反応制御が必要とされる。

## 核変換（Nuclear Transmutation）

**読み**: かくへんかん
**別名**: 元素変換 / 核種変換
**分野**: 核物理学・放射化学
**関連記事**: [wiim_024](../docs/biology/wiim_024.md), [wiim_069](../docs/physics/wiim_069.md), [wiim_070](../docs/physics/wiim_070.md), [wiim_071](../docs/physics/wiim_071.md)

[原子核](particle.md)の陽子数・中性子数を変化させることで、ある元素を別の元素へ転換する現象または技術の総称。核分裂・核融合・中性子捕獲・放射性崩壊などがすべて核変換の一形態だ。

歴史的には「[錬金術](sf-concepts.md)（g117）の科学的実現」とも呼ばれ、1919年にラザフォードが窒素に高速アルファ粒子を照射して酸素と水素を生成したのが人工核変換の最初の成功例とされる。現代では加速器で重イオンを衝突させて超重元素を合成したり、原子炉内の中性子照射で放射性廃棄物の半減期を短縮したりする研究が進んでいる。

WIIM世界では、[マイコプラズマギカ](wiim-concepts.md)（g130）が生物的に核変換を行う設定として登場するほか、架空粒子を利用した安価な核変換の実現可能性も思考実験の対象となっている（[wiim_069](../docs/physics/wiim_069.md)）。

## 核融合技術（Nuclear Fusion Technology）

**読み**: かくゆうごうぎじゅつ
**分野**: 核物理学・エネルギー工学
**関連記事**: —

水素の同位体（重水素・三重水素）などの軽い[原子核](particle.md)を超高温・超高圧下で融合させ、質量欠損によって莫大なエネルギーを取り出す技術。[太陽](astronomy.md)が内部でこのプロセスによってエネルギーを生成している。地上での制御された核融合の実現はITER（国際熱核融合実験炉）などで現在も研究中だ。

WIIMの技術ツリーでは、[クロノスフィア](wiim-concepts.md)内部の光量問題（[wiim_057](../docs/physics/wiim_057.md)）を解決するために「内部完結型核融合光源」として登場する。内部時間で億年単位の連続運転が必要とされるため、燃料補給の境界面通過問題という固有の障壁に直面する。

## 基本相互作用（Fundamental Interactions）

**読み**: きほんそうごさよう
**別名**: 四つの力 / 四種の力 / 自然界の四力
**分野**: 素粒子物理学・場の量子論
**関連記事**: —

自然界に存在する4種類の相互作用——重力・電磁力・強い力・弱い力——の総称。現代物理学では「力」よりも「相互作用」という表現が正確で、それぞれゲージボソンと呼ばれる媒介粒子によって伝わる。

電磁力は光子、強い力は[グルーオン](quantum.md)、弱い力はWボソン・Zボソンが媒介する。重力だけはまだ量子論的な記述（[重力子](speculative.md)による媒介）が確立しておらず、[一般相対性理論](physics.md)と[量子力学](quantum.md)の統合は現代物理学最大の未解決問題の一つだ。

4つの力の強さは桁違いに異なり、強い力を1とすると電磁力は約1/137、弱い力は約10⁻⁶、重力は約10⁻³⁸と極端に弱い。この「階層性問題」——なぜ重力だけこれほど弱いのか——は[余剰次元](particle.md)仮説などで説明が試みられている。

<a id="g271"></a>
## クーロン障壁（Coulomb Barrier）

**読み**: くーろんしょうへき
**分野**: 核物理学・素粒子物理学
**関連記事**: [wiim_024](../docs/biology/wiim_024.md), [wiim_069](../docs/physics/wiim_069.md), [wiim_070](../docs/physics/wiim_070.md)

同じ符号の電荷を持つ[原子核](particle.md)同士が近づくとき、電磁反発力（クーロン力）によって形成されるエネルギー障壁。核融合や[核変換](particle.md)を実現するには、衝突する原子核がこの障壁を越えるか、[量子トンネル効果](quantum.md)（g097）によってすり抜ける必要がある。

水素原子核（陽子）を別の陽子に近づけようとすると、近づくほど電磁反発力が急激に増大する。核力（強い力）が支配するのは約1フェムトメートル（10⁻¹⁵ m）以内だが、その距離まで近づくためのエネルギー障壁がクーロン障壁だ。[太陽](astronomy.md)の核融合では超高温（約1500万K）と量子トンネル効果の組み合わせでこの障壁をかろうじて越えている。

地上の核融合炉では、この障壁を越えるためにプラズマを1億K以上まで加熱する必要があり、それが核融合発電の最大の技術的障壁となっている。WIIMの思考実験では、[カシミールフォージ](wiim-engineering.md)・[コーラ粒子](wiim-concepts.md)・[ネゴトン](wiim-concepts.md)などの架空概念でこの障壁を回避する可能性が論じられる（[wiim_069](../docs/physics/wiim_069.md)）。

## 原子核の階層構造（Hierarchical Structure of Matter）

**読み**: げんしかくのかいそうこうぞう
**別名**: 原子 / 原子核 / 核子 / ハドロン / バリオン / メソン / クォーク / 陽子 / 中性子 / グルーオン / 水素 / プロトン
**分野**: 素粒子物理学・核物理学
**関連記事**: [wiim_006](../docs/quantum/wiim_006.md), [wiim_007](../docs/quantum/wiim_007.md)

物質は入れ子状の階層構造を持ち、より深い層ほど「強い力」が支配的になる。

```
クォーク（u/d/c/s/t/b）
└─ ハドロン（強い力・[グルーオン](quantum.md)で束縛）
     ├─ バリオン（クォーク×3）
     │    ├─ 陽子／プロトン（uud）── 水素原子核・安定
     │    ├─ 中性子／ニュートロン（udd）── 原子核内で安定
     │    └─ その他（Σ・Λ・Ξ・Ωなど、不安定）
     └─ メソン（クォーク＋反クォーク）
          ├─ パイオン（π）── 核力の媒介
          ├─ ケーオン（K）
          └─ その他（不安定・すぐ崩壊）
原子核（陽子＋中性子、核力で束縛）
└─ 原子（原子核＋電子、電磁力で束縛）
     └─ 分子（原子の結合、化学結合）
```

**クォーク**は現在知られる最小の物質構成要素で6種類ある。単独では存在できず（**クォーク閉じ込め**）、グルーオンを媒介とする強い力によって常に複数が束縛された状態にある。

**ハドロン**はクォークが束縛してできた粒子の総称。クォーク3個の**バリオン**と、クォーク＋反クォークの**メソン**に大別される。**陽子（プロトン）はバリオンの一種**であり、安定して存在できる唯一のバリオンでもある。中性子は単独では約15分で崩壊するが、原子核内では安定を保つ。陽子と中性子をあわせて**核子**と呼ぶ。

**水素とプロトン**は文脈によって同義で使われる特殊な例だ。水素原子は陽子1個＋電子1個という最も単純な原子であり、電子を取り去った水素イオン（H⁺）は陽子そのものとなる。化学では「酸がプロトンを放出する」と表現し（ブレンステッド・ローリー定義）、素粒子実験では「陽子ビーム＝水素の核」として扱う。

核子が集まり**原子核**を形成し（核力＝パイ中間子の交換による）、周囲を電子が取り巻いたものが**原子**、さらに原子が結合したものが**分子**となる。[パウリの排他原理](quantum.md)（g060）は電子・核子・クォークすべてのフェルミオン（g064）に適用され、この階層構造の安定性を根底で支えている。もしクォーク閉じ込めが解除されれば**クォーク・グルーオンプラズマ**が生じ、[ビッグバン](astronomy.md)（g082）直後の状態が再現される。

## CP対称性（CP Symmetry）

**読み**: しーぴーたいしょうせい
**別名**: CP不変性 / CP対称性の破れ
**分野**: 素粒子物理学・場の理論・宇宙論
**関連記事**: [wiim_005](../docs/cosmology/wiim_005.md), [wiim_015](../docs/physics/wiim_015.md)

物理法則が「電荷共役変換（C）」と「パリティ変換（P）」を同時に行っても変わらないという対称性。C変換は粒子を反粒子に置き換える操作、P変換は空間を鏡像反転する操作であり、両者を組み合わせたCP変換の下で自然法則が不変であれば「CP対称性が保たれている」という。

1964年にジェームズ・クローニンとヴァル・フィッチがK中間子の崩壊においてCP対称性が破れていることを発見した（1980年ノーベル物理学賞）。この「CP対称性の破れ」は宇宙論的に極めて重要だ——宇宙誕生時に物質と[反物質](speculative.md)が等量生成されたとすれば互いに消滅して何も残らないはずだが、現在の宇宙が物質で満ちているのはCP対称性の破れが物質優勢を生み出したためと考えられている。

[CPT定理](particle.md)によれば、CP変換に時間反転（T）を加えたCPT変換は必ず保存される。したがってCP対称性が破れていれば時間反転対称性（T対称性）も同程度破れていることになる。時間の矢や時間遡行を論じる[wiim_005](../docs/cosmology/wiim_005.md)・[wiim_015](../docs/physics/wiim_015.md)とも深く関わる概念だ。小林・益川理論はCP対称性の破れを[クォーク](quantum.md)の混合行列（CKM行列）で説明し、2008年のノーベル賞につながった。

<a id="g011"></a>
## CPT定理（CPT Theorem）

**読み**: しーぴーてぃーていり
**別名**: CPT対称性
**分野**: 素粒子物理学・場の量子論
**関連記事**: [wiim_015](../docs/physics/wiim_015.md)

素粒子の物理法則は、以下の三つの変換を**同時に**行っても変わらないという[定理](mathematics.md)。

- **C**（電荷共役変換）：粒子を反粒子に置き換える
- **P**（空間反転）：空間座標を鏡像に変える（左右反転）
- **T**（時間反転）：時間の方向を逆にする

C・P・Tをそれぞれ単独で行うと対称性が破れる場合がある（特に[CP対称性の破れ](particle.md)は実験で確認済み）。しかしCPT全てを同時に行うと常に対称になる——これが標準模型の根幹的な予測だ。

CPT定理は「時間を逆向きにした[反物質](speculative.md)宇宙は、私たちの宇宙と全く同じ物理法則に従う」ことを意味する。

<a id="g065"></a>
## 質量ゼロの粒子（Massless Particle）

**読み**: しつりょうぜろのりゅうし
**分野**: 素粒子物理学・特殊相対性理論
**関連記事**: [ローレンツ不変性](physics.md#g063), [wiim_020](../docs/physics/wiim_020.md)

静止質量がゼロの素粒子。質量を持つ粒子は理論上「静止した状態」が存在できるが、質量ゼロの粒子を止めると運動量もエネルギーもゼロになり存在できなくなる。そのため**必ず真空中を光速 c で伝播し続けるしかない**という構造的必然がある。これは[ローレンツ不変性](physics.md)から導かれる帰結だ。

現在確認されている質量ゼロの粒子は以下の2種類。**光子（フォトン）**は電磁力を媒介するボソンで、可視光・電波・X線などすべての電磁波の量子。**[グルーオン](particle.md)**は強い核力を媒介するボソンで、クォークをつなぎとめてハドロン（陽子・中性子など）を構成する。

理論上は**重力子（[グラビトン](speculative.md)）**も質量ゼロのボソンとして予測されているが、[重力波](astronomy.md)の量子として未だ直接検出されていない。

一方、[ニュートリノ](quantum.md)はかつて質量ゼロと考えられていたが、ニュートリノ振動の観測によって極めて小さいながらも質量を持つことが判明している。質量ゼロかどうかの境界は「光速で伝播するか否か」「時間の経過を経験するか否か」という根本的な差異をもたらす。

<a id="g093"></a>
## 超弦理論（Grand Unified Theory / Superstring Theory）

**読み**: ちょうひもりろん
**別名**: 超弦理論 / 弦理論 / ひも理論 / GUT / M理論 / 大統一理論 / 超ひも理論
**分野**: 理論物理・素粒子物理・量子重力
**関連記事**: —

自然界の基本的な力を一つの数式で記述しようとする、現代物理学における最大の挑戦。どちらも未証明の仮説である。

**大統一理論（GUT: Grand Unified Theory）**：電磁力・弱い力・強い力の3つを統一しようとする理論群。標準模型はこの3力をある程度統合しているが、重力だけが仲間外れのままだ。GUTは重力を除く3力の統一を目指し、多くの候補理論が[陽子](particle.md)崩壊を予言するが、観測されたことがない。

**超弦理論（超ひも理論）**：素粒子を点ではなく「1次元のひも」として扱うことで、[量子力学](quantum.md)と重力を含む4つの力すべてを統一しようとする理論。10〜11次元の[余剰次元](particle.md)を必要とし、数学的な整合性は非常に高い。しかし検証に必要なエネルギースケールは現在の[LHC](particle.md)（g074）の能力を遥かに超えており、理論の解（真空候補）が10の500乗個以上存在するため「どれが現実の宇宙か」を選ぶ[原理](physics.md)がない。反証可能性を持たないという批判も根強い。

両理論に共通するのは「数学的に美しく整合的でも、物理的に正しいとは限らない」という問題だ。これは[ゲーデルの不完全性定理](mathematics.md)（g090）の構図——矛盾のない体系が現実を完全に記述するとは限らない——と本質的に通じる。[量子力学の未解決問題](quantum.md)（g092）の中核に位置する課題でもある。

## ディラックの弦（Dirac String）

**読み**: でぃらっくのひも
**分野**: 素粒子物理学・場の理論
**関連記事**: [wiim_091](../docs/physics/wiim_091.md)

[磁気モノポール](speculative.md)（単磁荷）の理論的定式化においてポール・ディラックが導入した概念的な細線。電磁気学をモノポールが存在する場合に矛盾なく記述しようとすると、モノポールから無限遠に伸びる「観測不可能な弦」をベクトルポテンシャルの特異線として仮定する必要が生じる。

弦の向きはゲージの選択に依存し、観測可能な物理量には影響しないことが示されている。ディラックはこの議論から、電荷の量子化（素電荷の整数倍しか存在しない理由）がモノポールの存在によって自然に説明されることを導いた。[スピンアイス結晶](quantum.md)における準粒子モノポールの間にも類似の弦構造が現れる。

<a id="g070"></a>
## ヒッグス粒子（Higgs Boson）

**読み**: ひっぐすりゅうし
**別名**: ヒッグスボソン / 神の粒子
**分野**: 素粒子物理学・標準模型
**関連記事**: [ボソン／フェルミオン](quantum.md#g064)

素粒子に質量を与える機構（ヒッグス機構）の鍵となるボソン。1964年にイギリスの物理学者ピーター・ヒッグスらが理論的に予言し、2012年にCERN（欧州[原子核](particle.md)研究機構）の[大型ハドロン衝突型加速器](particle.md)（LHC）で発見された。この発見によりヒッグスとフランソワ・アングレールは2013年のノーベル物理学賞を受賞している。

素粒子物理学の標準模型では、素粒子はヒッグス場という空間全体に満ちた場と相互作用することで質量を獲得すると説明される。ヒッグス場との相互作用が強い粒子ほど重く、まったく相互作用しない粒子（光子・[グルーオン](quantum.md)）は質量ゼロになる。ヒッグス粒子はこのヒッグス場の量子（場の振動の粒子）だ。

ヒッグス粒子はスピン0のボソンで、現在確認されている基本粒子の中で唯一スピンゼロを持つ。質量は約125GeV/c²（陽子の約133倍）で非常に重く、LHCのような高エネルギー加速器でなければ生成できない。また非常に不安定で、生成後ただちに他の粒子へ崩壊する。

標準模型の最後の未発見ピースだったヒッグス粒子の発見は理論の完成を意味する一方、なぜヒッグス場が存在するのか・ヒッグス粒子が複数種類あるかどうかなど標準模型を超えた謎も残している。

<a id="g217"></a>
## マヨラナ粒子（Majorana Fermion）

**読み**: まよらなりゅうし
**別名**: マヨラナ準粒子 / マヨラナ
**分野**: 素粒子物理学・量子情報
**関連記事**: [wiim_050](../docs/quantum/wiim_050.md), [wiim_051](../docs/physics/wiim_051.md)

自分自身が反粒子でもあるフェルミオン。1937年にイタリアの物理学者エットレ・マヨラナが提唱した。通常の粒子（ディラックフェルミオン）は粒子と反粒子が別々に存在するが、マヨラナフェルミオンはそれ自体が反粒子と一致するため、粒子・反粒子の区別が消える。

素粒子としてのマヨラナ粒子は未発見だが、[ニュートリノ](quantum.md)がマヨラナ粒子である可能性は現在も研究されている。一方、物性物理学では[トポロジカル](physics.md)[超伝導](quantum.md)体の端点に「マヨラナ準粒子」として観測されており、2020年代から実験的証拠が積み重ねられている。

マヨラナ準粒子は局所的な摂動に対して量子情報を保護する性質を持ち、トポロジカル量子計算の基盤として注目される——[量子デコヒーレンス](quantum.md)（[wiim_050](../docs/quantum/wiim_050.md)）の壁を克服する候補技術の一つだ。「自分自身が反粒子」という性質は[パランティ粒子](wiim-concepts.md)（g161）や[パラポジ粒子](wiim-concepts.md)（g209）の設計思想とも対比できる。

## ミュオン触媒核融合（Muon-Catalyzed Fusion）

**読み**: みゅおんしょくばいかくゆうごう
**別名**: μ触媒核融合 / ミュオン核融合
**分野**: 核物理学・素粒子物理学
**関連記事**: [wiim_069](../docs/physics/wiim_069.md), [wiim_070](../docs/physics/wiim_070.md), [wiim_071](../docs/physics/wiim_071.md)

ミュオン（[電子](quantum.md)の約207倍の質量を持つ不安定な素粒子）を触媒として利用し、常温・常圧に近い環境で水素の核融合を引き起こす手法。電子より重いミュオンが水素[原子核](particle.md)の周囲を軌道運動すると、原子半径がミュオンの質量比に反比例して約207分の1に縮小し、2つの水素核が核力の届く距離まで近づける。これにより[クーロン障壁](particle.md)（g271）を熱エネルギーではなくミュオンの質量で回避する。

1957年にルイス・アルバレスが実験的に確認し、[原理](physics.md)的には実在する技術だ。しかし1個のミュオンが触媒できる核融合反応は平均100〜300回程度で、ミュオンが融合したヘリウム核に捕獲されて再利用できなくなる「[アルファ固着](particle.md)」が問題となる。また、ミュオンの生産に必要なエネルギーが得られる核融合エネルギーを上回るため、エネルギー収支はいまだ赤字だ。

WIIMの思考実験では、この触媒による障壁回避という概念を[コーラ粒子](wiim-concepts.md)や架空の重媒介粒子で代替することで、エネルギー収支の問題を解決できるかが論点となる（[wiim_069](../docs/physics/wiim_069.md)）。

## ミラー対称性（Mirror Symmetry）

**読み**: みらーたいしょうせい
**分野**: 素粒子物理学・弦理論・数理物理
**関連記事**: [wiim_007](../docs/quantum/wiim_007.md)

物理学における「鏡像」に関する対称性の総称。主に2つの文脈で使われる。

**パリティ対称性（P対称性）**：物理法則が空間反転（左右反転）に対して不変であるという性質。しかし弱い相互作用はパリティを破ることが1956年に理論予測・実験確認された。このパリティ破れを補償するために「ミラー物質」（鏡像粒子）の存在を仮定する理論がある——通常粒子と逆のパリティを持ち電磁相互作用せず、[ダークマター](speculative.md)（g056）候補の一つとされる。

**[弦理論](particle.md)のミラー対称性**：異なる2つの[カラビ・ヤウ多様体](mathematics.md)が物理的に等価な弦理論を生み出すという双対性。一方の複素幾何学が他方のシンプレクティック幾何学に対応し、「異なる空間形状が同一の物理法則を生む」ことを意味する。

いずれの意味でも、対称性の破れや物理法則の形が変わったとき何が起きるかという問いは、[パウリの排他原理](quantum.md)が変化した宇宙（[wiim_007](../docs/quantum/wiim_007.md)）の思考実験と通底する。

## 余剰次元（Extra Dimensions）

**読み**: よじょうじげん
**分野**: 高エネルギー物理学・弦理論
**関連記事**: —

我々が知覚する3次元空間（＋時間の4次元時空）の外側に存在すると予言される追加の空間次元。[超弦理論](particle.md)（g093）では理論的整合性のために6〜7個の余剰次元が必要とされ、[プランクスケール](quantum.md)（約10⁻³⁵m）に「丸まった（コンパクト化された）」状態にあると考えられている。

WIIMでは余剰次元が複数の技術の基盤概念として機能する。[コーラ粒子](wiim-concepts.md)（g127）の空間バイパスは余剰次元を経由するとされ、量子永久機関（[wiim_039](../docs/quantum/wiim_039.md)）は[エントロピー](physics.md)を余剰次元に転嫁することで「[余剰次元バンク](wiim-engineering.md)」を利用する。[グラビトーペイク](wiim-concepts.md)（g129）がエネルギーを転嫁する先も余剰次元とされる。

## ラザフォード散乱（Rutherford Scattering）

**読み**: らざふぉーどさんらん
**別名**: クーロン散乱
**分野**: 原子物理学・素粒子物理学・核物理学
**関連記事**: [wiim_029](../docs/physics/wiim_029.md)

アーネスト・ラザフォードが1911年に金箔実験で発見した荷電粒子の散乱現象。アルファ粒子を金箔に照射したところ、ほとんどが透過する一方でごく一部が大きく跳ね返された。この結果からラザフォードは原子の質量が中心の極めて小さな領域（[原子核](particle.md)）に集中しているという「ラザフォード模型」を導いた——それ以前の「プラム・プディング模型」を覆す発見だ。

散乱の数学的記述は「ラザフォード散乱公式」として定式化されており、クーロン力（電磁力）による散乱角θは衝突パラメータ（粒子の軌道と散乱中心の最短距離）と入射エネルギーで決まる。谷（ポテンシャル井戸）が深いほど大きな散乱角が得られるという関係は、電磁力だけでなく重力や核力にも一般化できる。[コーラ粒子](wiim-concepts.md)（[wiim_029](../docs/physics/wiim_029.md)）がストレンジスターの重力ポテンシャルで方向転換する[スイングバイ](astronomy.md)機構は、この散乱公式と同じ数学構造を[余剰次元](particle.md)に拡張したものとして理解できる。
