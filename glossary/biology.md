# 生物学・進化用語

---

## ウイルス（Virus）

**読み**: ういるす
**別名**: ウィルス / ウイルスと生命の境界 / 生命の境界 / 巨大ウイルス / ウイロイド
**分野**: ウイルス学・生命科学・哲学
**関連記事**: wiim_024

タンパク質の殻（カプシド）と核酸（DNAまたはRNA）からなる微小な粒子。細胞・リボソーム・代謝系を持たず、宿主細胞の機能を乗っ取ることでしか複製できない。この性質から「ウイルスは生物か否か」という問いが生まれ、現在も決着していない。

生命の条件と照らすと、ウイルスは**自己複製・遺伝情報・進化**という条件は満たすが、**代謝・恒常性・細胞構造**を持たない。シュレーディンガーが『生命とは何か』で問うた「負の[エントロピー](physics.md)を維持する系」という定義でも、宿主なしでは条件を満たせない。

境界をさらに曖昧にする存在も知られている。**ウイロイド**はタンパク質すら持たないRNAのみの病原体。**プリオン**はDNAもRNAも持たないタンパク質だけで「感染・複製」する。一方、**ミミウイルス**や**パンドラウイルス**などの**巨大ウイルス**は一部の細菌より遺伝子数が多く、独自のDNA修復機構まで持つ。

この境界の曖昧さはWhatIfImpossibleの思考実験と深く結びつく——「生命」の定義が揺らぐとき、生命を前提とした物理・倫理・進化の議論も根底から問い直される。

## エイリアン（Extraterrestrial Life）

**読み**: えいりあん
**別名**: 地球外生命体 / 宇宙人 / 地球外知的生命体 / 異星人 / ET / 異星生命
**分野**: 宇宙生物学・天文学・SF
**関連記事**: —

地球以外の[天体](astronomy.md)に起源を持つ生命体の総称。科学的文脈では**地球外生命体**（宇宙生物学の対象）、SF的文脈では知性を持つ**異星人**を指すことが多い。現在のところ地球外生命の存在は確認されていないが、宇宙の広大さと惑星の数を考えれば存在しない方が不自然だとする見方も根強い（[フェルミのパラドックス](sf-concepts.md) g037）。

宇宙生物学が最も可能性が高いとするのは微生物レベルの生命で、火星の地下・エウロパやエンケラドゥスの海洋・タイタンの液体メタン湖などが候補環境とされる（wiim_018）。[極限環境微生物](biology.md)（g030）の研究は、生命が想像以上に多様な条件に適応できることを示しており、炭素・水ベース以外の生化学（ケイ素・アンモニア溶媒など）の可能性も議論される。

知的エイリアンについては、[SETI](astronomy.md)（g094）が電波・光シグナルによる探索を続けているが成果は得られていない。この沈黙はフェルミの[パラドックス](philosophy.md)（g037）として知られ、「知的文明は[ワープ](sf-concepts.md)技術到達前に自滅する」「意図的に沈黙している」「我々の観測方法が間違っている」など多くの解釈を生む。WhatIfImpossibleの記事群はこの問いへの思考実験的アプローチでもある。

## 極限環境微生物（Extremophile）

**読み**: きょくかんきょうびせいぶつ
**別名**: エクストリモファイル / 好極限性生物 / 好熱菌 / 好圧菌 / 好酸菌 / クマムシ / 緩歩動物
**分野**: 微生物学・宇宙生物学
**関連記事**: [wiim_008](../docs/biology/wiim_008.md)

極端な環境——高温・低温・高圧・強酸・強アルカリ・高放射線・真空など——で生育できる微生物の総称。

- **好熱菌（Thermophile）**：100℃以上の熱水噴出孔に生育
- **好塩菌（Halophile）**：死海のような高塩濃度に生育
- **放射線耐性菌**：デイノコッカス・ラジオデュランスは人間の致死量の1000倍以上の放射線に耐える
- **クマムシ（Tardigrade）**：真空・宇宙線・極端な乾燥に耐える動物。宇宙空間への一時的な曝露でも生存例がある

宇宙空間での生命の可能性や[パンスペルミア](astronomy.md)説（生命の宇宙伝播）の議論において重要な証拠を提供する。

## 菌根ネットワーク（Mycorrhizal Network）

**読み**: きんこんねっとわーく
**別名**: ウッドワイドウェブ / Wood Wide Web
**分野**: 菌類学・生態学
**関連記事**: [wiim_008](../docs/biology/wiim_008.md)

樹木の根と菌類（菌根菌）が共生することで形成される地下のネットワーク。「ウッド・ワイド・ウェブ」とも呼ばれる。

木々はこのネットワークを通じて栄養素・水・化学的シグナル（隣の木が病害虫に攻撃されている情報など）をやり取りする。単独の木が持つ根の面積の数百倍に及ぶ面積のネットワークを形成することもある。

「森は個別の木の集合ではなく、一つの有機的なシステムである」という見方を支持する。分散型情報処理システムのモデルとして宇宙進化シナリオでも引用される。

## 菌糸（Mycelium / Hypha）

**読み**: きんし
**別名**: 菌糸体 / マイセリウム / ハイファ
**分野**: 生物学・菌類学・生態学
**関連記事**: [wiim_008](../docs/biology/wiim_008.md), wiim_017, wiim_018

菌類の基本的な構造単位である管状の細胞列。個々の管状細胞を「ハイファ（hyphae）」、その集合体を「菌糸体（マイセリウム）」と呼ぶ。肉眼で見えるキノコは菌糸体の一部が生殖のために地上に現れた「子実体」にすぎず、菌の本体は土壌中に広がる菌糸体だ。

菌糸は先端成長（apical growth）によって伸長し、障害物を回避しながら栄養源を探索する。この成長パターンは分散型コンピューティングや最短経路探索アルゴリズムの着想源にもなっている。菌糸の直径はわずか数マイクロメートルだが、一本の菌糸体が持つ総延長は数十キロメートルに及ぶこともある。

植物の根に共生する菌糸は菌根ネットワークを形成し、森林全体で栄養・情報・化学シグナルを輸送する。この巨大ネットワークは「[ウッドワイドウェブ](biology.md)」と呼ばれ、分散型知性の原型とも見なされる。菌糸体が宇宙空間で分散知性に進化する可能性を探った思考実験（wiim_008）や、[胞子](biology.md)（g111）による[惑星](astronomy.md)[テラフォーミング](sf-concepts.md)（wiim_017・wiim_018）において、菌糸は生態系構築の基盤として中心的役割を担う。

## クオリア（Qualia）

**読み**: くおりあ
**別名**: 感覚質
**分野**: 心の哲学・神経科学
**関連記事**: —

「赤の赤さ」「痛みの痛さ」「音楽の美しさ」——主観的な意識体験の質的な側面。英語の単数形は quale（クアーレ）。

物理的・機能的な説明では捉えられない「体験そのもの」の性質だとする立場（クオリア実在論）と、クオリアは概念的な混乱に過ぎないとする立場（クオリア排除論）に分かれる。

AI・人工生命・宇宙に生まれた知性体が「本当に何かを感じているか」を問うとき、クオリアの問題は避けられない。

## 自己組織化（Self-Organization）

**読み**: じこそしきか
**分野**: 複雑系科学・生物学
**関連記事**: [wiim_008](../docs/biology/wiim_008.md), [wiim_011](../docs/physics/wiim_011.md)

外部からの指令なしに、系の内部の相互作用だけから秩序・構造・パターンが自発的に生まれる現象。

例：雪の結晶・アリのコロニー・脳のニューラルネットワーク・細胞膜の形成・粘菌の集合体。共通するのは「局所的なルール（個々の要素の相互作用）からグローバルな構造が生まれる」点だ。

生命の起源を説明する重要な概念でもある。最初の細胞が「設計」されたのではなく、化学的な自己組織化によって自発的に生まれたとする説が主流だ。

## 収斂進化（Convergent Evolution）

**読み**: しゅうれんしんか
**別名**: 収束進化
**分野**: 進化生物学
**関連記事**: —

異なる系統の生物が、独立して似た形質・機能を進化させる現象。共通祖先ではなく、同様の環境的圧力への適応が原因。

例：魚類の鰭とイルカのひれ（水中での推進）、コウモリと鳥の翼（飛行）、タコとヒトの目（カメラ眼）——全く異なる進化の歴史から、機能的に類似した構造が生まれる。

「生命には取りうる形の制約がある」ことを示唆し、宇宙の別の星で生まれた生命が地球生命と似た形をとる可能性（収斂宇宙進化）の根拠として引用される。

## 胞子（Spore）

**読み**: ほうし
**別名**: スポア
**分野**: 生物学・菌類学・宇宙生物学
**関連記事**: wiim_017, wiim_018

菌類・植物・細菌などが無性生殖または有性生殖の一環として形成する、単独で新個体を生じうる細胞または細胞群。種子と異なり受精を必要とせず、極めて少ないエネルギーで大量に生産・散布できる。

胞子の最大の特徴は環境耐性の高さだ。菌類の胞子は乾燥・紫外線・高温・低温に強く、宇宙空間でも一定期間生存できるものがある。[極限環境微生物](biology.md)の胞子（特に芽胞形成細菌）は真空・放射線環境下でも数年単位で生存が確認されており、[パンスペルミア](astronomy.md)仮説において生命の[惑星](astronomy.md)間輸送手段として候補に挙がる。

散布手段も多様で、風・水・昆虫・爆発的な自己射出など生物ごとに異なる。[菌根ネットワーク](biology.md)を形成する菌類はネットワーク拡大のために胞子を広域に散布し、生態系の回復・拡張に寄与する。胞子雨による惑星[テラフォーミング](sf-concepts.md)（wiim_017）や、金星・タイタンへの生物気候工学（wiim_018）では、胞子の耐性と散布効率を宇宙規模で応用するアイデアが展開されている。

## ホモキラリティ（Homochirality）

**読み**: ほもきらりてぃ
**分野**: 生化学・宇宙生物学
**関連記事**: —

地球上の生命が、アミノ酸には左手型（L型）のみ、糖には右手型（D型）のみを使うという偏りの現象。鏡像関係にある左右の分子のうち、片方だけを選択的に使う。

化学的には左右どちらも同程度に生成されるはずなのに、なぜ地球生命は一方向に揃っているのかは未解明だ。宇宙線の偏極説、隕石からのホモキラル有機分子の持ち込み説などがある。

宇宙の別の星で独立に生命が生まれた場合、同じホモキラリティを持つかどうかは、その生命が地球生命と「同系」かどうかを判定する手がかりになりうる。

## ミュータント（Mutant）

**読み**: みゅーたんと
**別名**: 突然変異体 / 突然変異 / 変異体 / 変異種 / 変異
**分野**: 遺伝学・進化生物学・SF
**関連記事**: [wiim_008](../docs/biology/wiim_008.md)

遺伝子に変異（ミューテーション）を持つ生物の総称。変異の原因は放射線・化学物質・[ウイルス](biology.md)（g116）の挿入・DNA複製エラーなど多岐にわたる。変異の種類は塩基1個の置換（点変異）から染色体レベルの大規模な再編まで幅広く、タンパク質の機能変化・発現量の変化・まったく無影響など結果も様々だ。

変異は進化（g029）の原材料であり、自然選択によって有利な変異が集団に広まることで種が適応する。大半の変異は中立か有害だが、環境が変われば「有害」だった変異が「有利」に転じることもある——鎌状赤血球変異がマラリア地域では生存率を高める例が典型だ。

SFでは放射線などによって劇的な表現型変化を持つ個体（念動力・再生能力など）として描かれることが多いが、実際の変異は1塩基〜数塩基レベルの微細な変化であり、単一の変異が複雑な超能力を生むことはない。ただし**エピジェネティクス**（遺伝子の発現制御）や複数変異の相互作用は予測困難な表現型を生むことがあり、SF的想像の完全な否定もできない。

WhatIfImpossibleでは、宇宙放射線下での菌類の急速な変異（wiim_008）や、[マイコプラズマジカ](wiim-concepts.md)（wiim_024）における設計されたゲノム変異が関連する。
