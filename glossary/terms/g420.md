# ファスト・スクランブラー（Fast Scrambler）

**Fast Scrambler**  物理学 / 量子情報理論・量子重力 / g420
読み: ふぁすとすくらんぶらー　関連: —

入力された量子情報を最短時間で系全体に拡散・非局所化できる量子系の総称。スクランブリングにかかる最短時間は[熱力学](g250.md)的時間スケール（~ℏ/kBT）に対数補正を加えた値で与えられ、これを達成する系をファスト・スクランブラーと呼ぶ。

ブラックホールは究極のファスト・スクランブラーであるとSekinoとSusskind（2008）が提唱した。ホーキング温度と巨大なベッケンシュタイン–ホーキング[エントロピー](g008.md)を持ちながら、スクランブリング時間は地平線の光横断時間に対して対数補正程度にしか大きくならない。

量子[カオス](g179.md)的なユニタリ発展で記述され、OTOC（時間順序外相関関数）の急速な成長として計測できる。音響情報がブラックホールに入ると、このスクランブリングによって全ホーキング輻射に瞬時に分散される。
