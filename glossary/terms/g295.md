# アーベル群（Abelian Group）

**Abelian Group**  数学・論理・幾何学 / 抽象代数学・群論 / g295
読み: あーべるぐん　関連: [wiim_074](../../docs/physics/wiim_074.md)

交換法則（可換律）が成立する群。ノルウェーの数学者ニールス・ヘンリク・アーベル（1802–1829）にちなむ。

群とは「集合＋演算」の組み合わせで、①閉包性、②結合法則、③単位元の存在、④逆元の存在の4条件を満たすもの。アーベル群はこれに加えて⑤ a ∗ b = b ∗ a（どの2要素を演算しても順序によらず同じ結果になる）という交換法則が成り立つ。

整数の加法・実数の加法・時計の文字盤の数字（mod n の加法）などが典型例だ。一方、行列の乗法や空間の回転群（SO(3)）は交換法則が成り立たない「非アーベル群」となる。

WIIMの文脈では、[エニオン](g157.md)のブレイドがなぜ[位相](g242.md)的情報を蓄積できるかという問いに関わる。エニオンのうち「アーベル型エニオン」は粒子を交換する順序によらず同じ位相を蓄積するのに対し、「非アーベル型エニオン」は交換の順序が結果に影響する——これが [wiim_074](../../docs/physics/wiim_074.md) で論じた[トポロジカル](g158.md)接続の鍵となる。
