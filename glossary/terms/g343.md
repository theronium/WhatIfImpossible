# 虚数温度（Imaginary Temperature）

**Imaginary Temperature**  WIIM概念 / 負エネルギー熱力学 / g343
読み: きょすうおんど　関連: [ディラックサイフォン](g342.md)

負のエネルギー系に適用される[熱力学](g250.md)的概念。通常の温度は正のエネルギーの運動論的記述（β = 1/kT が実数）であるが、負エネルギー系の熱平衡状態を記述するには温度が虚数になると考えられる。

数学的にはWick回転（t → −iτ）や松原形式（有限温度場理論）と接続する。虚数温度状態ではβが虚数となり、熱平衡が確率分布ではなく量子振幅として振る舞う。これは「冷やすほど活発になる」反転した熱力学を意味する。

[ディラックサイフォン](g342.md)で抽出された負エネルギーは、この虚数温度状態にあることで自然に粘性ゼロの超流動的性質を持つと考えられる。さらにKSS境界（η/s ≥ ℏ/4πk_B）が虚数温度系では反転し、粘性が負になる（流れが自己加速する）可能性も示唆される。
