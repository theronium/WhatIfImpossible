# 両立論（Compatibilism）

**Compatibilism**  哲学・存在論 / 哲学・自由意志論 / g170
読み: りょうりつろん　関連: [自由意志](g018.md), [決定論](g165.md), [wiim_041](../../docs/logic/wiim_041.md), [wiim_047](../../docs/philosophy/wiim_047.md)

**別名**: コンパティビリズム / 自由意志の両立論

[自由意志](g018.md)と[決定論](g165.md)は矛盾しないという立場。「コンパティビリズム」とも呼ばれる。決定論が正しくても自由意志は成立しうる——なぜなら「自由」とは因果の外に出ることではなく、外部の強制なく自分の欲求や理性に従って行動できることだ、と再定義する。

伝統的な両立論はこの「言葉の再定義」で決定論と自由意志を和解させる。しかし近年の[情報理論](g251.md)・複雑系理論はより強い根拠を与える——人間の意思決定プロセスは[コルモゴロフ複雑性](g177.md)が最大に近く、決定論的に生成されていても外部から予測することが原理上不可能になる。「決まっているが解けない」系の内側では、選択は機能的に本物だ（[wiim_041](../../docs/logic/wiim_041.md)）。

対立する立場は「非両立論」で、自由意志には決定論からの真の逸脱が必要だとする強硬な立場と、決定論が正しければ自由意志は幻想に過ぎないとする立場に分かれる。自由意志（g???）・決定論（g165）と合わせて、哲学における[意志論](g234.md)の三つの柱をなす。
