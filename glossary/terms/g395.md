# ゼーベック効果・ペルティエ効果（Seebeck Effect / Peltier Effect）

**Seebeck Effect / Peltier Effect**  物理学 / 熱電工学・固体物理学 / g395
読み: ぜーべっくこうか・ぺるてぃえこうか　関連: [エクトロン](g394.md), [wiim_098](../../docs/physics/wiim_098.md)

温度差が電圧を生む「ゼーベック効果」と、電流が熱移動を生む「ペルティエ効果」の総称。どちらも熱エネルギー（[エントロピー](g008.md)流）と電気エネルギーの相互変換を担う熱電効果であり、オンサーガーの相反[定理](g304.md)によって対称的に結びついている。

ゼーベック効果は廃熱からの電力回収に、ペルティエ効果は可動部品なしに電場でエントロピー流を方向付けする半導体冷却素子（ペルティエ素子）に応用される。エントロピー流を電場で制御・整流するという点で、WIIMの思考実験上の概念である[エクトロン](g394.md)（g394）が想定するエクトロピア場の現実世界における類似例として位置づけられる。
