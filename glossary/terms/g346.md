# 有限温度場理論（Finite Temperature Field Theory）

**Finite Temperature Field Theory**  物理学 / 量子場理論・統計力学 / g346
読み: ゆうげんおんどばのりょうしろん　関連: [虚数温度](g343.md)

有限温度における量子場を記述する理論体系。松原（Matsubara）形式では時間をt → −iτと虚数化（Wick回転）することで、[量子力学](g253.md)の時間発展と統計力学の熱平衡が統一的に記述される。逆温度β = 1/kTの周期条件が時間軸に課されることで、温度が量子論に組み込まれる。

[虚数温度](g343.md)の概念はこの形式から自然に導かれる。βが虚数になると熱平衡が確率分布ではなく量子振幅として振る舞い、負エネルギー系の[熱力学](g250.md)を記述する候補として位置付けられる。高密度[クォーク](g114.md)・[グルーオン](g064.md)プラズマや初期宇宙の高温状態など、極端な温度環境の理論的記述にも用いられる。
