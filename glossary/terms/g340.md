# 極小曲面（Minimal Surface）

**Minimal Surface**  数学・論理・幾何学 / 微分幾何学・材料工学・生物構造 / g340
読み: きょくしょうきょくめん　関連: —

曲面上のすべての点で**平均曲率**がゼロとなる曲面。平均曲率とは曲面の一点における直交する二方向の主曲率の算術平均であり、正負が相殺されてゼロになるとき面は「鞍型」の均衡形状を取る。石鹸膜が枠に張られると自動的に極小曲面を形成するのはこの[原理](g306.md)による。

代表例として回転対称な[カテノイド](g335.md)や三次元周期構造を持つ[ジャイロイド](g330.md)・シュワルツP面が挙げられる。これらはいずれも平均曲率ゼロという一つの条件から導かれながら、まったく異なる幾何形態として現れる。

生体膜の膜融合中間体・多孔質骨格材料・建築薄殻構造など「最小の材料で最大の安定性・透過性」が求められる設計に自然に出現し、自然と工学が同じ数理的原理へ収斂する典型例とされる。
