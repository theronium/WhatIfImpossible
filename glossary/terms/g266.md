# 軌道計量汚染除去（Orbital Metric Pollution Remediation）

**Orbital Metric Pollution Remediation**  WIIM派生技術 / 架空環境管理・宇宙インフラ保守 / g266
読み: きどうけいりょうおせんじょきょ　関連: [wiim_022](../../docs/physics/wiim_022.md)

[アンキロン](g128.md)（g128）の大規模展開後に生じる「計量汚染」——空間計量テンソルの局所的乱れ——を検出・修復する架空の環境管理技術。[反アンキロン除去技術](g264.md)（g264）で捕捉した反粒子の[惑星](g123.md)圏外処理と、乱れた計量テンソルの再固着作[業](g206.md)からなる。

[計量暦システム](g265.md)（g265）の精度維持や[宇宙航法座標系](g267.md)（g267）の信頼性確保に直結するインフラ保守業務と位置づけられる。アンキロンが文明インフラの基盤になった段階では、計量汚染除去は電力網の保守や[衛星軌道](g077.md)管理に相当する社会的必須業務となる。
