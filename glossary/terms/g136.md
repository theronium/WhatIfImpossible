# アステロイドベルト（Asteroid Belt）

**Asteroid Belt**  天文学・宇宙論 / 太陽系・天体力学 / g136
読み: あすてろいどべると　関連: [wiim_025](../../docs/biology/wiim_025.md), [wiim_008](../../docs/biology/wiim_008.md)

**別名**: 小惑星帯 / 主小惑星帯

火星軌道と木星軌道の間（[太陽](g084.md)から約2.2〜3.2[AU](g095.md)）に位置する、無数の[小惑星](g123.md)が集中する帯状領域。主小惑星帯とも呼ばれる。総質量は月の約4%に過ぎず、映像作品のように密集しているわけではなく、平均的な小惑星間距離は数十万km以上ある。

構成天体の大半は珪岩質（S型）・炭素質（C型）・金属質（M型）の3種に分類される。特にC型小惑星は有機物・含水鉱物を豊富に含み、生命の原材料（アミノ酸・糖類の前駆体）の存在が隕石分析から示唆されている。

木星の重力による軌道共鳴がこの領域での惑星形成を妨げたと考えられており、太陽系形成期の原始的な物質が保存された「化石」的な領域だ。[シェルマイセリウム](g135.md)（g135）の思考実験では、C型小惑星の有機物資源と[ラグランジュ点](g049.md)L4/L5の重力安定性を組み合わせた「適地ゾーン」として位置づけられる。
