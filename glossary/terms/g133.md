# カシミールフォージ（Casimir Forge）

**Casimir Forge**  WIIM派生技術 / 架空装置・量子物理学・ワープ技術 / g133
読み: かしみーるふぉーじ　関連: [wiim_023](../../docs/physics/wiim_023.md), [wiim_063](../../docs/physics/wiim_063.md), [wiim_064](../../docs/physics/wiim_064.md), [wiim_065](../../docs/physics/wiim_065.md), [wiim_066](../../docs/physics/wiim_066.md), [wiim_074](../../docs/physics/wiim_074.md), [wiim_078](../../docs/cosmology/wiim_078.md), [wiim_085](../../docs/quantum/wiim_085.md), [wiim_086](../../docs/physics/wiim_086.md)

**別名**: QIE装置 / 量子不等式増幅装置 / エキゾチック物質生成装置

WhatIfImpossibleの思考実験（[wiim_023](../../docs/physics/wiim_023.md)）で命名された架空の装置。「[カシミール効果](g009.md)（Casimir effect）」と「鍛造（Forge）」を組み合わせた造語で、量子真空の揺らぎを増幅・同期させて[エキゾチック物質](g068.md)（g068）を実用規模で連続生成する装置を指す。記事内では「QIE（Quantum Inequality Enhancer：量子不等式増幅装置）」とも呼ばれる。

カシミール効果は2枚の金属板の間に微小な負のエネルギー密度を生み出すことが実証されているが、その量は[ワープ](g035.md)ドライブに必要な規模の遥かに下だ。カシミールフォージは多層の高精度カシミール共振器によって仮想粒子の[量子揺らぎ](g061.md)を同期・蓄積し、フォード＝ローマン不等式という量子的上限を突破することを仮定する。

生成されたエキゾチック物質はアルクビエレ・ドライブ（ワープドライブ g035）のバブル形成に使われ、船体周囲の空間を前方で収縮・後方で膨張させることで光速を超えずに超光速移動を実現する。駆動には[カルダシェフスケール](g002.md)・タイプII文明相当のエネルギー（[ダイソン球](g034.md) g034 規模）が必要とされる。[コスモシェル](g132.md)（g132）・[グラビトーペイク](g129.md)（g129）と並ぶWhatIfImpossibleの架空装置群の一つ。
