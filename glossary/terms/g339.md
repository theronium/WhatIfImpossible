# バブルシェルマイセリウム（Bubble Shell Mycelium）

**Bubble Shell Mycelium**  WIIM概念 / 架空生命体・宇宙生物学・泡構造工学 / g339
読み: ばぶるしぇるまいせりうむ　関連: [wiim_083](../../docs/biology/wiim_083.md), [wiim_084](../../docs/biology/wiim_084.md), [コズミックマイス](g134.md), [プラトーの法則](g338.md), [ルーネベルクレンズ](g334.md), [シェルマイセリウム](g135.md)

**別名**: BSマイセリウム / BSM

WIIMの思考実験（[wiim_084](../../docs/biology/wiim_084.md)）で命名された架空の多球コロニー構造体。「Bubble Shell（泡殻）」と「Mycelium（菌糸体）」を組み合わせた造語で、BS[マイセリウム](g112.md)・BSMと略される。

[コズミックマイス](g134.md)（g134）の球状コロニーを複数集合させた泡集合体であり、低重力環境で[プラトーの法則](g338.md)（g338）に従って自発的に面積最小化配置を取ると仮定される。個々の球は[wiim_083](../../docs/biology/wiim_083.md)で論じた疑似ルーネベルク構造（三層密度勾配による全方向集光体）を維持し、それらがプラトー境界を通じてネットワーク接続されることで[ハイヴマインド](g044.md)の接続基板が構造から自然発生する。

BSMは三つのスケールが入れ子になった階層構造を持つ。大域スケールでは正二十面体ベースのジオデシック分割がBSM全体の球形外形を規定し（[ウイルス](g116.md)カプシドやC₆₀が同じ[原理](g306.md)で最小遺伝子情報から最大構造強度を実現するのと同様）、中域スケールではプラトーの法則が隣接バブル間のH=0接触面を形成し、微域スケールでは各バブル内部の密度勾配が集光を担う。

[シェルマイセリウム](g135.md)（g135）が「1つの[コスモシェル](g132.md)に菌糸が共生する形態」であるのに対し、BSMは「複数の球状コロニーが泡力学で[自己組織化](g028.md)した超構造体」として位置づけられる。
