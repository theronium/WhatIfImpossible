# 余剰次元可塑体（Extra-dimensional Plastoid）

**Extra-dimensional Plastoid**  WIIM派生技術 / 余剰次元物理学・位相材料工学 / g437
読み: よじょうじげんかそたい　関連: [エネルギー紐](g131.md), [アンキロン](g128.md), [wiim_021](../../docs/physics/wiim_021.md), [wiim_022](../../docs/physics/wiim_022.md)

**別名**: Exプラスチック体 / ディムスパイラル

[余剰次元](g255.md)への巻き付き構造を利用した切断不可能な架空素材。WIIMで定義された造語。

引っ張ると巻き付きが解け始めるが、巻き解き前線の伝播速度は素材内の音速を超えられない。通常の破断を引き起こす「応力波が亀裂先端を追い越す」機構が成立せず、エネルギーは余剰次元の[位相](g242.md)変化として吸収される。[量子トンネリング](g097.md)による解きは[原理](g306.md)上可能だが、宇宙年齢を超える時定数で抑制される。

3次元から見ると可塑的に変形し続けるが、余剰次元から見ると巻き付きは保存されており弾性的に復元可能。[エネルギー紐](g131.md)（g131）が弾性的拘束であるのに対し、余剰次元可塑体は位相的粘弾性として機能する。
