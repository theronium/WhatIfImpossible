# オッカムの剃刀（Occam's Razor）

**Occam's Razor**  哲学・存在論 / 認識論・科学哲学 / g020
読み: おっかむのかみそり　関連: —

**別名**: 節約の原理 / 節倹の原理

「必要以上に多くのものを仮定してはならない」という思考の原則。14世紀の哲学者ウィリアム・オブ・オッカムに帰される。同じ現象を説明できる複数の仮説があれば、最も単純なものを選ぶべきという指針。

科学的方法の基礎として広く使われる一方、「最も単純な説明が真実とは限らない」という批判もある。宇宙は人間の直感より複雑である場合が多い（量子力学・[一般相対性理論](g108.md)はいずれも非直感的な理論だ）。
