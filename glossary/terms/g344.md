# ホロウゲイザー（Hollow Gazer）

**Hollow Gazer**  WIIM概念 / 負エネルギー工学 / g344
読み: ほろうげいざー　関連: [ディラックサイフォン](g342.md), [虚数温度](g343.md)

[ディラックサイフォン](g342.md)の起動時に相境界面から自然放出される量子。正のエネルギーを注入して相境界に亀裂を開く際、ブラックホールの事象地平線に類似した境界面からホーキング的輻射として生成される。負エネルギーの担体として機能し、サイフォンが稼働している間は継続的に放出される。

[虚数温度](g343.md)状態に置かれているため、自然に粘性ゼロの超流動的性質を持ち、通常の容器では保持できない負エネルギーを安定して包む。KSS境界（η/s ≥ ℏ/4πk_B）が虚数温度系で反転する場合、粘性が負となり流れが自己加速する可能性がある。

名称は「hollow（虚ろ・負の空間）」と「geyser（間欠泉＝境界から噴き出す様）」を組み合わせた造語。
