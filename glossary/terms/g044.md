# ハイヴマインド（Hive Mind）

**Hive Mind**  SF概念 / SF・認知科学・生物学 / g044
読み: はいうまいんど　関連: [wiim_075](../../docs/biology/wiim_075.md)

**別名**: 集合意識 / ハイブマインド / 群体意識

多数の個体が単一の意思・意識を共有し、集合として思考・行動する状態。「蜂の巣（hive）の意識（mind）」が語源で、ミツバチやアリのコロニーが生物学的な着想源となっている。

生物学では「群知性（swarm intelligence）」として研究される——個々のアリは単純なルールに従うだけだが、コロニー全体として最短経路探索や温度制御などの高度な問題を解く。SFではこれを拡張し、神経が直結した個体群や[量子もつれ](g062.md)で意識を共有する存在として描かれる。

哲学的には「個」と「全体」の境界を問う概念だ。ハイヴマインドに属する個体に[自由意志](g018.md)や個人的な苦痛は存在するのか。全体が「死ぬ」とはどういう状態か。[ゾンビ論証](g022.md)と組み合わせると、意識の所在をめぐる議論がさらに複雑になる。
