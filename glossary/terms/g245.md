# コスモライケン（Cosmo-Lichen）

**Cosmo-Lichen**  WIIM概念 / 架空生命体・宇宙生物学・テラフォーミング / g245
読み: こすもらいけん　関連: [wiim_043](../../docs/biology/wiim_043.md), [wiim_008](../../docs/biology/wiim_008.md), [wiim_025](../../docs/biology/wiim_025.md), [wiim_011](../../docs/physics/wiim_011.md)

**別名**: 宇宙ゴケ共生体 / リケン・ステラリス / 自律型テラフォーミング艦 / 四層共生体

WhatIfImpossibleの思考実験（[wiim_043](../../docs/biology/wiim_043.md)）で命名された架空の多層共生生命体。「Cosmo（宇宙的）」と「Lichen（地衣類）」を組み合わせた造語で、学名はリケン・ステラリス（Lichen Stellaris）。地衣類・[シアノバクテリア](g188.md)・[コズミックマイス](g134.md)[菌糸](g112.md)・[コスモシェル](g132.md)の四層構造からなる自律型[テラフォーミング](g042.md)艦だ。

コズミックマイス（g134）が分散知性と移動能力を持ちながらエネルギー自給手段を欠くのに対し、コスモ[ライケン](g187.md)はシアノバクテリアの光合成層を統合することで[恒星](g084.md)間空間での完全自給を実現する。菌糸網が軌道修正・[惑星](g123.md)評価・着地判断を担い、着地後は耐性地衣類が岩盤侵食と土壌形成を開始する。この「惑星を選ぶ意図」こそが従来の地衣類との本質的な違いだ。

もし地球の大酸化イベント（約27億年前）がコスモライケンの祖先によるテラフォーミングの結果であるならば、意図なき惑星改造はすでに一度起きたことになる。
