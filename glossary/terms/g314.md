# アンキロンピン（Ankyron Pin）

**Ankyron Pin**  WIIM派生技術 / 架空計量操作工学 / g314
読み: あんきろんぴん　関連: [wiim_077](../../docs/physics/wiim_077.md), [wiim_022](../../docs/physics/wiim_022.md)

WhatIfImpossibleの思考実験（[wiim_077](../../docs/physics/wiim_077.md)）で提案された架空の操作機構。[光ピンセット](g313.md)が光強度勾配で粒子を捕捉・移動するように、計量テンソルの変化率勾配を局所的に操作することで[アンキロン](g128.md)（g128）を配置・剥離・微調整する工具として定義される。

計量の「谷」構造を生成してアンキロンを特定座標へ誘導し、逆に計量変化率をゼロに抑える操作を加えることで固着力を弱めて剥離できる。アンキロン自体は計量座標を固定するのみで物質を直接捕捉する能力を持たないため、アンキロンピンで固定した座標を[コーラ粒子](g127.md)（g127）などが参照するという間接的な組み合わせとして機能する。[トポロフィピン](g315.md)（g315）と組み合わせることでコーラ粒子の出現先をミクロ単位で二重拘束する精密制御設計の起点となる。
