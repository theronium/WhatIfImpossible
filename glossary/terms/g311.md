# 不確定性原理（Uncertainty Principle）

**Uncertainty Principle**  量子物理学 / 量子力学・量子情報 / g311
読み: ふかくていせいげんり　関連: [ラプラスの悪魔](g204.md), [意識のハードプロブレム](g169.md)

1927年にヴェルナー・ハイゼンベルクが定式化した[量子力学](g253.md)の基本[原理](g306.md)。粒子の位置と運動量を同時に任意の精度で確定することは原理上不可能であり、一方を精密に測るほど他方の不確定さが増大する。これは測定器の精度の問題ではなく、自然界の構造に由来する根本的な限界だ。

数式では Δx・Δp ≥ ℏ/2 と表され、位置の不確定量 Δx と運動量の不確定量 Δp の積はプランク定数の半分以上になることを示す。同様に、エネルギーと時間の間にも ΔE・Δt ≥ ℏ/2 の関係が成り立つ。

[ラプラスの悪魔](g204.md)（g204）を封じる第一の論拠がこれだ。全粒子の位置と運動量を同時に確定することは原理上不可能であり、古典的[決定論](g165.md)の前提が量子スケールで崩壊する。また観測行為そのものが系を乱すという量子測定の問題とも深く結びついており、[意識のハードプロブレム](g169.md)（g169）と「誰が観測するのか」という問いで交差する。
