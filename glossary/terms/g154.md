# 宙域資産権（Spatial Domain Asset Rights）

**Spatial Domain Asset Rights**  WIIM概念 / 宇宙法・架空経済・空域所有権 / g154
読み: ちゅういきしさんけん　関連: —

**別名**: 宙域権 / 宙権

コズミックボイド（[銀河](g123.md)群外の宇宙膨張が顕著な空域）に対して設定される架空の所有権概念。正式名称は宙域資産権、略称は宙域権または宙権。地上権・漁[業](g206.md)権と同じ命名構造を持ち、「地面ではなく宙域を保有する権利」として設計された。

既存の不動産概念は適用できない。「不動産（動かない資産）」という語義に反し、宙域は宇宙膨張によって境界が常に外側へ移動し続けるからだ。また共動座標（膨張と一緒に伸びる座標系）と固有座標（物理的実距離）のどちらで区画を定義するかによって、膨張利回りが発生するかどうかが根本的に変わる——この座標定義問題は法整備以前の物理的難問だ。

制度発足当初は銀河群の[重力圏](g107.md)を脱した空域に限定されていたが、FTLインフラの整備が進み銀河重力圏内でも権益争いが激化したため、後の法改正によって適用範囲が拡大された。FTLインフラ（[ワープ](g035.md)ゲート網など）が整備されて初めてアクセス可能になることから、ゲート設置予定地の近傍空域を巡る投機と詐欺（宙権詐欺）が定番の手口として社会問題化している。登記・管轄機関をどの主体が担うかは星間条約上の未解決問題でもある。
