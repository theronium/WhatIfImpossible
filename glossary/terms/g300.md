# 三体問題（Three-body problem）

**Three-body problem**  物理学 / 古典力学・天体力学 / g300
読み: さんたいもんだい　関連: —

万有引力で相互作用する3つの[天体](g123.md)の運動を求める問題。2体問題（ケプラー問題）は解析的に解けるが、3体以上になると一般解が存在せず、数値積分でのみ近似できる。この「解析解の不在」はアンリ・ポアンカレが19世紀末に証明し、[カオス理論](g179.md)の先駆けとなった。

三体系の軌道はわずかな初期条件の差で指数関数的に乖離し、長期予測が[原理](g306.md)的に不可能になる（[初期値鋭敏性](g178.md)）。[太陽](g084.md)系の安定性問題や、連星に第三の天体が加わった系の軌道解析など、天体力学の根幹を成す問題である。特殊解（[ラグランジュ点](g049.md)・[オイラー](g229.md)解・コレオグラフィ解など）は有限個だけ存在し、探索が続いている。
