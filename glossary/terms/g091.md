# リーマン予想（Riemann Hypothesis）

**Riemann Hypothesis**  数学・論理・幾何学 / 数論・解析学・科学哲学 / g091
読み: りーまんよそう　関連: —

**別名**: リーマン仮説

リーマンゼータ関数とは、[複素数](g104.md) s に対して ζ(s) = 1 + 1/2^s + 1/3^s + 1/4^s + … と定義される関数だ（実部が1より大きい領域で収束し、それ以外は解析接続で定義を広げる）。この関数が 0 になる点（零点）のうち、-2, -4, -6, … といった負の偶数整数は「自明な零点」と呼ばれる。問題は「自明でない零点」——複素平面上に無数に存在するとされる点——がすべて実部 1/2 の直線（臨界線）上に並ぶかどうか、という問いだ。

1859年にドイツの数学者ベルンハルト・リーマンが提唱した未解決の予想。「リーマンゼータ関数の自明でないすべての零点は、実部が 1/2 の直線上にある」というものだ。数学における最重要未解決問題の一つとされ、クレイ数学研究所の[ミレニアム懸賞問題](g102.md)にも選ばれている（賞金100万ドル）。

素数の分布と深く結びついており、リーマン予想が正しければ素数がどのように散らばっているかについて非常に精密な記述が得られる。逆に言えば、宇宙の素数配置には「見えない秩序」があると示唆する。

[ゲーデルの不完全性定理](g090.md)（g090）と並んで「数学の限界」を語る文脈で頻繁に引用される。現行の公理系の中では証明も反証もできない可能性すら指摘されており、数学・哲学・物理学（量子[カオス理論](g179.md)との関連も指摘される）にまたがる謎として21世紀も議論が続く。

ゼータ関数の用途は広い。数論では素数計数関数 π(x)（x以下の素数の個数）の精密な近似を与え、RSA暗号の理論的土台である素数分布の記述に直結する。物理学では量子カオス研究（[原子核](g114.md)のエネルギー準位の統計がゼータ関数の零点間隔と一致することが知られる）や[カシミール効果](g009.md)の計算、[弦理論](g093.md)での発散の正則化にも登場する。「1+2+3+…= -1/12」という一見奇妙な結果も解析接続による正則化の産物だ。一言でいえば「不規則に見える素数の背後に潜む隠れた秩序を記述する関数」であり、純粋数学から暗号・量子物理まで横断的に活躍する。
