# フェルマーの最終定理（Fermat's Last Theorem）

**Fermat's Last Theorem**  数学・論理・幾何学 / 整数論・数論 / g309
読み: ふぇるまーのさいしゅうていり　関連: [定理](g304.md), [フェルマーの原理](g297.md)

「n が3以上の整数のとき、x^n + y^n = z^n を満たす正の整数の組 (x, y, z) は存在しない」という[定理](g304.md)（g304）。1637年にピエール・ド・フェルマーが余白に「驚くべき証明を発見したが、余白が狭すぎて書けない」と書き残したことで有名になり、以来358年間未解決のまま数学史上最大の難問とされた。

1995年、アンドリュー・ワイルズが楕円曲線・モジュラー形式・ガロア表現を組み合わせた200ページ超の証明を発表し、解決された。フェルマー自身が想定した「初等的な証明」は現代でも発見されておらず、本当に存在したかは謎のままである。なお同じフェルマーによる[フェルマーの原理](g297.md)（g297）は光学の[原理](g306.md)であり、全く別の概念である。
