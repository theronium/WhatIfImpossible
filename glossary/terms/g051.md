# コリオリの力（Coriolis Force）

**Coriolis Force**  物理学 / 古典力学・流体力学 / g051
読み: こりおりのちから　関連: —

**別名**: コリオリ効果 / コリオリ力

回転する座標系（非慣性系）の中で運動する物体に働くように見える見かけの力。静止した慣性系から見れば直線運動をしている物体が、回転する観測者の目には進行方向に対して横向きに曲げられているように見える現象として現れる。1835年にフランスの物理学者ガスパール＝ギュスターヴ・コリオリが数学的に定式化した。

地球上では自転が回転系に相当するため、大気や海流に顕著に現れる。北半球では進行方向の右側に、南半球では左側に曲げられるため、低気圧は北半球では反時計回り（左旋）、南半球では時計回り（右旋）のサイクロンとなる。台風・ハリケーン・温帯低気圧の渦巻き方向を決定づける要因の一つでもある。

コリオリの力は実際には力ではなく、座標系の選び方から生まれる幾何学的効果であるため「仮想力」または「見かけの力」とも呼ばれる。大規模な気象現象や弾道ミサイルの軌道計算では無視できないが、風呂の排水のような日常スケールではほぼ無視できる。
