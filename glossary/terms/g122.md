# エイリアン（Extraterrestrial Life）

**Extraterrestrial Life**  生物学・進化 / 宇宙生物学・天文学・SF / g122
読み: えいりあん　関連: [wiim_025](../../docs/biology/wiim_025.md)

**別名**: 地球外生命体 / 宇宙人 / 地球外知的生命体 / 異星人 / ET / 異星生命

地球以外の[天体](g123.md)に起源を持つ生命体の総称。科学的文脈では**地球外生命体**（宇宙生物学の対象）、SF的文脈では知性を持つ**異星人**を指すことが多い。現在のところ地球外生命の存在は確認されていないが、宇宙の広大さと惑星の数を考えれば存在しない方が不自然だとする見方も根強い（[フェルミのパラドックス](g037.md) g037）。

宇宙生物学が最も可能性が高いとするのは微生物レベルの生命で、火星の地下・エウロパやエンケラドゥスの海洋・タイタンの液体メタン湖などが候補環境とされる（[wiim_018](../../docs/biology/wiim_018.md)）。[極限環境微生物](g030.md)（g030）の研究は、生命が想像以上に多様な条件に適応できることを示しており、炭素・水ベース以外の生化学（ケイ素・アンモニア溶媒など）の可能性も議論される。

知的エイリアンについては、[SETI](g094.md)（g094）が電波・光シグナルによる探索を続けているが成果は得られていない。この沈黙はフェルミの[パラドックス](g113.md)（g037）として知られ、「知的文明は[ワープ](g035.md)技術到達前に自滅する」「意図的に沈黙している」「我々の観測方法が間違っている」など多くの解釈を生む。WhatIfImpossibleの記事群はこの問いへの思考実験的アプローチでもある。
