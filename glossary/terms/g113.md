# パラドックス（Paradox）

**Paradox**  哲学・存在論 / 論理学・哲学・数学・物理学 / g113
読み: ぱらどっくす　関連: —

**別名**: 逆説 / 矛盾命題

一見正しい前提から論理的に導かれるにもかかわらず、直感や常識に反する結論、あるいは自己矛盾に至る命題や状況。「パラドックス」はギリシャ語の「para（反して）」と「doxa（意見）」に由来する。

パラドックスは大きく3種類に分類できる。**真のパラドックス**は前提や論理に欠陥があるため矛盾が生じるもの（解決可能）。**二律背反**は前提が真でも互いに矛盾する結論が導かれるもの（カントの二律背反など）。**未解決のパラドックス**は解決策が見つかっていないもの。

物理・宇宙論の文脈では、[タイムパラドックス](g039.md)（g039）・[フェルミのパラドックス](g037.md)（g037）・EPRパラドックス（[量子もつれ](g062.md) g062）・[マクスウェルの悪魔](g059.md)（g059）・[シュレーディンガーの猫](g083.md)（g083）・[ボルツマン脳](g040.md)（g040）など多くの有名なパラドックスが存在する。数学では[ゲーデルの不完全性定理](g090.md)（g090）が「証明も反証もできない命題が存在する」という論理的パラドックスを厳密に定式化した。

WhatIfImpossibleの思考実験は本質的に「パラドックスの解消を試みる」営みとも言える——不可能に見える前提から何が導かれるかを探ることで、現実の物理法則の輪郭が浮かび上がる。
