# リングワールド（Ringworld）

**Ringworld**  SF概念 / SF・宇宙工学 / g041
読み: りんぐわーるど　関連: —

SF作家ラリー・ニーヴンが1970年の小説『リングワールド』で描いた巨大構造物。[恒星](g084.md)の周囲に巨大な環（リング）状の居住可能な帯を配置し、その内側の面に人が住む構想。

[ダイソン球](g034.md)の「シェル型」を薄いリング状に変えた形で、直径は地球軌道と同程度（約3億km）、幅は数百万km。リングを高速回転させることで遠心力が重力の代わりを果たす。居住可能な表面積は地球の数百万倍に達する計算になる。

ただし物理的には多くの問題がある。リングは重力的に不安定（微小なずれが増幅して恒星に落下しかねない）、材料強度は現在知られるいかなる物質をも超える「スクリスの素材」が必要、リム壁（縁の壁）による昼夜サイクルの人工的な再現が必要など。ダイソン球と並んで「[メガストラクチャー](g079.md)（超巨大構造物）」の代表例として宇宙工学・SFの議論に頻繁に登場する。
