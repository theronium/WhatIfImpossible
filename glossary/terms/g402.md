# 日震学（Helioseismology）

**Helioseismology**  天文学・宇宙論 / 太陽物理学・観測天文学 / g402
読み: にっしんがく　関連: —

太陽内部を伝播する音響波（圧力波）を利用して太陽の内部構造を研究する学問。地球内部を地震波で調べる地震学と同じ[原理](g306.md)を太陽に適用したもので、「日震（にっしん）」は太陽で起きる振動現象を指す。

太陽表面は常に数百万箇所で同時に微小な振動を起こしており、5分周期の振動（p-mode）が代表的である。これらは太陽内部で発生した音響波が表面に現れたものであり、その振動パターンを詳細に解析することで、表面からは直接見えない対流層・放射層・核の境界の位置や自転速度プロファイルを推定できる。

1995年に打ち上げられたSOHO[衛星](g123.md)の連続観測により、太陽内部の自転が緯度によって大きく異なること（差動自転）や、対流層の深さが約0.71太陽半径であることが精密に測定された。同様の手法を他の[恒星](g084.md)に適用する「[星震学](g329.md)（asteroseismology）」も盛んに行われている。
