# 負の質量（Negative Mass）

**Negative Mass**  仮説・未観測の粒子 / エキゾチック物質・理論物理学 / g072
読み: ふのしつりょう　関連: [エキゾチック物質](g068.md)

**別名**: 負質量 / ネガティブマス

質量が負の値を持つと仮定される仮説上の物質。通常の物質は正の質量を持ち、力を加えると加えた方向に加速する（ニュートンの第二法則: F = ma）。負の質量を持つ物体に力を加えると、逆方向に加速するという直感に反した性質を示す。

特に奇妙なのが「暴走運動（runaway motion）」だ。正の質量を持つ物体Aと負の質量を持つ物体Bを近づけると、Aは負の質量Bに引力を感じてBへ向かって加速し、BはAから斥力を受けてAから逃げようとする。結果として両者がエネルギーを消費せずに同じ方向へ永久に加速し続けるという、エネルギー保存則との矛盾を引き起こす。

[一般相対性理論](g108.md)の方程式は負の質量を数学的に禁じていないが、エネルギー条件（物理法則の基本的制約）を破るため自然界での存在は否定的に見られている。2017年にワシントン州立大学が「負の有効質量」を示すボース＝アインシュタイン凝縮体を生成したと発表したが、これは系の振る舞いが等価的に負の質量のように見えるもので、真の負の質量物質ではない。[エキゾチック物質](g068.md)（g068）の一種として議論される。
