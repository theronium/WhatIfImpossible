# コンドライト（Chondrite）

**Chondrite**  天文学・宇宙論 / 惑星科学・隕石学・宇宙化学 / g110
読み: こんどらいと　関連: —

**別名**: コンドリュール隕石 / 石質隕石

[太陽](g084.md)系で最も原始的な隕石の一種。約46億年前の太陽系形成初期に凝縮・合体した岩石物質をほぼそのまま保存しており、「太陽系のタイムカプセル」と呼ばれる。地球に落下する隕石の約85〜90%がコンドライトに分類される。

最大の特徴は「コンドリュール」と呼ばれる直径0.1〜3mm程度の球状組織を含むことだ。コンドリュールはかつて溶融した液滴が急冷して固まったものとされるが、何が熱源だったかは未解明で、衝撃波説・閃光加熱説などが議論されている。

コンドライトには炭素質コンドライト（Cコンドライト）・普通コンドライト・エン[スタタイト](g351.md)コンドライトなどの種類がある。特に炭素質コンドライトはアミノ酸・有機物・含水鉱物を含むものがあり、生命の材料物質が宇宙から地球にもたらされたとする[パンスペルミア](g007.md)仮説の文脈でも注目される。[小惑星](g123.md)[探査機](g076.md)「はやぶさ2」が持ち帰ったリュウグウのサンプルも炭素質コンドライトに近い組成を持つことが確認されている。
