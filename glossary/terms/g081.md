# スペースコロニー（Space Colony）

**Space Colony**  SF概念 / SF・宇宙工学・居住技術 / g081
読み: すぺーすころにー　関連: [ラグランジュポイント](g049.md), [メガストラクチャー](g079.md)

**別名**: 宇宙コロニー

宇宙空間に建設された人工的な居住施設。[惑星](g123.md)や月への移住とは異なり、重力・気圧・日照をすべて人工的に制御する閉鎖生態系を持つ。1970年代にNASAのジェラルド・オニールが提案した「O'ニール・シリンダー」が代表的な設計案で、直径数kmの円筒を自転させて遠心力による擬似重力を生み出す構造だ。

O'ニール・シリンダーは2本の円筒を逆回転させて角運動量を相殺し、間に鏡で反射した[太陽](g084.md)光を取り込む窓を設ける。内壁に土地・海・大気を配置し、地球に近い環境を実現できるとされる。NASAの1975年のスタンフォード・トーラス研究でも同様の概念が検討され、[ラグランジュポイント](g049.md)（g049）のL4・L5が建設候補地として挙げられた。

SFでは機動戦士ガンダムのスペースコロニー群（サイド）が日本で広く知られ、コロニー落としなどの概念も定着した。アーサー・C・クラークの『楽園の泉』やザ・エクスパンスのベルトのステーションなども同様の設定を持つ。ステラリスではHabitatとして軌道上に建設できる中規模居住施設として実装されている。

現実の課題は建設コスト・宇宙線遮蔽・心理的閉塞感・生態系の長期安定性など多岐にわたるが、地球の資源限界や惑星保護の観点から真剣に研究される機運が高まっている。
