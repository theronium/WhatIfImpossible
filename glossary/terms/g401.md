# ソノルミネッセンス（Sonoluminescence）

**Sonoluminescence**  物理学 / 音響物理学・非線形力学 / g401
読み: そのるみねっせんす　関連: —

液体中に閉じ込めた気泡に音波を照射すると、気泡が急激に収縮・崩壊する際に光を発する現象。音のエネルギーが光へと変換されるという点で直感に反する。気泡内部の圧力・温度は収縮時に極限まで上昇し、一時的に数千〜数万ケルビンに達すると推定されている。

発光の持続時間は数百ピコ秒以下と極めて短く、発光スペクトルは広帯域の連続光（白色光に近い）となる。発光機構については衝撃波説・熱放射説・量子真空説など複数の仮説が競合しており、完全には解明されていない。単一気泡を安定的に振動させる「シングルバブル・ソノルミネッセンス」は高い再現性を持ち、精密な実験が可能である。

極端に強い超音波を照射した場合に核融合反応が起きるとする「音響核融合（ソノフュージョン）」の主張が2002年に発表されたが、独立した再現実験には成功しておらず、現時点では否定的な評価が主流である。
