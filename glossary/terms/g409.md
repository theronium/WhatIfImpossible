# アルフベン波（Alfvén Wave）

**Alfvén Wave**  物理学 / 磁気流体力学・宇宙プラズマ物理 / g409
読み: あるふべんは　関連: —

磁化プラズマ中を磁力線に沿って伝播する横波。磁気張力が復元力として働き、磁力線がギターの弦のように振動する。1942年にスウェーデンの物理学者ハンネス・アルフベンが予言し、1970年のノーベル物理学賞の主な受賞理由となった。伝播速度（アルフベン速度）は磁場強度Bと流体密度ρで決まり、$v_A = B/\sqrt{\mu_0 \rho}$ で表される。

圧縮を伴わない純粋な横波であるため、[磁気音響波](g408.md)とは異なる独立したモードを構成する。宇宙では[太陽](g084.md)コロナ加熱の有力候補として研究が活発で、光球面で生成されたアルフベン波が磁力線を伝ってコロナへエネルギーを運ぶと考えられている。太陽風・[惑星](g123.md)磁気圏・星間媒質にも広く存在し、[SASER](g400.md)（音響レーザー）の[フォノン](g365.md)増幅と概念的に対応する磁気的な集団振動モードといえる。
