# 超弦理論（Grand Unified Theory / Superstring Theory）

**Grand Unified Theory / Superstring Theory**  物理学 / 理論物理・素粒子物理・量子重力 / g093
読み: ちょうひもりろん　関連: —

**別名**: 超弦理論 / 弦理論 / ひも理論 / GUT / M理論 / 大統一理論 / 超ひも理論

自然界の基本的な力を一つの数式で記述しようとする、現代物理学における最大の挑戦。どちらも未証明の仮説である。

**大統一理論（GUT: Grand Unified Theory）**：電磁力・弱い力・強い力の3つを統一しようとする理論群。標準模型はこの3力をある程度統合しているが、重力だけが仲間外れのままだ。GUTは重力を除く3力の統一を目指し、多くの候補理論が[陽子](g114.md)崩壊を予言するが、観測されたことがない。

**超弦理論（超ひも理論）**：素粒子を点ではなく「1次元のひも」として扱うことで、[量子力学](g253.md)と重力を含む4つの力すべてを統一しようとする理論。10〜11次元の[余剰次元](g255.md)を必要とし、数学的な整合性は非常に高い。しかし検証に必要なエネルギースケールは現在の[LHC](g074.md)（g074）の能力を遥かに超えており、理論の解（真空候補）が10の500乗個以上存在するため「どれが現実の宇宙か」を選ぶ原理がない。反証可能性を持たないという批判も根強い。

両理論に共通するのは「数学的に美しく整合的でも、物理的に正しいとは限らない」という問題だ。これは[ゲーデルの不完全性定理](g090.md)（g090）の構図——矛盾のない体系が現実を完全に記述するとは限らない——と本質的に通じる。[量子力学の未解決問題](g092.md)（g092）の中核に位置する課題でもある。
