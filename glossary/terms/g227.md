# ベルヌーイの定理（Bernoulli's Principle）

**Bernoulli's Principle**  物理学 / 流体力学 / g227
読み: べるぬーいのていり　関連: —

流体の速度・圧力・高さの関係を表す保存則。理想流体（粘性なし・非圧縮）が定常流れをするとき、流線に沿って次の式が成立する：P + ½ρv² + ρgh = 一定。圧力 P、密度 ρ、流速 v、重力加速度 g、高さ h の関係を示す。1738年にダニエル・ベルヌーイが導出した。

直感的には「流体が速く流れるほど圧力が下がる」。飛行機の揚力（翼上面で流速が速く低圧になる）、噴霧器の霧化、ベンチュリ管による流量測定など工学への応用は広い。

[ネイピア数](g226.md) e（g226）を発見したヤコブ・ベルヌーイとは異なり、ダニエルはヤコブの甥にあたる。[ベルヌーイ一族](g228.md)はスイスの数学者・物理学者の家系で、17〜18世紀の数学・力学の発展に複数世代にわたって貢献した。
