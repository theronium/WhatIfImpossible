# アニーリング（Annealing）

**Annealing**  物理学 / 統計力学・情報理論 / g449
読み: あにーりんぐ　関連: [wiim_052](../../docs/physics/wiim_052.md), [wiim_108](../../docs/physics/wiim_108.md)

**別名**: 焼きなまし法 / シミュレーテッドアニーリング / 量子アニーリング

金属を加熱後ゆっくり冷やし[原子](g114.md)を最低エネルギーの結晶格子へ再配列させる冶金技術「焼きなまし」を最適化問題に転用した手法。古典的な焼きなまし法（1983年提案）は探索中の「温度」を徐々に下げ、悪化する遷移もメトロポリス基準で確率的に許容して局所最適から脱出させる。量子アニーリングは熱によるくぼみの飛び越えを[量子トンネリング](g097.md)による障壁の直接貫通に置き換えた量子計算版である。

しかしどちらの方式も大域最適解の保証には現実的でない代償を要する——古典では対数的に遅い冷却スケジュール、量子では難問題ほど指数的に長い断熱時間。揺らぎの源を熱から量子に変えても計算複雑性は消えない。

この構図は、[カオスの悪魔](g210.md)（g210）が「完全制御」を「確率分布の精密な予測」へ引き下げて実現可能性を得たことと対応する。アニーリングはその経路の一つだが、悪魔の予測不能性——[バタフライ効果](g178.md)（g178）に類比できる困難——を消したのではなく、別の形に移し替えて引き継ぐと考えられる。
