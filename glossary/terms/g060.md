# パウリの排他原理（Pauli Exclusion Principle）

**Pauli Exclusion Principle**  量子物理学 / 量子力学・素粒子物理学 / g060
読み: ぱうりのはいたげんり　関連: —

**別名**: パウリの排他律 / 排他原理 / 排他律

同一の量子系において、二つ以上のフェルミオン（スピンが半整数の粒子）が完全に同じ量子状態を占めることはできないという[原理](g306.md)。1925年にオーストリアの物理学者ヴォルフガング・パウリが提唱した。

[電子](g064.md)はフェルミオンの代表例で、原子の中では同じエネルギー準位・スピン状態に2個までしか入れない。この制約が元素ごとに異なる電子配置を生み出し、化学的性質の多様性の根源となっている。つまり周期表の構造はパウリの排他原理なしには存在しない。

巨視的スケールでも重要な役割を果たす。[白色矮星](g123.md)や[中性子星](g084.md)が自己重力で無限に収縮せず一定の大きさを保つのは、それぞれ電子・[中性子](g114.md)の[縮退](g281.md)圧（パウリ圧力）によるものだ。排他原理を破ると仮定すると、すべての電子が最低エネルギー状態に落ち込み、物質の構造は崩壊する——固体も液体も存在できなくなる。
