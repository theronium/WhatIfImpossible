# 質量ゼロの粒子（Massless Particle）

**Massless Particle**  素粒子・核物理学 / 素粒子物理学・特殊相対性理論 / g065
読み: しつりょうぜろのりゅうし　関連: [ローレンツ不変性](g063.md), [wiim_020](../../docs/physics/wiim_020.md)

静止質量がゼロの素粒子。質量を持つ粒子は理論上「静止した状態」が存在できるが、質量ゼロの粒子を止めると運動量もエネルギーもゼロになり存在できなくなる。そのため**必ず真空中を光速 c で伝播し続けるしかない**という構造的必然がある。これは[ローレンツ不変性](g063.md)から導かれる帰結だ。

現在確認されている質量ゼロの粒子は以下の2種類。**光子（フォトン）**は電磁力を媒介するボソンで、可視光・電波・X線などすべての電磁波の量子。**[グルーオン](g114.md)**は強い核力を媒介するボソンで、クォークをつなぎとめてハドロン（陽子・中性子など）を構成する。

理論上は**重力子（[グラビトン](g109.md)）**も質量ゼロのボソンとして予測されているが、[重力波](g004.md)の量子として未だ直接検出されていない。

一方、[ニュートリノ](g064.md)はかつて質量ゼロと考えられていたが、ニュートリノ振動の観測によって極めて小さいながらも質量を持つことが判明している。質量ゼロかどうかの境界は「光速で伝播するか否か」「時間の経過を経験するか否か」という根本的な差異をもたらす。
