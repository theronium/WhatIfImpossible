# グラビトーペイク（Gravitopaque）

**Gravitopaque**  WIIM概念 / 架空素材・重力波物理学 / g129
読み: ぐらびとーぺいく　関連: [wiim_003](../../docs/physics/wiim_003.md), [wiim_010](../../docs/physics/wiim_010.md), [wiim_063](../../docs/physics/wiim_063.md)

**別名**: 重力的不透明体 / 重力波遮蔽材 / 重力波格子素材

WhatIfImpossibleの思考実験（[wiim_010](../../docs/physics/wiim_010.md)）で命名された架空の素材。「Gravitational（重力）」と「Opaque（不透明）」を組み合わせた造語で、[重力波](g004.md)を遮断・散乱・吸収できる「重力的に不透明な」物質の総称として定義される。

構造としては、[ネゴトン](g126.md)（g126）を規則的な格子状に配列した「重力波格子」が想定される。通常の物質は重力波と結合定数が電磁力の約40桁小さいため事実上透明だが、グラビトーペイクは重力波と異常に強く結合することで遮蔽を実現すると仮定する。

球殻状に形成すれば内部を「重力的に静寂な空間」にできる一方、副作用としてネゴトンの反重力効果により内外で時間の流れにずれが生じる。また重力波を反射モードで使えば、[ワープ航法](g035.md)（g035）のシグネチャを「霧の中に消す」重力波スモークスクリーンとしても機能する（[wiim_009](../../docs/cosmology/wiim_009.md)との対比）。[クロノスフィア](g125.md)（g125）・[アンキロン](g128.md)（g128）と並ぶWhatIfImpossibleの架空素材・架空粒子群の一つ。
