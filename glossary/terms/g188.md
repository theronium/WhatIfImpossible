# シアノバクテリア（Cyanobacteria）

**Cyanobacteria**  生物学・進化 / 微生物学・進化生物学 / g188
読み: しあのばくてりあ　関連: [内共生説](g185.md), [地衣類](g187.md), [wiim_043](../../docs/biology/wiim_043.md)

**別名**: 藍藻 / 藍色細菌

光合成を行う原核生物。約27億年前に地球上に出現し、酸素を大気中に放出することで「大酸化イベント」を引き起こした——嫌気性生物の大量絶滅と好気性生物の繁栄をもたらした、地球史上最大の環境変動の担い手だ。

進化生物学的に最も重要な点は、葉緑体の祖先であることだ。[内共生説](g185.md)（g185）によれば、約15億年前にシアノバクテリアが真核細胞に取り込まれ、やがて葉緑体へと変化した。現在のシアノバクテリアは「葉緑体になる前の姿」を保ったまま自由生活を続けている唯一の系統であり、いわば進化の分岐点の片方の末裔だ。

ミトコンドリアの祖先（アルファプロテオバクテリア）は現存する近縁種が全て細胞内寄生性に転じているのとは対照的に、シアノバクテリアは自由生活性を維持している。これは葉緑体の「逆進化」を考える上で決定的な差異だ——設計図が現存する。

[地衣類](g187.md)（g187）の光合成パートナーとしても機能し、菌類と共生することで砂漠・南極・岩肌など極限環境での生存を可能にしている。また一部のシアノバクテリアは窒素固定能力も持ち、光合成と窒素固定を単独でこなす自己完結型の代謝系を持つ。宇宙環境での生命維持や[惑星](g123.md)[テラフォーミング](g042.md)の文脈でも注目される存在だ。
