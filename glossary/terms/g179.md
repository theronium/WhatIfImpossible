# カオス理論（Chaos Theory）

**Chaos Theory**  数学・論理・幾何学 / 非線形力学・複雑系 / g179
読み: かおすりろん　関連: [コルモゴロフ複雑性](g177.md), [バタフライ効果](g178.md), [wiim_041](../../docs/logic/wiim_041.md), [wiim_052](../../docs/physics/wiim_052.md)

**別名**: カオス / カオス系 / 決定論的カオス

[決定論](g165.md)的な方程式に従いながらも、長期的な挙動の予測が実際上不可能になる系を研究する数学・物理学の分野。「カオス」とはこのような系の振る舞いそのものを指す。ランダムに見えるが本質的には決定論的——これがカオスの核心だ。

カオス系の最大の特徴は初期値鋭敏性で、初期条件のわずかな差が指数関数的に拡大して全く異なる結果をもたらす（[バタフライ効果](g178.md)・g178）。気象・乱流・[惑星](g123.md)軌道の長期変動・生態系の個体数変動など、自然界の多くの現象がカオス的な振る舞いを示す。

決定論とカオスは矛盾しない——カオス系は完全に決定論的な方程式から生まれる。しかしその複雑さは[コルモゴロフ複雑性](g177.md)（g177）を最大にまで押し上げ、決定論的でありながら予測不能という状態を作り出す。これが決定論内での[自由意志](g018.md)の感覚を生む一因であり（[wiim_040](../../docs/philosophy/wiim_040.md)・[wiim_041](../../docs/logic/wiim_041.md)）、「決まっているが解けない」という計算不能の典型例でもある。
