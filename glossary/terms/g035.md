# ワープドライブ / アルクビエレドライブ（Alcubierre Drive）

**Alcubierre Drive**  SF概念 / 相対論的宇宙物理・SF / g035
読み: あるくびえれどらいぶ　関連: [wiim_003](../../docs/physics/wiim_003.md), [wiim_004](../../docs/cosmology/wiim_004.md), [wiim_014](../../docs/physics/wiim_014.md), [wiim_079](../../docs/cosmology/wiim_079.md)

**別名**: アルクビエレドライブ / ワープ航法 / ワープ

1994年に物理学者ミゲル・アルクビエレが一般相対性理論の方程式を解いて提案したワープドライブの理論モデル。宇宙船自体は光速を超えず、船の前方の空間を収縮させ後方を膨張させることで、船ごと「空間の泡」を移動させる。

[特殊相対性理論](g108.md)の「物体は光速を超えられない」という制約は、空間の中を動く物体に対するもの。空間そのものが移動するなら制約がない——という発想。

問題は、[エキゾチック物質](g068.md)（負のエネルギー密度）が木星質量相当必要とされること、および[因果律](g017.md)の問題。2021年以降、必要エネルギーを劇的に減らす改良案が提案されているが、工学的実現にはほど遠い。
