# 記憶（Memory）

**Memory**  哲学・存在論 / 認知科学・心の哲学 / g180
読み: きおく　関連: [wiim_040](../../docs/philosophy/wiim_040.md), [wiim_008](../../docs/biology/wiim_008.md)

**別名**: 意味記憶 / エピソード記憶 / 手続き記憶 / ワーキングメモリ

過去の情報や体験を保持・再利用する脳の機能。大きく以下の種類に分類される。

**意味記憶**：「赤はリンゴの色」「水は液体だ」のような世界の一般知識。個人的な体験や時間・場所の文脈を持たない。赤からリンゴを連想する働きはここに属し、[クオリア](g032.md)（g032）の「機能的な土台」を担う——連想プロセス自体は意味記憶の活性化であり、クオリアそのものではない。

**エピソード記憶**：「あの夏、初めてリンゴを食べた」のような個人的な体験の記憶。時間・場所・感情の文脈を持ち、自己同一性の基盤となる。意味記憶が「何を知っているか」なら、エピソード記憶は「何を経験したか」だ。

**手続き記憶**：自転車の乗り方・楽器の演奏など「やり方」の記憶。言語化しにくく、意識せずに実行される。[自由意志](g018.md)（g018）の議論では、意識的な選択なしに行われる熟練した行動の説明に関わる。

**ワーキングメモリ**：今この瞬間に意識的に処理している情報の一時的な保持領域。意識の「舞台」とも呼ばれ、クオリアが体験される場所に最も近い概念だ。容量は限られており（7±2チャンク程度）、その制限が思考の流れを規定する。
