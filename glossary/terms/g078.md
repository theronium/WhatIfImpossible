# 仮説上の天体（Hypothetical Celestial Body）

**Hypothetical Celestial Body**  天文学・宇宙論 / 天文学・惑星科学・宇宙論 / g078
読み: かせつじょうのてんたい　関連: [反地球](g058.md)

**別名**: プラネット・ナイン / ネメシス / テイア

観測的証拠はないか不十分だが、理論的・数学的な根拠から存在が示唆される天体の総称。過去に仮説として提唱され後に発見された例もあれば、現在も未確認のものもある。

**第九惑星（プラネット・ナイン）**は海王星以遠の太陽系外縁天体の軌道の異常な集中から存在が示唆される未発見の惑星候補。質量は地球の5〜10倍と推定され、現在も探索が続いている。

**ネメシス**は太陽の伴星として提唱された仮説上の暗い恒星（[赤色矮星](g084.md)または[褐色矮星](g123.md)）。約2600万年周期の大量絶滅と隕石クレーターの周期性を説明するために1984年に提唱されたが、赤外線サーベイでも発見されておらず存在は否定的に見られている。

**テイア**は約45億年前に原始地球と衝突して月を形成したと考えられる火星サイズの原始惑星（ジャイアント・インパクト仮説）。直接的な痕跡は残っていないが、月の組成が地球のマントルと酷似していることなどが傍証とされる。

**[反地球](g058.md)（g058）**も[ラグランジュ点](g049.md)L3に想定された仮説上の天体の一例で、STEREO[探査機](g076.md)により否定されている。仮説上の天体は観測技術の向上によって発見または否定され、太陽系・宇宙の理解を更新し続けている。
