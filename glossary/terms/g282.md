# 基本相互作用（Fundamental Interactions）

**Fundamental Interactions**  素粒子・核物理学 / 素粒子物理学・場の量子論 / g282
読み: きほんそうごさよう　関連: —

**別名**: 四つの力 / 四種の力 / 自然界の四力

自然界に存在する4種類の相互作用——重力・電磁力・強い力・弱い力——の総称。現代物理学では「力」よりも「相互作用」という表現が正確で、それぞれゲージボソンと呼ばれる媒介粒子によって伝わる。

電磁力は光子、強い力は[グルーオン](g064.md)、弱い力はWボソン・Zボソンが媒介する。重力だけはまだ量子論的な記述（[重力子](g109.md)による媒介）が確立しておらず、[一般相対性理論](g108.md)と[量子力学](g253.md)の統合は現代物理学最大の未解決問題の一つだ。

4つの力の強さは桁違いに異なり、強い力を1とすると電磁力は約1/137、弱い力は約10⁻⁶、重力は約10⁻³⁸と極端に弱い。この「階層性問題」——なぜ重力だけこれほど弱いのか——は[余剰次元](g255.md)仮説などで説明が試みられている。
