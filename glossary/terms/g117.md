# 錬金術（Alchemy）

**Alchemy**  SF概念 / 哲学・歴史・化学史・思想 / g117
読み: れんきんじゅつ　関連: [wiim_024](../../docs/biology/wiim_024.md)

**別名**: アルケミー / 賢者の石 / 元素変換 / 生命の水 / 不老不死の霊薬

卑金属（鉛・銅など）を金・銀に変換すること、[不老不死の霊薬](g208.md)（エリクサー）や万能薬を作ることを目指した古代から中世にかけての思想体系・実践。古代エジプト・ギリシャに起源を持ち、イスラム黄金時代を経てヨーロッパに広まった。

錬金術師の最大の目標は**賢者の石**（Philosopher's Stone）の作成だった——あらゆる金属を金に変え、不老不死をもたらすとされた究極の物質。ニュートン、ボイル、パラケルススといった近代科学の父たちも錬金術に深く関わっており、錬金術は現代化学・薬学・冶金学の母胎でもある。

皮肉なことに「元素変換」自体は現代科学で**実現済み**だ。[粒子加速器](g074.md)や原子炉では陽子・[中性子](g114.md)の増減によって元素の変換（核変換）が可能であり、実際に鉛から少量の金が生成されている——ただしエネルギーコストが金の価値をはるかに上回るため実用性はない。

WhatIfImpossibleでは、この「夢の実現コスト問題」をどう突破するかが思考実験の核心となる。[マイコプラズマギカ](g130.md)（[wiim_024](../../docs/biology/wiim_024.md)）はその生物的アプローチの一つだ。
