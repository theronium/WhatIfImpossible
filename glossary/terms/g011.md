# CPT定理（CPT Theorem）

**CPT Theorem**  物理学 / 素粒子物理学・場の量子論 / g011
読み: しーぴーてぃーていり　関連: [wiim_015](../../docs/physics/wiim_015.md)

**別名**: CPT対称性

素粒子の物理法則は、以下の三つの変換を**同時に**行っても変わらないという定理。

- **C**（電荷共役変換）：粒子を反粒子に置き換える
- **P**（空間反転）：空間座標を鏡像に変える（左右反転）
- **T**（時間反転）：時間の方向を逆にする

C・P・Tをそれぞれ単独で行うと対称性が破れる場合がある（特に[CP対称性の破れ](g156.md)は実験で確認済み）。しかしCPT全てを同時に行うと常に対称になる——これが標準模型の根幹的な予測だ。

CPT定理は「時間を逆向きにした[反物質](g067.md)宇宙は、私たちの宇宙と全く同じ物理法則に従う」ことを意味する。
