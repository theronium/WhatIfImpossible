# キルヒホッフの法則（Kirchhoff's laws）

**Kirchhoff's laws**  物理学 / 電磁気学・分光学・熱放射 / g193
読み: きるひほっふのほうそく　関連: [wiim_044](../../docs/physics/wiim_044.md), [wiim_045](../../docs/physics/wiim_045.md)

ドイツの物理学者グスタフ・キルヒホッフ（1824–1887）が導いた複数の法則の総称。文脈によって指す内容が異なる。

**電気回路の法則（1845年）**：回路網の解析に使う二つの保存則。①電流則（KCL）——任意の節点に流れ込む電流の総和はゼロ。②電圧則（KVL）——任意の閉回路を一周したときの電位差の総和はゼロ。[電子](g064.md)回路設計の基礎として今日も広く使われる。

**熱放射・分光の法則（1859年）**：熱平衡状態にある物体の放射率と吸収率は等しい。この[原理](g306.md)から、物質は「吸収する波長と同じ波長の光を放射する」ことが導かれる。実用上は三つの分光法則として知られる——①高温固体・液体は連続スペクトルを放射する、②低圧の高温ガスは輝線スペクトル（固有波長の明るい線）を放射する、③連続光の前に置いた低温ガスは同じ波長に[暗黒線](g192.md)（吸収線、g192）を生じる。[フラウンホーファー線](g191.md)（g191）の物理的説明を与えた理論的基盤でもある。
