# 天文学（Astronomy）

**Astronomy**  天文学・宇宙論 / 天文学・宇宙物理学・宇宙論 / g124
読み: てんもんがく　関連: —

**別名**: 天体物理学 / 宇宙科学 / 観測天文学 / 電波天文学 / 宇宙物理学

宇宙に存在する[天体](g123.md)（g123）とその現象を研究する自然科学の一分野。人類最古の科学の一つであり、古代の暦計算・航海術から始まり、望遠鏡の発明（17世紀）、電磁波全域観測（20世紀）、[重力波](g004.md)検出（2015年）へと観測手段が拡張し続けている。

主な分野：

- **観測天文学**：可視光・電波・X線・ガンマ線・赤外線・重力波（g004）など全波長・全メッセンジャーで宇宙を観測
- **天体物理学**：物理法則を用いて天体の構造・エネルギー・進化を解明。恒星（g084）の核融合・ブラックホールの降着・[超新星](g084.md)爆発などを扱う
- **宇宙論**：宇宙全体の起源・構造・進化・終末を研究。[ビッグバン](g082.md)（g082）・宇宙膨張・[ダークエネルギー](g218.md)・[ダークマター](g056.md)（g056）が主題
- **宇宙生物学**：[エイリアン](g122.md)（g122）探索を含む、宇宙における生命の可能性を研究

現代天文学は[フェルミのパラドックス](g037.md)（g037）・[重力波検出器](g073.md)LIGO/LISA（g073）・[SETI](g094.md)（g094）など、WhatIfImpossibleの思考実験と直接交差する最前線の問いを抱えている。
