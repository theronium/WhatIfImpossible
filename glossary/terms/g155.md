# ハッブル地平線（Hubble Sphere）

**Hubble Sphere**  天文学・宇宙論 / 宇宙論・観測天文学 / g155
読み: はっぶるちへいせん　関連: [wiim_028](../../docs/cosmology/wiim_028.md), [wiim_079](../../docs/cosmology/wiim_079.md)

**別名**: ハッブル球 / ハッブル半径

宇宙膨張による後退速度がちょうど光速と等しくなる距離の境界面。現在の観測値ではハッブル定数H₀≒70 km/s/Mpcをもとに計算すると、その半径は約140億[光年](g096.md)とされる。この境界の外側では空間の膨張速度が光速を超えるため、そこから発せられた光は（たとえ光速で進んでも）永遠に地球に届かない。

ハッブル地平線は「現在観測できる限界」である宇宙の地平線（約460億光年）とは異なる。宇宙の地平線は過去に光が届いていた範囲であり、ハッブル地平線はあくまで現時点での膨張速度の境界だ。[ダークエネルギー](g218.md)によって膨張が加速し続ける限り、ハッブル地平線の内側にあった[天体](g123.md)が徐々に外側へ押し出され、永遠に通信も到達も不可能な領域が広がり続ける。

FTL通信（[wiim_028](../../docs/cosmology/wiim_028.md)）の記事でも言及されているように、ハッブル地平線の外へ情報を送るには[エキゾチック物質](g068.md)チャネルのような特殊な手段が必要となる。[宙域資産権](g154.md)（g154）の文脈では、保有する宙域がハッブル地平線の外に押し出されると物理的に到達不可能となり、資産が永久に失われる「ハッブル地平線リスク」として議論される。
