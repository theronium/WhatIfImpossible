# 星震学（Asteroseismology）

**Asteroseismology**  天文学・宇宙論 / 天文学・観測技術 / g329
読み: せいしんがく　関連: —

[恒星](g084.md)内部の振動（固有振動モード）を表面の微細な明るさ変動として観測し、恒星の質量・半径・内部構造を推定する学問分野。地球の地震波が内部構造を明らかにする地震学と同じ[原理](g306.md)を恒星に適用したもので、太陽に特化した呼称は「日震学（helioseismology）」という。

隣り合う振動モードの周波数間隔（大周波数間隔 Δν）は平均密度の平方根に比例し、半径が別手段で得られれば質量を導出できる。ケプラー[宇宙望遠鏡](g057.md)やTESSによる精密測光で多数の恒星に適用されており、衛星が存在しない恒星の質量決定に有力な手法となっている。[天体](g123.md)質量の測定手法の一つとして、WIIMの架空恒星系設計でも質量根拠として参照できる。
