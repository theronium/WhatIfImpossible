# プランクスケール（Planck Scale）

**Planck Scale**  物理学 / 量子重力論・理論物理学 / g014
読み: ぷらんくすけーる　関連: —

**別名**: プランク長 / プランク時間 / プランク単位

マックス・プランクが提唱した、重力定数 G・プランク定数 ℏ・光速 c の3つの基本定数から導かれる自然単位系。[量子力学](g253.md)と[一般相対性理論](g108.md)の双方が同等に重要になる極限スケールを表す。

**プランク定数（ℏ ≈ 1.055 × 10⁻³⁴ J·s）**は量子力学の根幹をなす定数で、エネルギーと振動数の比（E = hf）を与える。この定数が0でなく有限であることが、量子の「とびとびさ」の根源となっている。

**プランク長（≈ 1.616 × 10⁻³⁵ m）**はこれ以上小さい長さが物理的に意味を失うとされる最小スケール。[陽子](g114.md)の直径（約10⁻¹⁵ m）のさらに20桁小さい。この領域では時空が泡立ち、連続的な空間という概念が崩れると考えられる。

**プランク時間（≈ 5.391 × 10⁻⁴⁴ s）**はプランク長を光速で割った時間で、物理的に意味を持つ最短の時間間隔。[ビッグバン](g082.md)直後のプランク時間以前の宇宙は、既存の物理理論では記述できない。

3つはいずれも「量子重力理論が必要になる境界」を示しており、まだ完成していない統一理論の目標スケールでもある。
