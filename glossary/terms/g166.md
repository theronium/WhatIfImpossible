# ソリプシズム（Solipsism）

**Solipsism**  哲学・存在論 / 認識論・形而上学 / g166
読み: そりぷしずむ　関連: —

自分自身の心だけが確実に存在し、外部世界や他者の心は自分の意識が作り出した表象に過ぎないという立場。デカルトの「我思う、ゆえに我あり」は懐疑論の極致としてソリプシズムの入口に立っている。

反証不可能な立場であるため、哲学的に否定するのは難しい。外部世界が「存在する」と信じる根拠は、突き詰めれば習慣と実用性に基づく合理的な賭けだ。

量子力学との接続では、観測者が測定を行うまで系に確定した状態がない（[波動関数](g164.md)の崩壊・g164）という解釈が、ソリプシズム的な「観測者が現実を作る」という直感を強化するように見える。[多世界解釈](g162.md)（g162）はこの問いを「観測者は多数存在する」という方向に解消しようとする。
