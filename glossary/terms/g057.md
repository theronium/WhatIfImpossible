# 宇宙望遠鏡（Space Telescope）

**Space Telescope**  天文学・宇宙論 / 天文観測・宇宙探査 / g057
読み: うちゅうぼうえんきょう　関連: —

**別名**: ハッブル宇宙望遠鏡 / JWST / ジェームズウェッブ望遠鏡 / スピッツァー宇宙望遠鏡 / チャンドラX線望遠鏡

大気圏外の宇宙空間に設置された[天体](g123.md)観測装置の総称。地球大気による光の吸収・散乱・揺らぎ（シーイング）を受けないため、地上望遠鏡では困難な高解像度観測や、大気に遮られる波長帯（紫外線・X線・赤外線など）での観測が可能になる。

**ハッブル宇宙望遠鏡（HST、1990年〜）**は可視光・近紫外・近赤外で活躍し、宇宙膨張の加速発見や遠方銀河の観測に貢献した。打ち上げ直後に鏡の研磨ミスが発覚したが、1993年のスペースシャトルによる修理ミッションで復活し、現在も稼働中の長寿衛星となっている。

**ジェイムズ・ウェッブ宇宙望遠鏡（JWST、2021年打ち上げ）**は主鏡直径6.5mの赤外線特化型望遠鏡で、[ラグランジュポイント](g049.md)L2に配置されている。宇宙誕生後数億年の初期銀河や、[暗黒星雲](g055.md)内部の原始星系、系外惑星の大気成分分析など、ハッブルでは届かなかった領域を次々と観測している。

**スピッツァー宇宙望遠鏡（2003〜2020年）**は赤外線、**チャンドラX線観測衛星（1999年〜）**はX線帯域に特化し、それぞれブラックホール周辺や[超新星](g084.md)残骸の観測に成果を上げた。波長帯の異なる複数の宇宙望遠鏡が連携することで、宇宙の多層的な姿が明らかになりつつある。

宇宙望遠鏡は軌道の種類によって特性が大きく異なる。**[地球周回軌道](g077.md)型**はハッブルやチャンドラのように地球を周回する。スペースシャトルによるメンテナンスが可能だった反面、地球の影に入る周期や放射線帯の影響を受ける。**ラグランジュポイント型**はJWSTのようにL2に配置され、地球・太陽・月から等距離を保ちながら安定した熱環境と常時観測体制を確保できる。ただし地球から150万km離れるため修理は事実上不可能。**天体周回軌道型**はカッシーニ（土星）やガリレオ（木星）のように惑星を周回しながら観測する形態で、厳密には[探査機](g076.md)に分類されることも多いが望遠鏡機能を持つものもある。

また太陽を周回する**地球追従型**（スピッツァーなど）もあり、地球から離れることで地球の赤外線ノイズを避けられる利点があった。観測波長と目的に応じて最適な軌道が選択される。
