# 宇宙の距離梯子（Cosmic Distance Ladder）

**Cosmic Distance Ladder**  天文学・宇宙論 / 天文学・測距・宇宙論 / g356
読み: うちゅうのきょりはしご　関連: —

[天文学](g124.md)における距離測定手法の階層構造。近距離の手法が遠距離手法を校正し、それがさらに遠い手法へと引き継がれる「梯子」の比喩。各段は前の段の成果を土台にするため、下段の誤差が上段に伝播するという特性がある。

主な段は、①太陽系内：レーダー測距・光速による直接測定、②近傍恒星：[年周視差](g355.md)（〜数千[光年](g096.md)）、③銀河スケール：[セファイド変光星](g354.md)の周期–光度関係（〜数億光年）、④宇宙論スケール：Ia型[超新星](g084.md)（標準爆発）・ハッブルの法則（〜観測可能宇宙全体）。2010年代以降、ガイア[衛星](g123.md)による視差精度の向上と[超新星](g140.md)観測から独立に求めたハッブル定数の値が一致せず、「ハッブル張力」と呼ばれる未解決問題になっている。
