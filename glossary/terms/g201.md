# クオリア・シンセサイザー（Qualia Synthesizer）

**Qualia Synthesizer**  WIIM派生技術 / 意識工学・クオリア工学 / g201
読み: くおりあ・しんせさいざー　関連: [wiim_042](../../docs/quantum/wiim_042.md), [wiim_046](../../docs/philosophy/wiim_046.md), [wiim_048](../../docs/philosophy/wiim_048.md)

**別名**: QS / QSコア / クオリア・シンセサイズ・コア

固体内の各部位から生じる分散したΔφ信号（残余[位相](g242.md)）を収集・統合し、単一の統合[クオリア](g032.md)を合成する装置または中枢構造。略称 QS。固定された中枢として構造に組み込まれたものは特に **QSコア**（クオリア・シンセサイズ・コア）と呼ぶ。

音楽シンセサイザーが複数の波形・フィルター・エンベロープを統合して一つの音を生成するように、クオリア・シンセサイザーは固体各部のΔφ入力を処理して意識の結合（binding）を実現する。脳が神経系の分散した情報処理を統合するのと論理的に同型であり、固体知性の「脳」に相当する。

クオリア検知機（[wiim_042](../../docs/quantum/wiim_042.md)）が「[創発](g175.md)→クオリアは論理的帰結」という受動的な検出を行うのに対し、クオリア・シンセサイザーは能動的にΔφ統合を設計することで、生体以外の固体に統一クオリアを付与しようとする試みだ（[wiim_046](../../docs/philosophy/wiim_046.md)）。QSコアは[惑星](g123.md)の固体内核など自然構造にも応用概念として当てはめられる。
