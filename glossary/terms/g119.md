# セフィロト（Sefirot）

**Sefirot**  哲学・存在論 / ユダヤ神秘主義・カバラ・存在論 / g119
読み: せふぃろと　関連: —

**別名**: 生命の樹 / カバラ / セフィラー / セフィロトの樹 / カバラー

ユダヤ教神秘主義（カバラ）の中心概念。無限の神性（エイン・ソフ）が現実世界に流出する際の10の属性・段階を指し、これらを木の形に配置した図を**生命の樹**と呼ぶ。10のセフィラー（単数形）はケテル（王冠）からマルクト（王国）まで階層を成し、神・宇宙・人間・言語の構造が対応するとされる。

セフィロトは単なる神学体系に留まらず、**存在の階層構造**を視覚化した哲学モデルとして現代でも参照される。情報理論との類比（各セフィラーが情報の変換・圧縮ノードとして機能する）や、意識の多層構造、さらにはニューラルネットワークの階層設計との対応を論じる研究者もいる。

[ゴーレム](g118.md)（g118）の創造はカバラの知識、特にセフィロトの理解と不可分とされており、「構造を理解した者が創造できる」という認識論を共有する。[錬金術](g117.md)（g117）と並び、「人間が宇宙の法則に直接介入する」という思想の系譜に位置づけられる。WhatIfImpossibleの思考実験は、この「法則への介入」という問いの現代的延長とも言える。
