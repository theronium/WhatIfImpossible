# フラクタル分岐（Fractal Branching）

**Fractal Branching**  物理学 / 幾何学・生物形態・工学 / g333
読み: ふらくたるぶんき　関連: —

自己相似的な繰り返し分岐により、全体の形が部分と同じパターンを持つ構造。分岐の各レベルで断面積・角度・長さの比率が保存されるため、効率的な物質・エネルギー輸送と表面積の最大化を同時に実現する。数学的には[フラクタル](g015.md)次元（整数でない次元）で記述される。

工学ではフラクタルアンテナ（多バンド広帯域化）・マイクロ流体熱交換器・触媒担体に応用される。生物界での例は特に豊富で、肺の気管支（23段階の二分岐でガス交換面積約70㎡を確保）・全身血管系・神経突起の樹状突起・シダの葉・ロマネスコブロッコリーが代表例。スケールを跨いで同じ最適化戦略が収斂している点で、宇宙規模の分散生命体が取りうる形態として参照価値が高い。
