# 自由意志（Free Will）

**Free Will**  哲学・存在論 / 哲学・心の哲学 / g018
読み: じゆういし　関連: [wiim_015](../../docs/physics/wiim_015.md), [wiim_040](../../docs/philosophy/wiim_040.md), [wiim_041](../../docs/logic/wiim_041.md), [決定論](g165.md), [両立論](g170.md), [意志論](g234.md)

**別名**: フリーウィル

自分の行動を自分で決定できるという感覚・能力。哲学では大きく三つの立場がある。

- **[決定論](g165.md)（g165）**：全ての出来事は過去の原因によって必然的に決まり、自由意志は幻想だ
- **非決定論**：量子的なランダム性などにより、未来は一意に決まらない。ただしランダム性が自由意志かは別問題
- **[両立論](g170.md)（g170）**：決定論と自由意志は矛盾しない。「強制されていない」意味での自由があれば十分とする

スケールの視点を加えると議論は変容する。量子スケールでは確率的・人間スケールでは決定論的に見える同じ現実が、宇宙スケールでは人間の選択を「統計的ゆらぎ」として扱う。しかしその統計的許容範囲の内側では、生死ですら確率的に自由だ——決定論は宇宙規模の統計として成立しながら、その幅の内側では本物の自由を許容している（[wiim_040](../../docs/philosophy/wiim_040.md)）。

また決定論的プロセスでも計算複雑性が最大になると外部から予測不可能になる。「決まっているが解けない」系は機能的に自由意志と等価だ（[wiim_041](../../docs/logic/wiim_041.md)）。
