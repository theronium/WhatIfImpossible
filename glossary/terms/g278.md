# エバネッセント波（Evanescent Wave）

**Evanescent Wave**  物理学 / 波動光学・量子物理学 / g278
読み: えばねっせんとは　関連: —

波数kが虚数になる条件下で生じる、指数関数的に減衰する波。通常の伝播波（実数k）が遠方まで振動しながら伝わるのに対し、エバネッセント波は媒質の境界近傍にのみ存在する。「evanescent」はラテン語「消えゆく（evanescere）」に由来する。

全反射時に境界面の外側へわずかに染み込む光がその典型で、別の媒質を非常に近傍に置くと「トンネル」して透過する（フラストレイテッド全内部反射）。近接場光学顕微鏡（NSOM）ではこの性質を利用して回折限界を超えた分解能を実現する。[量子力学](g253.md)の[トンネル効果](g097.md)はエバネッセント波の量子版と見なせる。WIIM架空粒子[エイソン](g277.md)（g277）はこのモードを指数関数的減衰なしに伝播するものとして構想されている。
