# 音響複層体（Acoustic Gradient Composite）

**Acoustic Gradient Composite**  WIIM派生技術 / 音響工学・複合素材工学 / g438
読み: おんきょうふくそうたい　関連: [余剰次元可塑体](g437.md), [素材遺伝子](g441.md)

**別名**: フォノニック複層体

[余剰次元](g255.md)を必要とせず通常の物理法則のみで実現する、音波・衝撃耐性複合素材の設計[原理](g306.md)。高音響インピーダンス素材（ソニックホワイト）の外層が入射音波・衝撃波の大部分を反射し、高音響吸収素材（ソニックブラック）の内層が透過エネルギーを熱散逸に変換する。両層間のインピーダンス遷移層が再反射を防ぎ吸収効率を最大化する。電磁波の白色（全反射）・黒色（全吸収）との対比から命名された。

自然界はすでにこの設計を実現している。真珠層（アコヤ貝）は炭酸カルシウム板とタンパク質層の交互積層により純粋な炭酸カルシウムの約3000倍の靭性を持つ。骨の皮質骨（反射）と海綿骨（吸収）、シャコの捕脚の螺旋繊維層も同原理である。

内層ソニックブラックに[余剰次元可塑体](g437.md)を採用すれば、熱蓄積なしに繰り返し衝撃を吸収できる。
