# コスモシェル（CosmosShell）

**CosmosShell**  WIIM概念 / 架空素材・宇宙工学・テラフォーミング / g132
読み: こすもしぇる　関連: [wiim_011](../../docs/physics/wiim_011.md), [wiim_064](../../docs/physics/wiim_064.md)

**別名**: 閉鎖膜 / 宇宙のシャボン玉 / 天体膜 / 自己修復膜

WhatIfImpossibleの思考実験（[wiim_011](../../docs/physics/wiim_011.md)）で命名された架空の構造体。真空・宇宙空間においても自己維持できる薄膜の球体で、内部に大気や生態系を封じ込めることができると仮定される。「Cosmos（宇宙）」と「Shell（殻）」を組み合わせた造語。

地球上のシャボン玉は表面張力・水分・気圧均衡によって成立するが、宇宙空間ではこれらが全て崩壊する。コスモシェルはこの問題を「膜が能動的に自己修復する」ことで解決すると仮定される——細胞膜が脂質二重層を動的に再配置するように、膜を構成する分子が常に能動的に動いて損傷を修復し、内外の圧力差に抗う。つまり膜自体が「生きている」に等しい振る舞いをする構造体だ。

応用としては[小惑星](g123.md)への大気圏形成・宇宙を漂う自己完結した生態系の泡・[テラフォーミング](g042.md)（g042）の前段階としての環境封入などが考えられる。[グラビトーペイク](g129.md)（g129）が[重力波](g004.md)を遮断する「殻」であるのに対し、コスモシェルは大気・物質を保持する「膜」として、WhatIfImpossibleの宇宙工学系架空素材の一つに位置づけられる。
