# タビーの星（Tabby's Star）

**Tabby's Star**  天文学・宇宙論 / 天文観測・恒星物理・SETI / g462
読み: たびーのほし　関連: [wiim_008](../../docs/biology/wiim_008.md)

**別名**: ボヤジャンの星 / WTF星 / KIC 8462852

KIC 8462852の通称。白鳥座に位置する[恒星](g084.md)（地球から約1470[光年](g096.md)）で、ケプラー[宇宙望遠鏡](g057.md)のデータ分析中に発見した異常な減光パターンで注目を集めた。発見を主導したタベサ・ボヤジャンから「ボヤジャンの星」とも呼ばれる。

通常の[惑星](g123.md)トランジットによる減光は1%未満だが、タビーの星は不規則かつ非周期的に最大22%の急激な減光を繰り返し、長期的にも緩やかな減光傾向を示す。説明仮説の一つが「[ダイソン球](g034.md)建設中の宇宙文明による遮蔽」であり、[SETI](g094.md)コミュニティで広く議論された。現在の有力説は彗星群や周星塵による遮蔽だが、完全な解明には至っていない。

[コズミックマイス](g134.md)が形成する太陽系規模の[菌糸](g112.md)構造が類似の減光パターンを生み出しうるという思考実験の参照点として機能する。[パレイドリア仮説](g461.md)が扱う「自然成長を知的設計と誤認する問題」の実在する[天文学](g124.md)的事例だ。
