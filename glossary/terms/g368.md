# 時空メタマテリアル（Spacetime Metamaterial）

**Spacetime Metamaterial**  WIIM概念 / 時空間操作・材料工学・理論物理学 / g368
読み: じくうめたまてりある　関連: [wiim_090](../../docs/cosmology/wiim_090.md)

時空の特性（時間勾配・空間曲率）を空間的に周期配列した人工的な媒質構造。通常のメタマテリアルが電磁場の伝播特性を屈折率テンソルで制御するように、時空メタマテリアルは物質の世界線（時空軌跡）を時間勾配テンソルのパターンで制御する。

実物理学では「時間的メタマテリアル（Temporal Metamaterial）」として時間的に変調する媒質の研究が進んでおり、波の異常反射・周波数変換が実証されつつある。WIIMではこれを[クロノスフィア](g125.md)（g125）の回転[光子](g064.md)シェルへと拡張し、シェル上の時間勾配を空間的に周期配列することで通過粒子の世界線をブラッグ回折的に屈折させる構造として概念化される。単位セル周期 Λ によって共鳴エネルギー帯が決まり、入射粒子をエネルギー選択的に特定の角運動モードへ誘導できる点が通常の質量遮蔽との根本的な違いだ。

実現の壁は、時間勾配を空間的に周期配列するためにクロノスフィア境界の形状を亜粒子波長スケール（フェムトメートル以下）で制御する必要があることと、時間方向への世界線伸張が空間的な角運動量に変換される際の保存則が現在の物理理論では未定義であることにある。
