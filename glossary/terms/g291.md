# 四元数（Quaternion）

**Quaternion**  数学・論理・幾何学 / 数学・代数学 / g291
読み: しげんすう　関連: —

実数1つと虚数単位 i・j・k の3成分からなる数体系。ウィリアム・ローワン・ハミルトンが1843年に発見した。[複素数](g104.md)を2次元平面上の回転として解釈できるのと同様に、四元数は3次元空間上の回転を表す道具として機能する。

四元数では乗法の交換法則（ab = ba）が成り立たない。これは代数構造の「当然の性質」と思われていた交換法則を捨て去ることで新たな数体系を構築した歴史的な事例であり、数学的な常識の拡張を象徴する。現代では3Dコンピュータグラフィクスやロボット工学における姿勢制御に広く使われている。

WIIMの文脈では、[アーベル・ルフィニの定理](g290.md)（g290）が示した「5次方程式の解法の壁」をめぐる議論と並んで、「既存の規則を破ることで生まれる新しい秩序」の事例として参照される。
