# オイラーの定理（群）（Euler's Theorems）

**Euler's Theorems**  数学・論理・幾何学 / 数学・解析学・数論・位相幾何学 / g230
読み: おいらーのていり　関連: [オイラー](g229.md)

[オイラー](g229.md)（g229）の名を冠する[定理](g304.md)は数学の複数分野にまたがる。代表的なものは以下の4つ。①オイラーの公式：e^(iθ) = cosθ + i sinθ（[複素数](g104.md)と三角関数を結ぶ）。②多面体定理：凸多面体の頂点数 V・辺数 E・面数 F について V−E+F=2 が成立する（[位相幾何学](g158.md)の先駆）。③数論のオイラー定理：a と n が互いに素なとき a^φ(n) ≡ 1 (mod n)（RSA暗号の数学的基盤）。④オイラー・ラグランジュ方程式：変分法による運動方程式の統一的導出（解析力学の根幹）。

それぞれ独立した分野の定理だが、いずれもオイラーが一人で確立した。各定理の詳細は補遺「オイラーの主要定理一覧」を参照。
