# ノーファペン（No-P-Happen）

**No-P-Happen**  WIIM概念 / 架空素材・因果律工学・FTLステルス技術 / g152
読み: のーふぁぺん　関連: [wiim_030](../../docs/physics/wiim_030.md), [wiim_023](../../docs/physics/wiim_023.md), [wiim_010](../../docs/physics/wiim_010.md)

**別名**: ノーP / 因果ステルス材

[パラドックス粒子](g148.md)（g148）を吸収または霧散させ、FTL操作や時間遡行の痕跡を隠蔽する素材の総称。元は「No P-Happen（Pが起こらない）」という製品名だったが、競合他社の類似品も含めたカテゴリ全体を指す一般名詞として定着した——「ゼロックス」が複写全般を指すようになったのと同じ経緯だ。

[パラドックス](g113.md)粒子は因果矛盾が解消された後に排出される残滓であり、因果プロセスそのものではない。そのため吸収・変換・放出のいずれの操作も[因果律](g017.md)に影響しない。ノーファペンは純粋に「証拠の痕跡を消す」技術であり、操作後の因果構造は手を加える前と変わらない。吸収型の限界は因果的なものではなく、蓄積による素材の物性的な劣化・破損にとどまる。

現行製品の主流は「変換放出型」だ。吸収したパラドックス粒子を[エキゾチック物質](g068.md)（[wiim_023](../../docs/physics/wiim_023.md)）に変換して放出する機構を持ち、素材の物性限界を回避しながら痕跡を隠蔽できる。副産物としてエキゾチック物質が排出されるためパラドックス粒子検出網はすり抜けられる一方でエキゾチック物質検出網には反応する——FTLドライブの通常排気への偽装が実質的な抜け穴となっている。消費したエキゾチック物質の一部を回収できる点から変換効率（民生用30〜50%・[業](g206.md)務用70〜85%・軍用90%以上）が製品グレードの主要スペックとなり、希少資源の回収装置としての需要も生まれている。素材の特性はエキゾチック物質の近縁とみられ、[グラビトーペイク](g129.md)（[wiim_010](../../docs/physics/wiim_010.md)）と同様に蓄積量に応じた物性劣化管理が必要な点でも知られる。
