# メスバウアー効果（Mössbauer Effect）

**Mössbauer Effect**  物理学 / 核物理学・量子力学 / g387
読み: めすばうあーこうか　関連: [wiim_094](../../docs/quantum/wiim_094.md)

結晶格子中の[原子核](g114.md)がガンマ線を放出・吸収する際、反動（反跳）が結晶全体のN個の原子核によって分担されることで、1原子あたりの反動が事実上ゼロになる現象。1958年にルドルフ・メスバウアーが発見し、翌1961年にノーベル物理学賞を受賞した。

通常、放射性核がガンマ線を放出すると運動量保存則により核自身が反跳する。しかし固体結晶中では核が格子と強く結合しているため、反動が格子全体の集団自由度に分配される。N個の核が反動を等分すれば1核あたりの反動は1/Nとなり、Nが十分大きければ原子レベルの反動エネルギーが量子的に無視できる水準に達する。

この「反動のない」核遷移により、放出・吸収されるガンマ線は極めてシャープなエネルギー分布を持ち、精密分光法として重力[赤方偏移](g001.md)の測定や物質の[電子](g064.md)状態解析などに応用されている。[チェシャ磁場格子](g385.md)における反作用分散型推進力の理論的類比として参照される。
