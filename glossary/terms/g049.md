# ラグランジュポイント（Lagrange Point）

**Lagrange Point**  天文学・宇宙論 / 天体力学・軌道力学 / g049
読み: らぐらんじゅぽいんと　関連: [コリオリの力](g051.md)

**別名**: ラグランジュ点 / L点

二つの天体（例: [太陽](g084.md)と地球）が重力で引き合う系の中で、第三の小天体が両者の重力と遠心力のバランスにより相対位置を保ったまま運動できる5つの特別な点。1772年にイタリア・フランスの数学者ジョセフ＝ルイ・ラグランジュが解析的に導いた。

| 点 | 位置 | 安定性 |
|----|------|--------|
| L1 | 二天体の間 | 不安定 |
| L2 | 小天体の外側（太陽と反対側） | 不安定 |
| L3 | 大天体の裏側 | 不安定 |
| L4 | 軌道上60°前方 | 安定 |
| L5 | 軌道上60°後方 | 安定 |

L1・L2・L3は不安定な均衡点で、微小なずれが増幅するが、少量の燃料で軌道を維持できるため宇宙機の「駐機場」として活用される。ジェイムズ・ウェッブ[宇宙望遠鏡](g057.md)はL2に配置されており、地球の陰に隠れながら太陽・地球・月の熱源から遠ざかった安定した観測環境を得ている。L4・L5は安定で、木星の[トロヤ群](g050.md)[小惑星](g123.md)群がL4・L5に集積している。将来の[宇宙コロニー](g081.md)や燃料補給基地の候補地としても注目される。

L4・L5の安定性は[コリオリの力](g051.md)と深く関係している。共回転座標系（二天体と一緒に回転する視点）で考えると、重力・遠心力に加えてコリオリの力が現れる。L1〜L3では釣り合い点からわずかにずれるだけで発散してしまうが、L4・L5ではコリオリの力がずれた物体を横向きに曲げ続け、発散を防ぐ「馬蹄形軌道」を描かせる。つまりコリオリの力がなければL4・L5の安定性は成立せず、トロヤ群のような集積も生まれなかった。
