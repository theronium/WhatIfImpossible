# ER=EPR予想（ER=EPR Conjecture）

**ER=EPR Conjecture**  物理学 / 量子重力・量子情報・ブラックホール物理 / g322
読み: いーあーるいこーるいーぴーあーるよそう　関連: [wiim_081](../../docs/physics/wiim_081.md)

2013年にマルダセナとサスキンドが提唱した仮説。アインシュタイン・ローゼン橋（ER）とアインシュタイン・ポドルスキー・ローゼン相関（EPR、[量子もつれ](g062.md)）が同一の物理現象の異なる記述だという主張だ。

ERはアインシュタインが1935年に発表した[ワームホール](g036.md)解、EPRは同年に発表した量子もつれの論文に由来する。同じ1935年に生まれた全く異なる概念が実は同じものだという逆説的な提唱だ。

具体的には、[ホーキング放射](g248.md)で生成される[光子](g064.md)のペアはEPR的にもつれており、放出光子と残されたブラックホール内部の光子がERブリッジで幾何学的につながっているという描像を与える。これにより**ブラックホール情報[パラドックス](g113.md)**の解消への道筋として注目された。[コーラ粒子](g127.md)（g127）によるブラックホール間接続（[wiim_081](../../docs/physics/wiim_081.md)）との関係——両者が等価かどうか——は未解決の問いとして残る。
