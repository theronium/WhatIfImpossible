# 地衣類（Lichen）

**Lichen**  生物学・進化 / 菌類学・生態学 / g187
読み: ちいるい　関連: [菌根ネットワーク](g031.md), [菌糸](g112.md), [内共生説](g185.md), [wiim_043](../../docs/biology/wiim_043.md)

**別名**: ライケン

菌類と光合成生物（緑藻または[シアノバクテリア](g188.md)、あるいは両者）が密接に共生した複合生命体。単独では「地衣類」という種は存在せず、異なる生物界をまたぐ組み合わせによって初めて成立する。菌類が構造・保水・ミネラル供給を担い、光合成パートナーが有機物（糖）を供給する。

地衣類が興味深いのは、植物が葉緑体を「細胞内に完全統合した」のとは異なる戦略をとっているからだ。菌類は葉緑体を持つ生物を細胞外でパートナーとして抱え込む——内共生（g185）の途中段階が安定的に固定化した形とも見える。光合成能力を「器官として所有」するのではなく「生物として同居」させる別解だ。

三者共生の地衣類も存在する。マツゲゴケ（Lobaria pulmonaria）は菌類＋緑藻＋シアノバクテリアの三者共生で、緑藻が光合成を担いシアノバクテリアが窒素固定を担う分[業](g206.md)体制をとる。単一の「個体」の中に三つの独立した生物の系統が共存している。

代表的な種には以下のものがある。ハナゴケ（Cladonia rangiferina、別名トナカイゴケ）は北極圏の氷雪環境でトナカイの主食となる極限共生の例。リトマスゴケ（Roccella tinctoria）はリトマス試験紙の原料となる色素を含む。地図ゴケ（Rhizocarpon geographicum）は年数ミリ以下という極めて遅い成長速度を持ち、岩盤の露出年代測定（地衣計年法）に使われる——生物が時計として機能する例だ。イワタケ（Umbilicaria esculenta）は食用で乾燥・放射線への耐性が高い。サルオガセ（Usnea属）は木からぶら下がる糸状で二酸化硫黄に敏感な大気汚染の指標種として知られる。

極限環境耐性が極めて高く、南極の岩肌・砂漠・高山帯でも生育できる。宇宙空間の真空・放射線・極低温にも短期間耐えることが実験で確認されており、[惑星](g123.md)間での生命の移動（[パンスペルミア](g007.md)仮説）の候補として挙げられることがある。

[菌根ネットワーク](g031.md)（g031）が地下の[菌糸](g112.md)で樹木をつなぐのと対照的に、地衣類は二者以上の生物が肉体的に融合した形で存在する——どこまでが「一つの個体」かという問いを根本から揺さぶる存在だ。
