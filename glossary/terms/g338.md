# プラトーの法則（Plateau's Laws）

**Plateau's Laws**  物理学 / 流体力学・表面物理・泡構造数学 / g338
読み: ぷらとーのほうそく　関連: [wiim_084](../../docs/biology/wiim_084.md), [コズミックマイス](g134.md), [バブルシェルマイセリウム](g339.md)

複数の石鹸膜や泡が接するとき、界面が従う幾何学的法則。19世紀ベルギーの物理学者ジョゼフ・プラトー（Joseph Plateau）が実験的に定式化した。

法則は4条件からなる。①各膜面は一定の平均曲率を持つ。②膜が2枚の気泡膜で構成される場合、内外の圧力差はヤング＝ラプラス方程式で記述される（石鹸膜では両面に表面張力が働くためΔP = 4γH）。③3枚の膜が交わる「プラトー境界」では膜が互いに120°で交差する。④4本のプラトー境界が交わる頂点では正四面体角（約109.47°）が形成される。

2つの等圧バブルが共有する壁では平均曲率H = 0となり、完全に平坦な膜が形成される。この「H=0面」は両側の圧力差がゼロであるため、化学シグナルや電荷が圧力勾配に妨げられない中立接続面として機能する。

「同体積・等圧の多数セルを最小面積で充填する最適構造」の問いはケルビンの予想（1887年）として長く未解決だったが、1994年にウェア＝フェラン構造がケルビン構造より約0.3%少ない表面積を実現することが示された。この最適配置は自然界の泡・細胞充填・結晶構造などに広く出現する。WIIMの思考実験では、[コズミックマイス](g134.md)の多球コロニー（[バブルシェルマイセリウム](g339.md)）がこの法則に従って自発的に最小エネルギー配置を形成すると仮定される（[wiim_084](../../docs/biology/wiim_084.md)）。
