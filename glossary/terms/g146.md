# 時空の泡（Quantum Foam / Spacetime Foam）

**Quantum Foam / Spacetime Foam**  物理学 / 量子重力・素粒子物理学・宇宙論 / g146
読み: じくうのあわ　関連: [wiim_029](../../docs/physics/wiim_029.md), [wiim_030](../../docs/physics/wiim_030.md)

**別名**: 量子泡沫 / スペースタイムフォーム

ジョン・ホイーラーが1955年に提唱した概念。[プランクスケール](g014.md)（約10⁻³⁵メートル）では、量子力学の不確定性原理により時空の幾何学そのものが激しく揺らぎ、滑らかな連続体としての時空が崩れて泡のような複雑な[位相](g242.md)構造を持つという仮説。

通常スケールでは時空は滑らかに見えるが、プランク長に近づくにつれ仮想ブラックホールや仮想[ワームホール](g036.md)が絶え間なく生成・消滅し、時空の位相（[トポロジー](g158.md)）が瞬間的に変化し続けると考えられる。この「泡」は量子真空における重力場の揺らぎの極限的表れであり、[カシミール効果](g009.md)（g009）が電磁場の真空揺らぎを示すのと対応する。

より低いエネルギースケールでも、仮想[重力子](g109.md)の絶え間ない交換が静的重力場を[ファンデルワールス力](g052.md)的に揺らがせるという議論があり、[LIGO](g073.md)の量子ノイズはその実用的な影響例の一つだ。フェルミ[宇宙望遠鏡](g057.md)のガンマ線バースト観測データからプランクスケール以下の時空の泡に制約が置かれているが、直接検出にはいたっていない。[コーラ粒子](g127.md)（[wiim_029](../../docs/physics/wiim_029.md)）が余剰次元を経路として使う場合、この重力真空揺らぎが誘導精度に影響すると考えられる。
