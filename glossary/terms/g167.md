# ブロック宇宙（Block Universe）

**Block Universe**  哲学・存在論 / 時間の哲学・相対性理論 / g167
読み: ぶろっくうちゅう　関連: [wiim_040](../../docs/philosophy/wiim_040.md)

**別名**: 永久主義 / エターナリズム

過去・現在・未来が等しく実在し、時空間が一つの固定した四次元ブロックとして存在するという時間観。「永久主義」とも呼ばれる。[特殊相対性理論](g108.md)の同時性の相対性——ある観測者にとって「今」の出来事が別の観測者には「過去」や「未来」になる——はこの描像を支持する。

ブロック宇宙はしばしば[決定論](g165.md)と混同されるが、独立した概念だ。決定論は「未来が過去から一意に計算できる」という因果の問いであり、ブロック宇宙は「未来がすでに存在するか」という時間の存在論の問いだ。量子[多世界解釈](g162.md)（g162）はその好例で、全分岐がブロックに実在する——「未来はすでにある」がブロック宇宙的だが、どの分岐に進むかは確率的であり決定論ではない。逆に「成長するブロック宇宙」という立場は、未来はまだ存在しないが因果的に決まっていく、という非ブロック宇宙の決定論だ。

ブロック全体を「外から」見れば[エントロピー](g008.md)は消費されない——[ビッグバン](g082.md)端の低エントロピーから[熱的死](g085.md)端の高エントロピーへの勾配が構造として刻まれており、その勾配こそが時間の矢の方向を定義している。しかしブロック内の観測者が因果連鎖をたどり情報を記録する行為は、[ランダウアー原理](g172.md)に従い必ずエントロピーを消費する。ブロックは静的だが、内側から閲覧するコストは免除されない。

[アカシックレコード](g086.md)との接続は深い。ブロック宇宙が正しければ全時間の全事象はすでに存在しており、アカシックレコードはその静的な時空構造そのものとなる——レコードは「無料」で存在するが、内側から読む行為には必ず代償が伴う。また[レトロン](g163.md)（[wiim_037](../../docs/physics/wiim_037.md)）が引き起こす遡及[因果性](g017.md)も、ブロック宇宙では「閉じた因果ループとして時空に最初から書き込まれている」として吸収できる。
