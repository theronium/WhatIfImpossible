# ユークリッド距離（Euclidean Distance）

**Euclidean Distance**  物理学 / 数学・幾何学・情報科学 / g283
読み: ゆーくりっどきょり　関連: [ミンコフスキー距離](g285.md)

2点間の「まっすぐな直線」の長さを表す最も基本的な距離の概念。古代ギリシャの数学者ユークリッドが体系化した幾何学に由来する。2次元平面上の点 A(x₁, y₁) と B(x₂, y₂) の距離は √((x₂-x₁)² + (y₂-y₁)²) で求められ、これはピタゴラスの定理の直接的な応用だ。

高次元へも自然に拡張でき、n次元空間でも各座標差の二乗和の平方根として定義される。物理学・統計学・機械学習・コンピューターグラフィックスなど幅広い分野で「距離」の基準として使われる。一方、格子状の空間や経路が制限された状況では、直線距離より実際の移動コストを反映した[マンハッタン距離](g284.md)（g284）のほうが適切な場合がある。

[量子力学](g253.md)や[一般相対性理論](g108.md)では「距離」の定義そのものが拡張され、ユークリッド距離が成り立つ平坦な空間は特殊なケースとして位置づけられる。
