# スタタイト（Statite）

**Statite**  天文学・宇宙論 / 宇宙推進工学・軌道力学 / g351
読み: すたたいと　関連: [ソーラーセイル](g054.md)

**別名**: 停止衛星

[ソーラーセイル](g054.md)の放射圧と[太陽](g084.md)重力を釣り合わせることで、軌道運動をせずに太陽に対して静止した位置を保つ[人工天体](g076.md)の概念。[宇宙物理学](g124.md)者ロバート・フォワードが1993年に提唱した。「Static（静止）」＋「Satellite（[衛星](g123.md)）」の合成語。

通常の人工衛星は重力と遠心力のバランスで軌道を周回するが、スタタイトは十分に軽量な太陽帆を展開し、[光子](g064.md)が帆に衝突・反射する際の運動量（放射圧）で重力を相殺することで「軌道を持たずに静止する」状態を維持する。

理論上は黄道面の内外・太陽と地球の間など任意の位置に配置でき、太陽極域の連続監視や太陽嵐の早期警戒センサーへの応用が期待される。地球近傍での実現には帆面1 m²あたり総質量0.77 g以下という高比強度が必要で、現行の材料技術では実証段階に達していない。
