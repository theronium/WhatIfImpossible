# 真空崩壊（Vacuum Decay）

**Vacuum Decay**  物理学 / 量子場理論・宇宙論 / g012
読み: しんくうほうかい　関連: [wiim_014](../../docs/physics/wiim_014.md)

**別名**: 偽真空崩壊 / フォールスバキューム

現在の宇宙の真空状態（ヒッグス場の値）が「偽の真空」であり、より低いエネルギーを持つ「真の真空」状態へと相転移する可能性を指す。

もし宇宙のどこかで真空崩壊が始まると、光速で広がる「泡」が形成され、泡の内側では物理定数が全て変化する。[電子](g064.md)の質量、光速、[原子](g114.md)の安定性——あらゆるものが書き換わり、現在の物理で成立する構造は全て崩壊する。泡は光速で広がるため、私たちが気づく前に到達する。

2012年に発見された[ヒッグス粒子](g070.md)の質量（約125GeV）は、現在の宇宙が「準安定な偽の真空」にあることを示唆する計算結果と一致しており、真空崩壊は完全な空想ではない。ただし起きるとしても宇宙年齢をはるかに超えるスケールの話とされる。
