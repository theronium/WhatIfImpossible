# 熱力学（Thermodynamics）

**Thermodynamics**  物理学 / 熱力学・統計力学 / g250
読み: ねつりきがく　関連: —

熱とエネルギーの変換を扱う物理学の一分野。第0〜第3法則を柱とし、特に「[エントロピー](g008.md)は孤立系で増大する（第二法則）」はWIIMの思考実験の多くを根底から制約する原理だ。

WIIM世界では、[レトロン](g163.md)（g163）・[パランティ粒子](g161.md)（g161）・量子永久機関（[wiim_039](../../docs/quantum/wiim_039.md)）などの架空技術がいずれもこの壁を超えようとする試みとして位置づけられる。エントロピーの「格子間転嫁」「[余剰次元](g255.md)へのオフロード」「負エントロピー粒子による逆転」の3経路が主な突破口の候補として論じられる。
