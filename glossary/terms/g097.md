# 量子トンネル効果（Quantum Tunneling）

**Quantum Tunneling**  物理学 / 量子力学・素粒子物理・固体物理 / g097
読み: りょうしとんねるこうか　関連: —

**別名**: トンネル効果 / 量子トンネリング

古典力学では越えられないはずのエネルギー障壁を、量子粒子が確率的に「すり抜ける」現象。粒子が[波動関数](g164.md)として記述されるとき、障壁の向こう側にも波動関数が染み出しており、一定確率で粒子がそちら側に現れる。

日常的な比喩では「壁にボールを投げても跳ね返るだけだが、[電子](g064.md)なら一定確率で壁を通り抜ける」となる。障壁が薄いほど・粒子が軽いほど確率が高くなる。

応用は広範で、[太陽](g084.md)の核融合（[陽子](g114.md)がクーロン障壁をトンネルすることで点火する）・半導体トランジスタの動作・走査型トンネル顕微鏡（STM）・フラッシュメモリの書き込みなど、現代技術の根幹を支える。[量子揺らぎ](g061.md)（g061）と密接に関連しており、[真空崩壊](g012.md)（宇宙の真空が別の真空状態にトンネルする）という宇宙論的な文脈でも登場する。[量子力学の未解決問題](g092.md)（g092）の一つである測定問題とも接点があり、「トンネルした瞬間」をどう解釈するかは解釈論上の議論を呼ぶ。
