# ワームホール（Wormhole）

**Wormhole**  SF概念 / 一般相対性理論・SF / g036
読み: わーむほーる　関連: [wiim_065](../../docs/physics/wiim_065.md)

**別名**: アインシュタイン＝ローゼン橋 / 時空トンネル

時空を「つなぐ」仮説上の構造。アインシュタイン＝ローゼン橋とも呼ばれる。[一般相対性理論](g108.md)の方程式の解として数学的には存在するが、安定して維持するには大量の[エキゾチック物質](g068.md)が必要で、物理的な実現可能性は極めて低い。

SFでは星間移動の手段として頻繁に使われる。最近の理論（ER = EPR仮説）では、[量子もつれ](g062.md)でつながった二つのブラックホールがワームホールで接続されているという対応関係が示唆され、[量子情報理論](g252.md)と重力の橋渡しとして注目されている。

ブラックホールの時間反転として導かれる[ホワイトホール](g100.md)（g100）とは表裏一体の関係にある。シュヴァルツシルト計量の最大拡張では「ブラックホール → ワームホール → ホワイトホール」という構造が一つの時空図に現れる。
