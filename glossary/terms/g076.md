# 人工天体（Artificial Celestial Body）

**Artificial Celestial Body**  天文学・宇宙論 / 宇宙工学・宇宙探査 / g076
読み: じんこうてんたい　関連: [ラグランジュポイント](g049.md), [スイングバイ](g053.md), [ソーラーセイル](g054.md), [宇宙望遠鏡](g057.md)

**別名**: ボイジャー / 探査機 / 宇宙ステーション / ニューホライズンズ

人間が製造し宇宙空間に打ち上げた物体の総称。目的・軌道・設計によって多様な種類に分類される。

**人工衛星**は地球を周回する最も一般的な人工[天体](g123.md)で、用途により通信衛星・気象衛星・測位衛星（GPS・みちびき）・地球観測衛星などに分かれる。現在地球周回軌道には数千機の運用中衛星と大量のデブリが存在し、軌道の混雑が課題となっている。

**惑星探査機**は特定の天体を目指す一方向または周回飛行の探査機。ボイジャー1・2号は木星・土星などを[スイングバイ](g053.md)（g053）しながら外惑星系を探査し、現在は太陽圏を脱して[恒星](g084.md)間空間を航行中だ。カッシーニは土星を13年間周回し、ニューホライズンズは冥王星に最接近した。

**宇宙ステーション**は長期滞在を前提とした有人施設。[国際宇宙ステーション](g077.md)（ISS）は高度約400kmの低軌道を周回し、中国の天宮も独自に運用されている。

**ローバー・ランダー**は天体表面で活動する探査機。火星では複数のローバー（キュリオシティ・パーサヴィアランス）が地質調査を継続中だ。

**[ラグランジュポイント](g049.md)への設置**は燃料効率の高い選択肢として注目される。L2はジェイムズ・ウェッブ[宇宙望遠鏡](g057.md)（g057）が配置され、地球・太陽・月の熱源から離れた安定した環境を確保している。L1はSOHO（太陽観測衛星）が太陽を常時監視する位置として使われる。L4・L5は安定点のため将来の[宇宙コロニー](g081.md)や燃料補給基地の候補として議論されている。[ソーラーセイル](g054.md)（g054）はこれら人工天体の推進技術の一つとして研究が進む。
