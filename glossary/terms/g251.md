# 情報理論（Information Theory）

**Information Theory**  物理学 / 情報科学・数理物理 / g251
読み: じょうほうりろん　関連: —

クロード・シャノンが1948年に創始した、情報の定量化・圧縮・伝送を扱う数理理論。「情報量（ビット）」「[エントロピー](g008.md)（情報の不確かさ）」「チャネル容量」を基本概念とする。

ランダウアーの原理により情報の消去には必ずkT ln2 の熱が発生することが示され、[熱力学](g250.md)（g250）と深く結びつく。[マクスウェルの悪魔](g059.md)問題（悪魔が情報を取得・消去する際のエネルギー収支）はこの原理で解決され、[カオスの悪魔](g210.md)方程式（[wiim_052](../../docs/physics/wiim_052.md)）の議論の背景にある。
