# 活動銀河核（Active Galactic Nucleus）

**Active Galactic Nucleus**  天文学・宇宙論 / 天文学・銀河物理・高エネルギー天文学 / g319
読み: かつどうぎんがかく　関連: —

銀河の中心部に存在し、異常に高い光度を放射する[天体](g123.md)領域。英語の頭文字からAGNとも呼ばれる。その正体は超大質量ブラックホール（[太陽](g084.md)の数百万〜数十億倍）とその周囲に形成された降着円盤であり、ガスや塵が螺旋状に落ち込む際に重力エネルギーが電磁波として解放される。

AGNは放射の強度や観測波長帯によって複数の種類に分類される。光学的に明るく点状に見えるものは**クエーサー**（quasi-stellar object）と呼ばれ、宇宙初期の遠方宇宙に多く観測される。電波を強く放射するものは**電波銀河**、比較的低光度なものは**セイファート銀河**と分類される。これらはブラックホールの質量・回転・降着率・観測角度の違いにより異なる姿を見せているという「統一モデル」が現在の主流見解だ。

活動銀河核は宇宙最大規模のエネルギー源の一つであり、銀河全体の星形成史や進化にも影響を与えると考えられている。
