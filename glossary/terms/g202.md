# ド・ブロイ波（de Broglie Wave）

**de Broglie Wave**  量子物理学 / 量子力学 / g202
読み: どぶろいは　関連: —

物質粒子が示す波動的性質を表す概念。1924年にルイ・ド・ブロイが提唱し、すべての粒子は波長 λ = h/p（h: プランク定数、p: 運動量）をもつ「物質波」として振る舞うとした。

粒子の運動量が大きいほど波長は短くなるため、[電子](g064.md)のような軽い粒子では干渉・回折が実験で観測されるが、野球ボールのような巨視的物体では波長が[原子核](g114.md)より遥かに小さく、波動性は事実上無視できる。この波動性は[量子力学](g253.md)の[波動関数](g164.md)の起源であり、二重スリット実験における干渉縞や電子顕微鏡の動作原理の基礎をなす。

[ボース・アインシュタイン凝縮](g196.md)（g196）では多数の粒子のド・ブロイ波長が熱的ド・ブロイ波長と同程度になることで波動関数が巨視的に重なり、集団的なコヒーレンスが生まれる。
