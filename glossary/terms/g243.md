# コヒーレンス長（Coherence Length）

**Coherence Length**  物理学 / 光学・波動物理学 / g243
読み: こひーれんすちょう　関連: [位相](g242.md)

光や電磁波が互いに干渉できる最大の光路差。ある光源から出た光は完全に単一の周波数・[位相](g242.md)（g242）を持つわけではなく、わずかな周波数揺らぎ（線幅）を持つ。この揺らぎが大きいほど位相の相関が崩れやすく、コヒーレンス長は短くなる。

単色性の高いレーザーのコヒーレンス長は数メートル〜数百メートルに達するが、白色光（[太陽](g084.md)光・電球）は数マイクロメートル程度しかない。[干渉計](g073.md)・ホログラフィー・光コヒーレンストモグラフィー（OCT）では必要な光路差に応じてコヒーレンス長が設計基準になる。

[リングレーザー](g244.md)（g237）のサニャック干渉計では、長いコヒーレンス長を持つレーザーが必要であり、[菌糸](g112.md)リン[グレーザー](g066.md)のような超長光路設計ではコヒーレンス長の維持が重要な課題となる。
