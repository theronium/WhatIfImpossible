# パランティ粒子生成技術（Paranti-Particle Production Technology）

**Paranti-Particle Production Technology**  WIIM派生技術 / 架空粒子工学・防衛技術 / g261
読み: ぱらんてぃりゅうしせいせいぎじゅつ　関連: [wiim_038](../../docs/physics/wiim_038.md)

[パランティ粒子](g161.md)（g161）を制御可能な形で生成し安定的に供給する架空の技術体系。[エキゾチック物質](g068.md)（g068）の精密配置と[レトロン](g163.md)（g163）の[エントロピー](g008.md)操作を組み合わせることで、CPT変換と独立したエネルギー反転（E共役）を持つ粒子を作り出す。

静かな対消滅（[wiim_038](../../docs/physics/wiim_038.md)）の実現に不可欠な上流工程であり、完全防御・完全ステルスという防衛応用の入口にあたる。根本的な障壁は「安定した負エネルギー状態が量子場理論に存在しない」という理論的問題であり、パランティ粒子の生成経路自体が標準理論の外側にある。
