# ベイズ定理（Bayes' Theorem）

**Bayes' Theorem**  数学・論理・幾何学 / 確率論・統計学・認識論 / g088
読み: べいずていり　関連: —

**別名**: ベイズの定理 / ベイズ推論

新しい証拠が得られるたびに、ある仮説の確からしさ（確率）を更新する数学的な枠組み。18世紀のイギリスの牧師・数学者トーマス・ベイズが考案し、ピエール＝シモン・ラプラスが発展させた。式は P(A|B) = P(B|A)×P(A) / P(B) と表され、「事後確率は尤度と事前確率の積に比例する」という意味を持つ。

直感的な例として、スパムフィルタがある。「このメールに〈無料〉という単語が含まれる」という新たな証拠を得たとき、そのメールがスパムである確率がどれだけ上がるかを過去のデータから計算し、単語が増えるたびに確率を更新し続ける——これがベイジアンフィルタの[原理](g306.md)だ。

医療診断でも重要で、検査で陽性反応が出たとしても「実際に病気である確率」は病気の母集団での発生率（事前確率）によって大きく変わる。珍しい病気ほど、陽性でも実は健康である確率が高い——これを理解しないと検査結果を過大評価してしまう。

科学哲学の観点では、ベイズ推論は「観測によって理論の信頼度を更新し続ける」という帰納的推論の数学的基盤を提供する。[シミュレーション仮説](g019.md)や[人間原理](g021.md)のような思考実験においても、「どれだけの証拠があればその仮説を信じるべきか」を定量的に論じる道具として機能する。
