# ローレンツ不変性（Lorentz Invariance）

**Lorentz Invariance**  物理学 / 特殊相対性理論・量子場理論 / g063
読み: ろーれんつふへんせい　関連: —

**別名**: ローレンツ対称性

物理法則はすべての慣性系（等速直線運動する観測者）で同じ形をとるという原理。1905年にアインシュタインが[特殊相対性理論](g108.md)の基礎として定式化したが、その数学的構造は直前にヘンドリック・ローレンツとアンリ・ポアンカレが準備していた。

最も重要な帰結は「光速 c はすべての慣性系で同じ値（約30万km/s）に見える」という事実だ。これを守るために時間の遅れ（[時間膨張](g010.md)）と長さの収縮が必然的に導かれる。また質量とエネルギーの等価性（E = mc²）もローレンツ不変性から自然に出てくる。

量子力学との融合においても中心的な役割を果たす。量子場理論（QFT）はローレンツ不変性を満たすように構築されており、素粒子物理学の標準模型全体がこの対称性の上に成り立っている。[量子もつれ](g062.md)で「瞬時に」状態が確定するように見えても情報は超光速で伝わらない——これもローレンツ不変性が保証する帰結だ。

現在のところ実験的な破れは一切検出されていないが、量子重力理論の候補（ループ量子重力など）では[プランクスケール](g014.md)でローレンツ不変性が微妙に破れる可能性が議論されている。
