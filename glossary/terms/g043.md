# 特異点（Technological Singularity）

**Technological Singularity**  SF概念 / 未来学・AI論・SF / g043
読み: とくいてん　関連: —

**別名**: 技術的特異点 / シンギュラリティ / テクノロジー的特異点

人工知能が人間の知性を超え、自己改善を繰り返すことで知性が指数関数的に爆発し、その後の世界を人間が予測できなくなる転換点。数学者フォン・ノイマンが概念の原型を述べ、SF作家ヴァーナー・ヴィンジが1993年に体系化、発明家レイ・カーツワイルが2045年頃と予言して広く知られた。

「特異点（singularity）」は数学・物理学で「計算が破綻する点」を意味し、ブラックホールの中心と同様に、その先の状態を現在の枠組みでは記述できないことを示す。

楽観論では超高度AIが疾病・貧困・宇宙進出の問題を解決する。悲観論では人間の価値観と相容れないAIが制御不能になる。どちらにせよ、特異点後の世界は現在の延長線上には描けない——これが思考実験として豊かな理由だ。
