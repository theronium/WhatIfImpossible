# 第五の力（Fifth Force）

**Fifth Force**  仮説・未観測の粒子 / 素粒子物理学・理論物理学 / g115
読み: だいごのちから　関連: —

**別名**: 第５の力 / 第5の力 / 第五の相互作用

重力・電磁力・強い力・弱い力という4つの基本相互作用に加えて存在すると仮説される、未知の第五の力。標準模型では説明できない現象——[ダークマター](g056.md)（g056）や[ダークエネルギー](g218.md)の振る舞い、宇宙の加速膨張、重力と量子力学の統合困難——を説明する候補として複数の理論が提唱されている。

代表的な候補として、**クインテッセンス**（宇宙のダークエネルギーを生み出すスカラー場）、**ディラトン**（超弦理論（g093）から導かれるスカラー粒子）、**B-Lゲージボソン**（[バリオン](g114.md)数とレプトン数の差に結合する力）などが挙げられる。2016年・2019年にハンガリーの研究チームが発見を主張した**X17粒子**（質量約17 MeVのボソン）も候補の一つとして注目された。

第五の力は一般に非常に弱く・短距離に限られるため、実験での検出が極めて難しい。[大統一理論](g093.md)（g093）や余剰次元理論はこの力を自然に組み込む枠組みを提供しており、WhatIfImpossibleの思考実験では「第五の力を制御できたら？」という問いが[エキゾチック物質](g068.md)（g068）の代替手段として浮上する。
