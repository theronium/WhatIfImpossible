# 軌道エレベーター（Space Elevator）

**Space Elevator**  SF概念 / SF・宇宙工学・メガストラクチャー / g433
読み: きどうえれべーたー　関連: —

地表と[静止軌道](g077.md)（高度約3万6千km）を繋ぐ超高強度ケーブルで、ロケットを使わずに宇宙へアクセスする構想上の構造物。アーサー・C・クラークが1979年の小説『楽園の泉』で普及させたが、構想自体は1895年のツィオルコフスキーに遡る。

[原理](g306.md)は単純で、静止軌道上のカウンターウェイトへ垂らしたケーブルを地球の自転による遠心力が張った状態に保ち、カゴが昇降して宇宙へのアクセスコストを大幅に削減できると考えられている。

最大の障壁は素材の引張強度だ。必要な比強度はケブラーの約50倍で、カーボンナノチューブ（理論値）がかろうじて届く水準にある。デブリ衝突・落雷・地震・[コリオリ力](g051.md)による横ぶれ、赤道限定の地理的制約、ケーブル切断時の崩壊エネルギーが地球全周を覆う規模という試算もある。

[カルダシェフスケール](g002.md)のタイプI移行を加速させる技術として、「宇宙の民主化」の象徴と語られ続けている。
