# マグナス効果（Magnus Effect）

**Magnus Effect**  物理学 / 流体力学・航空工学 / g370
読み: まぐなすこうか　関連: [wiim_091](../../docs/physics/wiim_091.md)

**別名**: マグナス力 / マグナス回転

回転する物体が流体（気体・液体）中を移動するとき、回転軸と移動方向の両方に垂直な揚力が発生する現象。回転によって物体の片側では流速が加速し、反対側では減速することで圧力差が生じる（ベルヌーイの[原理](g306.md)による）。野球のカーブボール・卓球のスピン・サッカーのバナナシュートがその代表例。

宇宙工学的応用として、大気上端をかすめる軌道で高速自転する物体は、大気との相互作用でマグナス揚力が生じ再突入を遅らせる効果が期待される。フレットナーローターを搭載した船舶では回転円筒が帆の代わりに風の力を揚力へ変換し実用化されている。
