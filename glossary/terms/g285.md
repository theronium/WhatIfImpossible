# ミンコフスキー距離（Minkowski Distance）

**Minkowski Distance**  物理学 / 数学・幾何学・情報科学 / g285
読み: みんこふすきーきょり　関連: [ユークリッド距離](g283.md), [マンハッタン距離](g284.md)

**別名**: Lpノルム / p-ノルム

[ユークリッド距離](g283.md)と[マンハッタン距離](g284.md)を1つのパラメータ p で統一的に記述する距離の一般形。2点間の距離を (|x₂-x₁|ᵖ + |y₂-y₁|ᵖ)^(1/p) で定義する。

p=1 にするとマンハッタン距離（g284）、p=2 にするとユークリッド距離（g283）と一致する。p を無限大に近づけると各座標差の最大値だけが残り、チェビシェフ距離（L∞距離）になる。つまり「距離の形」はpという1つの数で連続的に変形でき、p=1の四角い等距離線からp=2の円形を経てp=∞の正方形へと滑らかに移り変わる。

機械学習では問題に合わせてpを調整することで、外れ値への感度や高次元空間での性質を制御できる。「どの距離で測るか」が分析結果そのものを変えるという事実は、距離の選択が暗黙の前提として潜んでいることを示している。
