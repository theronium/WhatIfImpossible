# デバイカットオフ（Debye Cutoff）

**Debye Cutoff**  物理学 / 固体物理学・量子音響学 / g415
読み: でばいかっとおふ　関連: —

結晶格子中を伝播できる[フォノン](g365.md)（格子振動の量子）の波長・周波数に存在する上限。格子間隔（[原子](g114.md)間距離、~0.1 nm）より短い波長のフォノンは物理的に意味をなさないため、これに対応する周波数（デバイ周波数）以上のモードは存在しない。デバイモデルが固体の低温比熱を正確に記述するうえでの核心をなす。

結晶内の音波はこのカットオフで「打ち切られ」、それ以上の高周波では光学フォノン分岐に移行するか伝播しない。この構造は宇宙の音響スペクトルが持つ紫外カットオフ——[プランクスケール](g014.md)（~10⁻³⁵ m）——の結晶スケール類似として理解できる。

ただし両者の帰結は異なる。デバイカットオフ以下では別の振動モードが現れるが、プランクスケール以下への閉じ込めは[シュワルツシルト半径](g048.md)の論法で微小ブラックホールを生成しエネルギーが不可逆に吸収される。エバネッセント的な消失ではなく破壊的な吸収境界として機能する。
