# ピエゾアンキロン効果（Piezoankyronic Effect）

**Piezoankyronic Effect**  WIIM概念 / 架空物理効果・一般相対性理論 / g317
読み: ぴえぞあんきろんこうか　関連: [wiim_078](../../docs/cosmology/wiim_078.md), [wiim_022](../../docs/physics/wiim_022.md)

WhatIfImpossibleの思考実験（[wiim_078](../../docs/cosmology/wiim_078.md)）で命名された架空の物理効果。「圧電効果（ピエゾ効果）」と「[アンキロン](g128.md)（g128）」を組み合わせた造語。

圧電効果が結晶格子の物理的変形を電気エネルギーに変換するように、ピエゾアンキロン効果は時空の計量テンソルの変形をエネルギーに変換することを仮定する。具体的には、アンキロン類似の場が計量座標に「錨」を打ち、宇宙膨張（[ダークエネルギー](g218.md)）がその錨を引き剥がそうとする際に生じる「計量的歪み」を動力として回収する機構だ。

この効果を実装する素子を**ピエゾアンキロン素子**と呼ぶ。素子は局所空間にアンキロン類似の場を展開し、膨張による歪みを変換素子が受けてエネルギーとして取り出す。圧電素子の時空版と見なすことができる。

ただし成立条件として「計量錨の維持コスト＜回収エネルギー」が必要であり、計量テンソルへの継続的な干渉コストが回収量を超えると装置はエネルギー赤字に陥る。現行の物理学ではこの収支を見積もる手段がない。
