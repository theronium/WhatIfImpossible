# 世界五分前仮説（Five-minute Hypothesis）

**Five-minute Hypothesis**  哲学・存在論 / 認識論・哲学 / g027
読み: せかいごふんまえかせつ　関連: —

**別名**: 五分前仮説

「世界は5分前に、過去の[記憶](g180.md)や痕跡をすべて含んだ状態で突然創造されたかもしれない」という思考実験。哲学者バートランド・ラッセルが1921年の著作『心の分析』で提示した。「5分」は象徴的な数字であり、「つい先ほど」を意味する。

ポイントは反証不可能性だ。あなたの記憶・化石・光の到達・書物——あらゆる「過去の証拠」は、創造の瞬間から最初からそこにあった可能性がある。どんな証拠を持ち出しても「それも創造時から存在した」と言われれば論理的に否定できない。

デカルトの「悪魔の仮説」や[シミュレーション仮説](g019.md)と構造的に同型だが、この仮説が特に鋭いのは「過去」という概念そのものの信頼性を問う点だ。[ボルツマン脳](g040.md)（熱揺らぎで記憶ごと生まれる意識）とも深く重なる——どちらも「今この瞬間の意識だけが確かで、それ以前は存在しなかったかもしれない」という問いを突き付ける。
