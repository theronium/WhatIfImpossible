# ミュータント（Mutant）

**Mutant**  生物学・進化 / 遺伝学・進化生物学・SF / g121
読み: みゅーたんと　関連: [wiim_008](../../docs/biology/wiim_008.md)

**別名**: 突然変異体 / 突然変異 / 変異体 / 変異種 / 変異

遺伝子に変異（ミューテーション）を持つ生物の総称。変異の原因は放射線・化学物質・[ウイルス](g116.md)（g116）の挿入・DNA複製エラーなど多岐にわたる。変異の種類は塩基1個の置換（点変異）から染色体レベルの大規模な再編まで幅広く、タンパク質の機能変化・発現量の変化・まったく無影響など結果も様々だ。

変異は進化（g029）の原材料であり、[自然選択](g181.md)によって有利な変異が集団に広まることで種が適応する。大半の変異は中立か有害だが、環境が変われば「有害」だった変異が「有利」に転じることもある——鎌状赤血球変異がマラリア地域では生存率を高める例が典型だ。

SFでは放射線などによって劇的な表現型変化を持つ個体（念動力・再生能力など）として描かれることが多いが、実際の変異は1塩基〜数塩基レベルの微細な変化であり、単一の変異が複雑な超能力を生むことはない。ただし**[エピジェネティクス](g183.md)**（遺伝子の発現制御）や複数変異の相互作用は予測困難な表現型を生むことがあり、SF的想像の完全な否定もできない。

WhatIfImpossibleでは、宇宙放射線下での菌類の急速な変異（[wiim_008](../../docs/biology/wiim_008.md)）や、[マイコプラズマギカ](g130.md)（[wiim_024](../../docs/biology/wiim_024.md)）における設計されたゲノム変異が関連する。
