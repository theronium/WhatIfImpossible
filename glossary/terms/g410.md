# 磁歪効果（Magnetostriction）

**Magnetostriction**  物理学 / 磁性体物理・音響工学 / g410
読み: じわいこうか　関連: —

強磁性体を磁場中に置いたとき、磁場の変化に応じて材料が伸縮する現象。1842年にジェームズ・ジュールが発見した。電磁場のエネルギーが機械的歪みに直接変換されるため、磁場→振動→音波という変換経路の基礎をなす。

身近な例として、電力変圧器が発する低いうなり音は鉄芯の磁歪振動によるものである。磁歪量が磁場の方向によらず正・負両ピークで同方向に生じるため、振動周波数は商用周波数の2倍（100または120Hz）になる。逆効果（逆磁歪・ヴィラリ効果）として外部応力が加わると磁化が変化する現象もあり、力センサーや地磁気計に応用される。

音響工学では磁歪振動子が軍事用SONAR送受波器の標準部品として使われてきた。[SASER](g400.md)の文脈では、磁歪を量子レベルで制御することで「磁場が駆動する[フォノン](g365.md)増幅器」の実現可能性が議論されている。
