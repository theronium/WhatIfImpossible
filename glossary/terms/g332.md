# ジオデシック構造（Geodesic Structure）

**Geodesic Structure**  物理学 / 幾何学・建築・ウイルス学 / g332
読み: じおでしっくこうぞう　関連: —

球面上の最短経路（[測地線](g296.md)）を辺とする多角形で球面を分割した構造。正三角形・正五角形・正六角形の組み合わせで球面を覆い、全方向に均一な剛性を持つ最効率の殻構造となる。建築家バックミンスター・フラーがドーム建築として普及させた。

工学ではレーダードーム・大型天文台ドーム・宇宙居住モジュールの候補設計に用いられる。自然界では[ウイルス](g116.md)のカプシド（外殻タンパク質の正二十面体対称配列）がジオデシック[原理](g306.md)に従っており、最少のタンパク質量で最大容積を囲む最適解として進化している。放散虫の鉱物質外骨格も同原理の球殻を形成する。
