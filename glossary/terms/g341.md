# 逆カシミール装置（Inverse Casimir Device）

**Inverse Casimir Device**  WIIM派生技術 / 架空装置・量子真空工学・エキゾチック物質工学 / g341
読み: ぎゃくかしみーるそうち　関連: [wiim_085](../../docs/quantum/wiim_085.md), [wiim_086](../../docs/physics/wiim_086.md)

[カシミールフォージ](g133.md)（g133）の逆転設計として想定される架空装置。通常のカシミールフォージが多層共振器で真空モードを抑制・同期して[エキゾチック物質](g068.md)（負エネルギー密度）を生成するのに対し、逆カシミール装置は抑制されたモードを板の外側の焦点に集束させることで、[ディラック海](g145.md)（g145）の充填[電子](g064.md)を選択的に抽出する操作を目的とする。

抽出された充填電子はパランティ電子（[パランティ粒子](g161.md)（g161）の電子版）として振る舞い、通常の電子との静かな対消滅（[wiim_038](../../docs/physics/wiim_038.md)）を引き起こす。この性質から二つの用途が想定される。一つはパランティ電子冷却（[wiim_086](../../docs/physics/wiim_086.md)）——対象物質の熱電子と対消滅させることで電子蒸発冷却を実現し、[テルモクラシス板](g195.md)（g195）の設定閾値温度 T_th では届かない極低温域を補完する。もう一つは陽電子砲の弾薬製造——穴の引き出し方を切り替えることで同一設備から陽電子を生成できる可能性がある。

根本的な障壁は、真空モードの焦点集束が工学的に実現可能かどうかが理論的にも未解明であることだ。カシミールフォージと同様に[カルダシェフスケール](g002.md)・タイプII文明に近いエネルギーが必要と考えられ、宇宙船搭載用途ではエネルギーコストの逆転（抽出エネルギー << 稼働コスト）が最大の問題となる。
