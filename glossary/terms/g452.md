# 散逸構造（Dissipative Structure）

**Dissipative Structure**  物理学 / 熱力学・非線形物理学 / g452
読み: さんいつこうぞう　関連: —

大域的な不均衡（エネルギー勾配・物質流）の中で系が自発的に形成する秩序ある局所構造。イリヤ・プリゴジンが提唱し、1977年ノーベル化学賞の受賞につながった概念。

熱平衡から遠く離れた開放系が示す[自己組織化](g028.md)現象であり、外部からエネルギーを受け取り廃熱を排出し続けることで内部の秩序を維持する。ベナール対流セル（加熱された流体に生まれる六角形の規則的渦）や生物の細胞がその代表例だ。

重要な点は、散逸構造が均衡に「達する」のではなく、不均衡を消費しながら均衡を「演じ続ける」ことだ。大域的な不均衡が局所的な均衡を生み出し、その局所均衡がさらに高次の不均衡の構成要素となる——この入れ子構造が生命・生態系・宇宙の階層的秩序の基盤と考えられる。
