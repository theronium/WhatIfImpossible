# 菌糸（Mycelium / Hypha）

**Mycelium / Hypha**  生物学・進化 / 生物学・菌類学・生態学 / g112
読み: きんし　関連: [wiim_008](../../docs/biology/wiim_008.md), [wiim_017](../../docs/biology/wiim_017.md), [wiim_018](../../docs/biology/wiim_018.md), [wiim_059](../../docs/biology/wiim_059.md)

**別名**: 菌糸体 / マイセリウム / ハイファ

菌類の基本的な構造単位である管状の細胞列。個々の管状細胞を「ハイファ（hyphae）」、その集合体を「菌糸体（マイセリウム）」と呼ぶ。肉眼で見えるキノコは菌糸体の一部が生殖のために地上に現れた「子実体」にすぎず、菌の本体は土壌中に広がる菌糸体だ。

菌糸は先端成長（apical growth）によって伸長し、障害物を回避しながら栄養源を探索する。この成長パターンは分散型コンピューティングや最短経路探索アルゴリズムの着想源にもなっている。菌糸の直径はわずか数マイクロメートルだが、一本の菌糸体が持つ総延長は数十キロメートルに及ぶこともある。

植物の根に共生する菌糸は菌根ネットワークを形成し、森林全体で栄養・情報・化学シグナルを輸送する。この巨大ネットワークは「[ウッドワイドウェブ](g031.md)」と呼ばれ、分散型知性の原型とも見なされる。菌糸体が宇宙空間で分散知性に進化する可能性を探った思考実験（[wiim_008](../../docs/biology/wiim_008.md)）や、[胞子](g111.md)（g111）による[惑星](g123.md)[テラフォーミング](g042.md)（[wiim_017](../../docs/biology/wiim_017.md)・[wiim_018](../../docs/biology/wiim_018.md)）において、菌糸は生態系構築の基盤として中心的役割を担う。
