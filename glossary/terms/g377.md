# アーンショーの定理（Earnshaw's Theorem）

**Earnshaw's Theorem**  物理学 / 電磁気学・ポテンシャル論 / g377
読み: あーんしょーのていり　関連: [wiim_091](../../docs/physics/wiim_091.md)

逆二乗則に従う力（静電気・静磁気・重力）だけで構成された系において、荷電体・磁性体を静的に安定して保持することは不可能であると示した[定理](g304.md)（1842年、サミュエル・アーンショー）。ラプラス方程式を満たすポテンシャル場には局所的な極大・極小が存在できないことが数学的根拠であり、安定な釣り合い点があるように見えても、わずかな摂動でいずれかの方向へ逸脱してしまう。

例外は存在する。反磁性体（ビスマス・水など）や[超電導](g197.md)体の[マイスナー効果](g198.md)は動的な磁束排除を伴うため定理の前提外となり、安定な磁気浮上が可能となる。回転ジャイロ安定化（レビトロン）や能動フィードバック制御も例外に数えられる。磁力型極[衛星](g123.md)（[wiim_091](../../docs/physics/wiim_091.md)）で「不安定平衡」が根本的な問題となる背景もこの定理にある。
