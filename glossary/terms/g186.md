# ユニバース25（Universe 25）

**Universe 25**  生物学・進化 / 行動生物学・生態学 / g186
読み: ゆにばーすにじゅうご　関連: [自然選択](g181.md), [断続平衡説](g184.md)

**別名**: カルホーン実験 / ネズミのユートピア実験

ジョン・B・カルホーンが1968年から行った一連の実験の中で最も有名な第25回目の試み。食料・水・巣材が無限に供給され、捕食者も病気もない完全な「ネズミのユートピア」を構築し、その後の社会動態を観察した。

実験の経過は四段階に分かれる。最初の増殖期には個体数が急増する。やがて密度の上昇とともに社会秩序が崩壊し——攻撃行動、無秩序な性行動、子育て放棄が蔓延する「行動の沈滞（behavioral sink）」が起きる。その中から**「美しい者たち（Beautiful Ones）」**と呼ばれる個体群が出現する。彼らは争いに参加せず、交尾も育児も行わず、ひたすら毛づくろいと食事だけをする——見た目は完璧だが社会的機能を喪失した存在だ。最終段階では個体数がゼロに向かって収束し、密度を下げても回復しなかった。

カルホーンは「社会的役割の枯渇」が崩壊の本質だと主張した。空間ではなく**意味のある社会的ポジションが有限**であり、それが埋まった時点で新参個体は正常な行動を学ぶ機会を失う——という解釈だ。

ただし現在この実験の解釈には批判もある。ネズミの行動をそのまま人間社会に適用する過度な一般化、人工環境の特殊性（実際の生態系では捕食・疾病・飢餓が密度を調整する）、「Beautiful Ones」をひきこもりや現代的社会離脱の比喩として使う文化的転用への疑問などだ。また実験の完全な再現に成功した例は限られており、結果の普遍性については慎重な見方もある。

WIIM的に興味深いのは「選択圧がなくなった環境が何を生むか」という問いだ。[自然選択](g181.md)（g181）が停止した先に待つのは楽園ではなく、社会的意味を失った存在の出現かもしれない。
