# 超低温・レーザー冷却技術（Cryogenic / Laser Cooling Technology）

**Cryogenic / Laser Cooling Technology**  量子物理学 / 低温物理学・量子光学 / g256
読み: ちょうていおんれーざーれいきゃくぎじゅつ　関連: —

**別名**: 超低温技術 / レーザー冷却

物質をケルビン換算でミリケルビン以下の極低温に冷却する技術群。希釈冷凍機・断熱消磁冷却がハードウェア手法の主流であり、レーザー冷却（[光子](g064.md)の運動量反動を使って[原子](g114.md)の熱運動を抑える手法）と組み合わせることでナノケルビン域も実現できる。

BEC（ボーズ・アインシュタイン凝縮）の実現に不可欠であり、マクロ量子状態ブランチ（[wiim_050](../../docs/quantum/wiim_050.md)関連）の前提技術として技術ツリーに登場する。また、[ネゴトロン](g247.md)（g247）のコヒーレンス時間延長策として照射経路全体への超低温環境展開も検討される。
