# オウムアムア（'Oumuamua）

**'Oumuamua**  天文学・宇宙論 / 天文観測・恒星間天体・SETI / g463
読み: おうむあむあ　関連: [wiim_008](../../docs/biology/wiim_008.md)

**別名**: 1I/2017 U1

2017年10月に観測された史上初の[恒星](g084.md)間天体。正式名称1I/2017 U1。ハワイ語で「遠方から来た使者」を意味し、ハワイのパンスターズ望遠鏡が発見した。長軸100〜800mの葉巻型または極端に扁平な円盤型と推定されており、通常の彗星・[小惑星](g123.md)とは形状が大きく異なる。

最大の謎は太陽最接近後に示した異常な非重力加速だ。通常の彗星であれば揮発性物質のアウトガスで説明できるが、同等の放出が観測されなかった。この説明として提唱された仮説は多岐にわたり、[水素](g114.md)氷山・窒素氷板から、恒星間文明の[探査機](g076.md)（[エイリアン](g122.md)説）まで含む。

WhatIfImpossibleの文脈では、[コズミックマイス](g134.md)が[菌糸](g112.md)コロニーを表面に持つ岩塊を摂動させ別の[重力圏](g107.md)へ投射する「生物的恒星間播種」の思考実験における実在の参照事例となる。[タビーの星](g462.md)と同様に「自然現象か知的設計か」の判定困難性を体現している。
