# ゴーレム（Golem）

**Golem**  SF概念 / ユダヤ神秘主義・民間伝承・人工生命思想 / g118
読み: ごーれむ　関連: —

**別名**: 人工生命の原型 / シェム / エメット

ユダヤ教の伝承に登場する、粘土や土から作られた人工の存在。額や舌の下に**シェム**（神の名を記した羊皮紙）または「エメット（真実）」の文字を置くことで生命が宿り、「エメット」の最初の文字を消して「メット（死）」にすると動作が止まる。最も有名なのは16世紀プラハのラビ・レーヴが作ったとされるゴーレムで、ユダヤ人コミュニティを守護したという伝説が残る。

ゴーレムは**人工知能・合成生命・ロボット**という現代的概念の原型として広く参照される。「言語（コード）が物質に命を吹き込む」という構造は、DNAプログラムによる合成生物（[wiim_024](../../docs/biology/wiim_024.md)）や機械学習モデルへの命令との類比として語られることも多い。カバラの知識体系（[セフィロト](g119.md)）に精通した者だけがゴーレムを作れるとされた点も、「創造には深い構造理解が必要」という認識論的示唆を含む。

[錬金術](g117.md)（g117）と同時代のヨーロッパ・ユダヤ神秘主義の産物であり、両者は「人間が神の領域（創造）に踏み込む」という共通の問いを持つ。
