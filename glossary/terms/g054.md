# ソーラーセイル（Solar Sail）

**Solar Sail**  天文学・宇宙論 / 宇宙推進工学・宇宙探査 / g054
読み: そーらーせいる　関連: —

**別名**: 光帆 / 光子帆船 / 太陽帆船

太陽光の放射圧を帆に受けて推進力を得る宇宙航行技術。燃料を一切消費しないため、長期間の加速が可能で理論上は非常に高速に達しうる。[光子](g064.md)（光の粒子）が帆に衝突・反射する際の運動量変化が推力の源であり、1光子1回の力は極めて微小だが、巨大な帆面積と時間の積み重ねで実用的な加速が得られる。

推力は太陽からの距離の2乗に反比例して弱まるが、燃料切れがないという利点は外[惑星](g123.md)探査や[恒星](g084.md)間飛行の構想において大きな強みとなる。実際の運用例として、JAXAの「IKAROS」（2010年）が世界初の実証に成功し、惑星間空間でのソーラーセイル航行を実現した。米国の「ライトセイル2」（2019年）も[低軌道](g077.md)での軌道変更を実証している。

恒星間探査の構想「ブレークスルー・スターショット」では、強力なレーザーをセイルに照射することで光速の20%まで加速し、α星ケンタウリへ20年以内に到達させる計画が検討されている。ソーラーセイルはSFの定番技術でもあり、アーサー・C・クラークの短編「太陽からの風」（1964年）が広く知られる。
