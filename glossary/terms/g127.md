# コーラ粒子（Chora Particle）

**Chora Particle**  WIIM概念 / 架空粒子・時空間物理学 / g127
読み: こーらりゅうし　関連: [wiim_013](../../docs/physics/wiim_013.md), [wiim_050](../../docs/quantum/wiim_050.md), [wiim_063](../../docs/physics/wiim_063.md), [wiim_072](../../docs/quantum/wiim_072.md), [wiim_074](../../docs/physics/wiim_074.md)

**別名**: 空間超越粒子 / χώρα粒子

WhatIfImpossibleの思考実験（[wiim_013](../../docs/physics/wiim_013.md)）で命名された架空粒子。プラトンが対話篇『ティマイオス』で「空間・場所の器」を意味するために用いた古代ギリシャ語「χώρα（chora）」に由来する。

[光子](g064.md)が時間を超越する（[固有時間](g174.md)ゼロ）のと対称的に、**時間は経過するが空間を経由せずに別の場所へ出現する**粒子として定義される。いわば「光子の逆数」であり、時空の非対称性——なぜ宇宙は時間超越粒子を許し空間超越粒子を許さないのか——という根本的な問いを提起する。

その振る舞いの根拠として、コーラ粒子は**[波動関数](g164.md)（g164）に従わない**という性質が考えられる。通常の量子粒子も波動関数の裾は理論上無限遠まで延びるが、出現確率は距離とともに指数関数的に減衰する。コーラ粒子にはこの減衰機構が存在しない——「遠くに現れることへの抑制」がない粒子だ。波動関数に従わないため、[デコヒーレンス](g173.md)（g173）もまた適用されない。通常の量子コヒーレンスが熱雑音や環境との相互作用で失われるのは、波動関数の重ね合わせが壊れる現象だからだ。

物理的解釈としては[余剰次元](g255.md)を内在的な近道として利用するという[弦理論](g093.md)的枠組みが考えられるが、「空間だけ超越して時間は経過する」という条件は[特殊相対性理論](g108.md)の同時性の相対性と衝突する。出現位置が量子確率的であれば[因果律](g017.md)との折り合いはつくかもしれないが、確定的な情報を運べるなら光速を超えた情報伝達となり相対性理論と直接矛盾する。[アンキロン](g128.md)（g128）・[ネゴトン](g126.md)（g126）と並ぶWhatIfImpossible架空粒子の一つ。
