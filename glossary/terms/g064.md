# ボソン／フェルミオン（Boson / Fermion）

**Boson / Fermion**  物理学 / 量子統計力学・素粒子物理学 / g064
読み: ぼそん／ふぇるみおん　関連: [パウリの排他原理](g060.md), [wiim_020](../../docs/physics/wiim_020.md)

**別名**: ボース粒子 / フェルミ粒子 / ボゾン / 光子 / グルーオン / W粒子 / Z粒子 / 電子 / クォーク / ニュートリノ

素粒子をスピン（自転に相当する量子数）の値で二分した分類。スピンが整数（0, 1, 2…）の粒子を**ボソン**、半整数（1/2, 3/2…）の粒子を**フェルミオン**と呼ぶ。

ボソンは[パウリの排他原理](g060.md)に縛られず、同じ量子状態に何個でも入ることができる。この性質が**ボース＝アインシュタイン凝縮**を生む——極低温で多数のボソンが同一の最低エネルギー状態に落ち込み、量子効果が巨視的スケールで現れる超流動・[超伝導](g197.md)の根源となる。代表例は光子・[グルーオン](g114.md)・Wボソン・Zボソン・[ヒッグス粒子](g070.md)など「力を伝える粒子」に多い。

フェルミオンはパウリの排他原理に従い、同じ量子状態を二つ以上の粒子が占められない。この制約が原子の電子配置の多様性を生み、元素ごとに異なる化学的性質をもたらす。また[白色矮星](g123.md)・[中性子星](g084.md)の[縮退](g281.md)圧もフェルミオンの性質による。クォーク・電子・ニュートリノなど「物質を構成する粒子」のほとんどがフェルミオンだ。

この分類は素粒子物理学の標準模型の骨格をなしており、「力を運ぶボソン」と「物質を作るフェルミオン」という対比で宇宙の構造を理解する基本的な枠組みとなっている。
