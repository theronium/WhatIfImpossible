# ヒッグス粒子（Higgs Boson）

**Higgs Boson**  素粒子・核物理学 / 素粒子物理学・標準模型 / g070
読み: ひっぐすりゅうし　関連: [ボソン／フェルミオン](g064.md)

**別名**: ヒッグスボソン / 神の粒子

素粒子に質量を与える機構（ヒッグス機構）の鍵となるボソン。1964年にイギリスの物理学者ピーター・ヒッグスらが理論的に予言し、2012年にCERN（欧州[原子核](g114.md)研究機構）の[大型ハドロン衝突型加速器](g074.md)（LHC）で発見された。この発見によりヒッグスとフランソワ・アングレールは2013年のノーベル物理学賞を受賞している。

素粒子物理学の標準模型では、素粒子はヒッグス場という空間全体に満ちた場と相互作用することで質量を獲得すると説明される。ヒッグス場との相互作用が強い粒子ほど重く、まったく相互作用しない粒子（光子・[グルーオン](g064.md)）は質量ゼロになる。ヒッグス粒子はこのヒッグス場の量子（場の振動の粒子）だ。

ヒッグス粒子はスピン0のボソンで、現在確認されている基本粒子の中で唯一スピンゼロを持つ。質量は約125GeV/c²（陽子の約133倍）で非常に重く、LHCのような高エネルギー加速器でなければ生成できない。また非常に不安定で、生成後ただちに他の粒子へ崩壊する。

標準模型の最後の未発見ピースだったヒッグス粒子の発見は理論の完成を意味する一方、なぜヒッグス場が存在するのか・ヒッグス粒子が複数種類あるかどうかなど標準模型を超えた謎も残している。
