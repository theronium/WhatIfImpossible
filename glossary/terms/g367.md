# エンタルピー（Enthalpy）

**Enthalpy**  物理学 / 熱力学 / g367
読み: えんたるぴー　関連: —

[熱力学](g250.md)における状態量の一つ。内部エネルギー U に圧力 P と体積 V の積を加えた量で、H = U + PV と定義される。第一法則（エネルギー保存）から直接導かれ、dU = dQ − PdV において圧力一定とおくと dH = dQ が得られる。エンタルピーは「定圧過程での熱の出入りそのもの」を状態量として切り出した量であり、化学反応や相転移の熱収支を記述するのに広く使われる。

発熱反応では ΔH < 0（系がエネルギーを放出）、吸熱反応では ΔH > 0（系がエネルギーを吸収）。燃焼熱・溶解熱・蒸発熱などはすべてエンタルピー変化として表される。

[エントロピー](g008.md)と並んで熱力学の中心的な状態量であり、ギブズ自由エネルギー（G = H − TS）を介して化学平衡や相平衡の条件を導く。エントロピーが第二法則（不可逆性・方向性）と結びつくのに対し、エンタルピーは第一法則（エネルギー保存）と結びつく。
