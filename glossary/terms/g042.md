# テラフォーミング（Terraforming）

**Terraforming**  SF概念 / 惑星科学・宇宙工学・SF / g042
読み: てらふぉーみんぐ　関連: —

**別名**: 地球化 / 惑星改造

[惑星](g123.md)や衛星の大気・温度・地表環境を人工的に改変し、地球に近い居住可能な状態へ変える技術・構想。「テラ（地球）の形に造る」を意味する造語で、SF作家ジャック・ウィリアムソンが1942年に初めて用いた。

最も議論されるのが火星のテラフォーミングだ。温室効果ガスを大量放出して気温を上げ、CO₂を溶かした液体の水を表面に取り戻し、磁場の喪失（大気散逸の原因）への対処が主な課題とされる。NASAや民間研究者が真剣に検討しているが、完了まで数百〜数千年のスケールが必要と試算される。

倫理的な問いも生じる。もし火星に微生物レベルの生命が存在した場合、テラフォーミングは生態系の破壊にあたるのか——[パンスペルミア](g007.md)仮説とあわせて考えると、惑星改造は同時に「生命圏の侵略」になりうる。
