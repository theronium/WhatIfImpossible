# エントロピー（Entropy）

**Entropy**  物理学 / 熱力学・統計力学 / g008
読み: えんとろぴー　関連: [wiim_005](../../docs/cosmology/wiim_005.md), [wiim_015](../../docs/physics/wiim_015.md)

系の「乱雑さ」「無秩序さ」の度合いを表す量。より正確には、ある巨視的な状態を実現するミクロな配置の数（場合の数）の対数に比例する。

熱力学第二法則は「孤立系のエントロピーは増大するか、一定のままである」と述べる。割れた卵が元に戻らないのは、「割れた状態」を実現する分子配置が「完全な卵」を実現する配置より圧倒的に多いためだ。

エントロピーは時間の方向性（時間の矢）の根拠とされる。過去はエントロピーが低く、未来はエントロピーが高い——この非対称性が「時間の流れ」を生む。
