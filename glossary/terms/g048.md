# シュワルツシルト半径（Schwarzschild Radius）

**Schwarzschild Radius**  天文学・宇宙論 / 一般相対性理論・ブラックホール物理 / g048
読み: しゅわるつしるとはんけい　関連: —

**別名**: シュヴァルツシルト半径 / 重力半径

ある質量を持つ[天体](g123.md)を、光さえ脱出できない球状領域に圧縮したときの半径。1916年にドイツの物理学者カール・シュワルツシルトがアインシュタインの[一般相対性理論](g108.md)の方程式を解いて導いた。この半径より内側に質量が収まると、その天体はブラックホールになる。

計算式は非常に単純で、質量 M に対して半径 r = 2GM/c²（G は重力定数、c は光速）と表せる。具体例を挙げると——太陽のシュワルツシルト半径は約3km、地球は約9mm、人間（体重70kg）はおよそ10⁻²⁵ m（[陽子](g114.md)の直径の10兆分の1以下）になる。

通常の天体はシュワルツシルト半径よりはるかに大きいため、この条件は自然には満たされない。恒星の場合、核融合が尽きた後に自己重力が勝って圧縮が進み、質量が太陽の約3倍以上あると[中性子星](g084.md)にもなれず最終的にシュワルツシルト半径以下に収縮してブラックホールとなると考えられている。この半径の球面が[事象の地平線](g006.md)にあたる。
