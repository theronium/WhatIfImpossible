# 宇宙速度（Cosmic Velocity）

**Cosmic Velocity**  天文学・宇宙論 / 天文学・軌道力学・宇宙工学 / g106
読み: うちゅうそくど　関連: —

**別名**: 脱出速度 / 第一宇宙速度 / 第二宇宙速度 / 第三宇宙速度

地球から宇宙へ脱出するために必要な速度の段階的な基準。主に第一・第二・第三宇宙速度の三段階がある。

**第一宇宙速度（約7.9 km/s）**：地表すれすれの円軌道を維持できる最低速度。これ以上の速さで水平に打ち出された物体は落下し続けても地球の丸みに沿って落ちず、そのまま周回軌道に乗る。[地球周回軌道](g077.md)（g077）のうち最も低い理論的軌道に対応する。

**第二宇宙速度（約11.2 km/s）**：地球の重力を完全に振り切るために必要な脱出速度。これを超えると物体は地球の引力圏を離れ、[太陽](g084.md)系内を自由に飛行できる。[スイングバイ](g053.md)（g053）はこの速度を追加燃料なしに稼ぐ技術だ。

**第三宇宙速度（約16.7 km/s）**：太陽の重力からも脱出して太陽系外へ飛び出すための速度。[ボイジャー](g076.md)探査機はスイングバイを駆使してこの速度を達成した。[ソーラーセイル](g054.md)（g054）は推進剤なしにこの速度域への到達を目指す技術の一つだ。

**第四宇宙速度（約550 km/s）**：天の川[銀河](g123.md)の重力から脱出するために必要な速度。第一〜三と異なりI[AU](g095.md)等の公式定義はなく、文献によって300〜600 km/sの幅がある。[カルダシェフスケール](g002.md)のタイプIII文明への移行と対応させて語られることが多い。銀河団スケールになると宇宙膨張（ハッブル定数）の影響が無視できなくなり、古典的な脱出速度の概念自体が成立しなくなるため、第五宇宙速度以上は理論的に曖昧な領域に入る。
