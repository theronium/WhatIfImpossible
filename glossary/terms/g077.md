# 地球周回軌道（Earth Orbit）

**Earth Orbit**  天文学・宇宙論 / 宇宙工学・軌道力学 / g077
読み: ちきゅうしゅうかいきどう　関連: [人工天体](g076.md), [ラグランジュポイント](g049.md)

**別名**: 衛星軌道 / LEO / GEO / 国際宇宙ステーション / ISS / 静止軌道 / 低軌道 / 太陽同期軌道

人工[衛星](g123.md)が地球を周回する軌道は高度・形状・傾斜角によって分類される。高度が高いほど周回速度は遅くなり（ケプラーの第三法則）、放射線環境・通信遅延・打ち上げコストが変化するため、用途に応じた軌道選択が重要となる。

| 軌道 | 高度 | 周期 | 主な用途 |
|------|------|------|----------|
| LEO（低軌道） | 200〜2,000km | 90〜120分 | ISS・地球観測・スターリンク |
| MEO（中軌道） | 2,000〜35,786km | 2〜24時間 | GPS・測位衛星・ガリレオ |
| GEO（静止軌道） | 約35,786km | 24時間（地球と同期） | 通信衛星・気象衛星 |
| HEO（高楕円軌道） | 近地点数百km〜遠地点数万km | 12〜24時間 | 高緯度通信・シリウスXM |
| グレーブヤード軌道 | GEOより約300km上 | — | 寿命を終えた衛星の墓場 |

**LEO**はロケットの燃料コストが低く多くの衛星が集中するが、大気抵抗で徐々に高度が落ちるため定期的な軌道修正が必要だ。スターリンクなど大規模コンステレーション（衛星群）の増加でデブリ問題が深刻化している。

**GEO（静止軌道）**は地球の自転と同期するため地上から見て静止して見え、通信・放送・気象観測に適する。ただし赤道上空に限定されるため高緯度地域では仰角が低くなる制限がある。

[ラグランジュポイント](g049.md)（g049）への配置は厳密には地球周回軌道ではないが、燃料消費を最小化する安定点として[宇宙望遠鏡](g057.md)や将来の施設に活用されている。
