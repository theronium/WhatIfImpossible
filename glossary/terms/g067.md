# 反物質（Antimatter）

**Antimatter**  仮説・未観測の粒子 / 素粒子物理学・宇宙論 / g067
読み: はんぶっしつ　関連: [wiim_069](../../docs/physics/wiim_069.md)

**別名**: アンチマター

通常の物質を構成する粒子と質量・スピンが同じで電荷などが逆符号の「反粒子」からなる物質。[電子](g064.md)の反粒子は陽電子（ポジトロン）、陽子の反粒子は反陽子と呼ばれる。反物質と通常物質が接触すると対消滅を起こし、両者の質量がすべてエネルギー（γ線）に変換される。E = mc² に従うとこの変換効率は核融合すら大きく上回る。

反物質は理論上の存在ではなく、実験室で生成・観測されている。1932年にカール・アンダーソンが宇宙線の観測中に陽電子を発見し、CERN（欧州[原子核](g114.md)研究機構）では反水素原子の生成・保存にも成功している。ただし生成には消費エネルギーをはるかに上回るコストがかかり、現在の技術では燃料として実用できる量の製造は不可能だ。

宇宙論上の最大の謎の一つが「なぜ宇宙は反物質ではなく物質で満ちているか」だ。[ビッグバン](g082.md)では物質と反物質が等量生成されたはずだが、現在の宇宙にはほぼ物質しか存在しない。このわずかな非対称性（[CP対称性の破れ](g156.md)）の完全な説明はいまだなされていない。
