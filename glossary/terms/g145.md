# ディラックの海（Dirac Sea）

**Dirac Sea**  量子物理学 / 量子力学・量子場理論・素粒子物理学 / g145
読み: でぃらっくのうみ　関連: [wiim_030](../../docs/physics/wiim_030.md), [wiim_085](../../docs/quantum/wiim_085.md), [wiim_086](../../docs/physics/wiim_086.md)

**別名**: ディラック海

ポール・ディラックが1930年に提唱した理論モデル。ディラック方程式には正のエネルギー解と負のエネルギー解の両方が存在するが、[電子](g064.md)が際限なく低エネルギー状態に落ち込む問題が生じる。これを解決するためディラックは「負エネルギー状態はすでに無数の電子で満たされている」と仮定した——これがディラックの海だ。[パウリの排他原理](g060.md)により、満員の海には新たな電子が入れない。

この海に「穴」が空くと、[負の質量](g072.md)・正の電荷を持つ空席が生じる。ディラックは当初これを[陽子](g114.md)と考えたが、1932年にカール・アンダーソンが陽電子（ポジトロン）を発見し、[反物質](g067.md)（g067）の存在が実証された。ディラックの海の穴が反粒子に対応するという予言が的中した瞬間だ。

現代の量子場理論（QFT）ではディラックの海という描像は使われず、生成・消滅演算子によって粒子と反粒子を対称に扱う。しかし量子真空の「満員の海」というイメージは、[カシミール効果](g009.md)（g009）や仮想粒子の直感的理解に今も使われる。
