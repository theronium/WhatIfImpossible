# 定理（Theorem）

**Theorem**  数学・論理・幾何学 / 数学・論理学 / g304
読み: ていり　関連: [公理](g305.md), [原理](g306.md), [CPT定理](g011.md)

数学や論理学において、[公理](g305.md)（g305）や既に証明された命題から、厳密な論理的推論によって導かれる命題。「証明が存在する」ことが定理の条件であり、証明なしに仮定される公理とは明確に区別される。

例として、ピタゴラスの定理（直角三角形の三辺の関係）や[フェルマーの最終定理](g309.md)（350年かけて証明）が挙げられる。物理学でも「[CPT定理](g011.md)（g011）」のように対称性から導かれる強い結論を定理と呼ぶ場合がある。[原理](g306.md)（g306）が自然観察から抽出された根本法則であるのに対し、定理は純粋な演繹の産物であり、自然界の実在とは独立に成立する点が特徴的である。
