# 超新星（Supernova）

**Supernova**  天文学・宇宙論 / 恒星進化・核合成・高エネルギー天文学 / g140
読み: ちょうしんせい　関連: [wiim_008](../../docs/biology/wiim_008.md)

**別名**: スーパーノヴァ

恒星が一生の最期に起こす爆発的な増光現象。数週間にわたって銀河全体に匹敵する光度に達することもある。大きく分けてII型（大質量星のコア崩壊型）とIa型（[白色矮星](g084.md)の暴走核融合型）の2種がある。

超新星が宇宙に果たす役割は単なる「死」ではない。恒星内部の核融合では鉄までしか合成できないが、超新星爆発の極限環境では鉄より重い元素（金・銀・ウランなど）が合成され、宇宙空間に撒き散らされる。酸素・炭素・窒素・鉄といった生命に必須の元素も超新星によって銀河に供給されており、「私たちは星屑でできている」という表現の根拠がここにある。

[コズミックマイス](g134.md)（g134）の観点では、超新星は脅威と恵みの両面を持つ。近傍の超新星爆発は[菌糸](g112.md)ネットワークを壊滅させる高エネルギー放射線を放つ一方、爆発の衝撃波が分子雲を圧縮して次世代の恒星・惑星系形成を促し、新たな[テラフォーミング](g042.md)先を生み出す。また爆発で撒かれた重元素が[アステロイドベルト](g136.md)（g136）や[小惑星の分類](g139.md)（g139）のC型・D型[天体](g123.md)に取り込まれ、[シェルマイセリウム](g135.md)（g135）の構造材料にもなる。
