# 天体（Celestial Body）

**Celestial Body**  天文学・宇宙論 / 天文学・宇宙論 / g123
読み: てんたい　関連: —

**別名**: 天文天体 / 惑星 / 衛星 / 小惑星 / 彗星 / 星雲 / 銀河 / 中性子星 / 白色矮星 / 褐色矮星

宇宙に存在する自然の物体の総称。塵・岩石から惑星・恒星・銀河まで、スケールは数センチから数億[光年](g096.md)に及ぶ。

主な天体の分類：

- **恒星**（g084）：核融合によって自ら輝く星。太陽はG型主系列星
- **惑星**：恒星の周囲を公転し、軌道近傍を支配する天体。地球型・巨大ガス型・氷巨星などに分類
- **衛星**：惑星の周囲を公転する天体（月など）
- **矮小天体**：小惑星・彗星・冥王星型天体など。太陽系小天体とも
- **白色矮星・[中性子星](g084.md)・ブラックホール**：恒星の末路として生まれる高密度天体
- **褐色矮星**：核融合に至らなかった「失敗した恒星」
- **星雲**：ガスと塵の雲。恒星の誕生地（散光星雲）や死骸（惑星状星雲・[超新星](g140.md)残骸）がある
- **銀河**：数億〜数兆個の恒星・ガス・[ダークマター](g056.md)（g056）が重力で束縛された巨大系
- **銀河団・超銀河団**：宇宙の大規模構造を形成する最大スケールの構造体

天体のスケールと密度の多様性はWhatIfImpossibleの思考実験の舞台そのものであり、[ダイソン球](g034.md)（g034）・[テラフォーミング](g042.md)（g042）・[ワープ航法](g035.md)（g035）などのSF概念も特定の天体環境を前提とする。
