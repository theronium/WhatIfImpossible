# 宇宙航法座標系（Cosmic Navigation Coordinate System）

**Cosmic Navigation Coordinate System**  WIIM派生技術 / 架空宇宙航法・測量工学 / g267
読み: うちゅうこうほうざひょうけい　関連: [wiim_022](../../docs/physics/wiim_022.md)

**別名**: アンキロン基準点網

[アンキロン](g128.md)（g128）を物理的な基準点として[銀河](g123.md)規模に展開した架空の座標網。通常の宇宙測量は[電波天文学](g124.md)やVLBI（超長基線干渉法）による観測値に依存するが、この系はアンキロンの計量テンソル固着特性により、銀河公転や[重力波](g004.md)の影響を受けない絶対基準を提供する。

[計量暦システム](g265.md)（g265）・[計量測量学](g263.md)（g263）・銀河慣性計測網と組み合わせることで、[恒星](g084.md)間航行・[ワープ](g035.md)ゲート網の精密誘導・FTL通信の到達点指定が可能になる。アンキロン基準点の密度と精度が「文明のリーチ（到達可能距離）」を決定するという意味で、宇宙文明のインフラ指標と見なせる。
