# 光年（Light-year / Parsec）

**Light-year / Parsec**  天文学・宇宙論 / 天文学・測量・観測天文学 / g096
読み: こうねん　関連: —

**別名**: 光年 / パーセク / ly / pc / 光年・パーセク

宇宙の距離を表す二つの主要な単位。用途と定義の由来が異なる。

**光年（light-year, ly）**：光が真空中を1年間に進む距離。約9兆4600億km（9.461×10¹² km）。「時間×速度」で定義されるため直感的にわかりやすく、一般向けの説明やSFで広く使われる。

**パーセク（parsec, pc）**：[年周視差](g355.md)が1秒角になる距離。1 pc ≈ 3.26光年 ≈ 206,265 AU（[天文単位](g095.md) g095）。地球の公転を利用して近くの星の見かけの位置がずれる角度（年周視差）を基準とした観測由来の単位で、「parallax second（視差の秒角）」が語源。観測データが視差として得られる[天文学](g124.md)者にとって計算に直結するため、星カタログや研究論文ではパーセクが標準。[銀河](g123.md)系・銀河団スケールではキロパーセク（kpc）・メガパーセク（Mpc）が使われる。

距離の目安として、最も近い[恒星](g084.md)プロキシマ・ケンタウリは約1.3 pc（約4.2光年）、天の川銀河の直径は約30 kpc（約10万光年）、アンドロメダ銀河までは約0.78 Mpc（約250万光年）。
