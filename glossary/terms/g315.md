# トポロフィピン（Topolophy Pin）

**Topolophy Pin**  WIIM派生技術 / 架空トポロジー操作工学 / g315
読み: とぽろふぃぴん　関連: [wiim_077](../../docs/physics/wiim_077.md), [wiim_074](../../docs/physics/wiim_074.md)

WhatIfImpossibleの思考実験（[wiim_077](../../docs/physics/wiim_077.md)）で提案された架空の閉じ込め機構。[トポロフィ](g293.md)場（g293）を局所的に励起して閉じた多様体構造を作ることで、[コーラ粒子](g127.md)（g127）が出現できる[トポロジー](g158.md)領域を特定の閉域に絞り込む装置として定義される。

コーラ粒子は[波動関数](g164.md)に従わず電場との相互作用が成立しないため、[光ピンセット](g313.md)のような電磁的手法では捕捉できない。トポロフィピンは粒子そのものではなく「粒子が出現できる空間のトポロジー」を制限するという、一段深い操作層に働く。[離散](g232.md)的な励起事象（[トポロン](g294.md)、g294）を利用した精密指定版を特にトポロンピンと呼ぶ。[アンキロンピン](g314.md)（g314）と組み合わせることで計量座標（どの点か）とトポロジー形式（どう接続されているか）を同時に固定し、コーラ粒子の出現先をミクロ単位で二重拘束する設計が可能になる。
