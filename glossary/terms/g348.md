# カノニカルアンサンブル（Canonical Ensemble）

**Canonical Ensemble**  物理学 / 統計力学・熱力学 / g348
読み: かのにかるあんさんぶる　関連: [虚数温度](g343.md), [有限温度場理論](g346.md)

統計力学において、温度・体積・粒子数が外部との熱交換によって一定に保たれる系を記述する統計的枠組み。各微視的状態の出現確率はボルツマン因子に比例し、分配関数がすべての[熱力学](g250.md)的量を導く要となる。

[有限温度場理論](g346.md)ではこの分配関数がZ = Tr[e^(−βH)]として表現される。[虚数温度](g343.md)の文脈ではβが虚数になることで、熱平衡状態が確率分布ではなく量子振幅として振る舞う。相転移の記述にも中心的な役割を担い、正負エネルギーの相転移（終焉の時代）を論じる際の理論的土台の一つとなる。
