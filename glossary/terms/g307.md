# 原則（Rule / Guiding principle）

**Rule / Guiding principle**  哲学・存在論 / 科学哲学・倫理学・工学 / g307
読み: げんそく　関連: [定理](g304.md), [原理](g306.md), [オッカムの剃刀](g020.md), [ロボット三原則](g120.md)

特定の目的・文脈において行動や判断の指針となる規範的な命題。数学的に証明される[定理](g304.md)（g304）や、自然法則としての[原理](g306.md)（g306）とは異なり、例外が存在しうる点が特徴である。

科学では「[オッカムの剃刀](g020.md)（g020）」（説明はより単純なものを優先する）が典型例で、絶対的な真理ではなく方法論的な指針として機能する。工学や倫理では「[ロボット三原則](g120.md)（g120）」のように、設計思想を支える規範として機能する。原理が「自然界はそうなっている」という記述的命題であるのに対し、原則は「そうすべき・そうするのが望ましい」という規範的命題である。この区別は科学的言明と倫理的言明の境界を考えるうえで重要になる。
