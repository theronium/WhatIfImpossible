# 侵害受容（Nociception）

**Nociception**  生物学・進化 / 神経科学・生理学 / g203
読み: しんがいじゅよう　関連: [wiim_042](../../docs/quantum/wiim_042.md), [wiim_047](../../docs/philosophy/wiim_047.md)

**別名**: 侵害受容器 / ノシセプション

有害な刺激（熱・機械的損傷・化学物質など）を検出し、回避行動や防御反応を引き起こす神経的メカニズム。侵害受容器（nociceptor）と呼ばれる感覚神経末端が刺激を感知し、脊髄・脳へ信号を伝達する。

侵害受容は**三人称的・客観的**な機能であり、外部から観測・測定できる。これに対して「痛い」という体験の質感——痛みの[クオリア](g032.md)——は一人称的・主観的であり、侵害受容が存在しても痛みのクオリアが伴うかどうかは別問題だ。侵害受容のメカニズムを完全に解明しても、「なぜそれが苦しいのか」という[ハードプロブレム](g169.md)（g169）には答えられない。

植物が損傷を受けてジャスモン酸を放出したり、単細胞生物が有害物質から遠ざかったりする反応は、神経系をもたない単純な化学・物理的応答であり、侵害受容とも区別される。WIIMのΔφ的な観点では、統合処理ノードを持たない植物のΔφはほぼゼロであり、侵害受容以前の単純反応にとどまると考えられる（[wiim_042](../../docs/quantum/wiim_042.md)）。
