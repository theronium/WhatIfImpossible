# 量子情報理論（Quantum Information Theory）

**Quantum Information Theory**  量子物理学 / 量子情報・量子計算 / g252
読み: りょうしじょうほうりろん　関連: —

[量子力学](g253.md)の原理（重ね合わせ・[量子もつれ](g062.md)・測定による[波動関数](g164.md)の収縮）を情報処理に応用する理論体系。古典[情報理論](g251.md)（g251）を量子系に拡張したもので、量子コンピュータ・量子暗号・[量子テレポーテーション](g141.md)の理論基盤となる。

WIIMでは[創発](g175.md)検知（Δφ測定）の理論的背景として登場し、[クオリア](g032.md)検知機（[wiim_042](../../docs/quantum/wiim_042.md)）のΔφ測定が「波動関数を収縮させずに[位相](g242.md)情報を得る」という量子情報上の矛盾に直面する場面で参照される。[エントロピー](g008.md)倍化（補遺[wiim_037](../../docs/physics/wiim_037.md)）の観測操作理論も量子情報理論の枠組みから論じられる。
