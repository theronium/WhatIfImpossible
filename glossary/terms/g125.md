# クロノスフィア（Chronosphere）

**Chronosphere**  WIIM概念 / 時空間操作・相対性理論・SF / g125
読み: くろのすふぃあ　関連: [wiim_002](../../docs/cosmology/wiim_002.md), [wiim_057](../../docs/physics/wiim_057.md), [wiim_058](../../docs/biology/wiim_058.md)

**別名**: 時間加速空間 / 時間操作球

WhatIfImpossibleの思考実験記事（[wiim_002](../../docs/cosmology/wiim_002.md)）で命名された架空の装置・空間概念。「Chrono（時間）」と「Sphere（球・領域）」を組み合わせた造語で、外部から内部の時間進行速度を任意の倍率で加速できる閉じた空間領域を指す。

物理的実現の候補[原理](g306.md)として、**球状シェル上を光速に近い速度で回転する粒子（または[光子](g064.md)）の集団**による時間差生成がある。[特殊相対性理論](g108.md)の[時間膨張](g010.md)（g010）により高速回転するシェルの[固有時](g174.md)が外部より遅れ、内部は外部より速い時間の中に置かれる。光子シェルの場合は γ → ∞ が自動達成されエネルギー発散の問題が消えるため、ブラックホールのフォトン球からの着想・観測による偶然の発見が有力な起源とされる。

シェルの曲率を「完全捕捉」より僅かに弱く調整することで、内部からの放射を安全レベルに制限する**自己封鎖型設計**が可能だ。入射方向（外→内）はシェルの回転に沿って取り込まれやすく、出射方向（内→外）は大半が反射される非対称構造——「入れるが（ほぼ）出られない」ブラックホールの性質を工学的に再現する。実現の壁は境界面での[因果律](g017.md)（g017）の整合・光子シェルの閉じ込め機構・内外補給経路の設計だ（補遺 wiim_002_rotation_principle 参照）。
