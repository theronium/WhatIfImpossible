# FTL航法（Faster-Than-Light Travel）

**Faster-Than-Light Travel**  SF概念 / 相対性理論・SF / g045
読み: えふてぃーえるこうほう　関連: [wiim_003](../../docs/physics/wiim_003.md), [wiim_004](../../docs/cosmology/wiim_004.md)

**別名**: 超光速航法

光速を超えて移動・情報伝達を行う技術の総称。[特殊相対性理論](g108.md)は「質量を持つ物体は光速に到達できない」と定めており、FTLは現在の物理学の枠組みでは不可能とされる。

SFでは様々な回避策が提案・描写されている。

| 方式 | 概要 |
|------|------|
| [ワープ](g035.md)ドライブ（アルクビエレ方式） | 空間自体を歪めて「泡」ごと移動。物体は光速を超えない |
| [ワームホール](g036.md) | 時空を折り畳んで遠距離を近道する |
| ハイパースペース | 通常空間と別の次元を経由する |
| ジャンプドライブ | 空間を瞬時に飛び越える（テレポート的） |
| アンサイブル | FTL通信のみ（移動ではなく情報伝達） |

物理的に深刻な問題が[因果律](g017.md)だ。特殊相対性理論では、ある基準系でFTL移動が可能なら、別の基準系では「過去に向かう移動」として観測される——すなわちFTLは時間旅行と等価になり、[タイムパラドックス](g039.md)を引き起こしかねない。「FTLは不可能」という制約は、宇宙の因果構造を守るための物理的な「安全装置」とも言える。
