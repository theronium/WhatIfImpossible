# セルオートマトン（Cellular Automaton）

**Cellular Automaton**  哲学・存在論 / 複雑系・情報科学・計算理論 / g298
読み: せるおーとまとん　関連: [wiim_008](../../docs/biology/wiim_008.md), [wiim_075](../../docs/biology/wiim_075.md)

単純なルールを持つ「セル（格子点）」が隣接するセルの状態を参照して次の状態を決定する、[離散](g232.md)的な計算モデル。「細胞（cell）」と「自動機械（automaton）」を組み合わせた名称だ。

最も有名な例はジョン・ホートン・コンウェイが1970年に考案した「ライフゲーム」で、「生」か「死」かの二状態・3〜4個の隣接セルという極めて単純なルールから、グライダー（移動する構造体）や発振器など驚くほど多様なパターンが[創発](g175.md)する。スティーブン・ウルフラムによるルール110は、単純なセルオートマトンがチューリング完全であることを証明した——つまり、一次元の格子でも理論上あらゆる計算が実行できる。

[コズミックマイス](g134.md)（[wiim_008](../../docs/biology/wiim_008.md)）の[菌糸](g112.md)ネットワークが周期的なパターンや創発的構造を示す仕組みを理解するうえでの参照概念として有用だ。創発（g175）や[ハイヴマインド](g044.md)（g044）が「なぜ複雑さが生まれるか」を哲学的に問うのに対し、セルオートマトンは「いかにして複雑さが生まれるか」を計算論的に示す。
