# マテリオマイス（Materio Myce）

**Materio Myce**  WIIM概念 / 架空生命体・合成生物学・素材工学 / g444
読み: まてりおまいす　関連: [コズミックマイス](g134.md), [シェルマイセリウム](g135.md), [素材遺伝子](g441.md), [音響複層体](g438.md)

[素材遺伝子](g441.md)を取り込んだ[コズミックマイス](g134.md)系列の架空生物の総称。[音響複層体](g438.md)に限らず、遺伝子設計によって任意の複合素材構造を生体プロセスで自律生産できる[菌糸](g112.md)生命体を指す。

コズミックマイスが「宇宙的規模の分散知性」、[シェルマイセリウム](g135.md)が「生きた宇宙カプセル膜」であるのに対し、マテリオマイスは「生きた素材製造装置」として機能する。菌糸ネットワークが位置情報を読み取りながら必要な素材を必要な場所に分泌するため、工[業](g206.md)製造では困難な複雑なグラデーション構造や自己修復素材を自律生産できる。増殖により製造がスケールし、環境から調達した鉱物・電力のみで稼働する。
