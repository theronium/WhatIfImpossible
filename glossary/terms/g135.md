# シェルマイセリウム（Shell Mycelium）

**Shell Mycelium**  WIIM概念 / 架空生命体・宇宙生物学・共生系 / g135
読み: しぇるまいせりうむ　関連: [wiim_008](../../docs/biology/wiim_008.md), [wiim_011](../../docs/physics/wiim_011.md), [wiim_025](../../docs/biology/wiim_025.md), [wiim_075](../../docs/biology/wiim_075.md), [wiim_084](../../docs/biology/wiim_084.md)

**別名**: 宇宙生命カプセル / 菌糸宇宙カプセル

WhatIfImpossibleの思考実験（[wiim_025](../../docs/biology/wiim_025.md)）で命名された架空の自律型宇宙生命体。「Shell（殻）」と「Mycelium（菌糸体）」を組み合わせた造語で、[コスモシェル](g132.md)（g132）の構造的課題と[コズミックマイス](g134.md)（g134）の生物的能力を統合した共生体だ。

真空中に自己維持する閉鎖膜（コスモシェル）は「膜が生きた振る舞いをしなければならない」という論理的壁を持つ。シェル[マイセリウム](g112.md)はこの問いへの答えとして、宇宙適応した菌糸（コズミックマイス）で膜そのものを織り上げる——最初から生きている素材で作ることで、損傷検知・分子再配置・気密維持を生物的プロセスとして実現する。

外層の菌糸多層網は宇宙線を栄養として吸収しながら自己修復し、内部には閉じた生態系と大気循環を保持する。最も衝撃的な特性は自律的な増殖能力で、内部で生成された[胞子](g111.md)が外部に放出され、[小惑星](g123.md)や彗星を核として新たなシェルマイセリウムを形成する。設計者の介入なしに[太陽](g084.md)系全体へ拡散する「宇宙の細胞」とも呼ばれる概念だ。
