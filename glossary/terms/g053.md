# スイングバイ（Gravity Assist）

**Gravity Assist**  天文学・宇宙論 / 天体力学・宇宙探査 / g053
読み: すいんぐばい　関連: —

**別名**: 重力アシスト / フライバイ

[惑星](g123.md)や衛星の重力を利用して探査機の速度・方向を変える飛行技術。グラビティアシスト（重力アシスト）とも呼ぶ。追加の燃料を消費せずに大幅な加速・減速・進路変更が可能なため、外惑星探査には不可欠な手法となっている。

仕組みを直感的に理解するには「惑星との追いかけっこ」が便利だ。探査機が惑星の後方から接近すると、惑星の重力に引かれて加速しながら惑星の公転運動のエネルギーをわずかに受け取る。探査機は惑星に対しては等速だが、[太陽](g084.md)を基準にした速度は惑星の公転速度分だけ増加する。逆に前方から接近すれば減速にも使える。

実績として、[ボイジャー](g076.md)1・2号は木星・土星・天王星・海王星を次々とスイングバイして外太陽系を踏破し、現在は太陽系外に到達している。カッシーニ探査機も金星2回・地球1回・木星1回のスイングバイで土星軌道に投入された。

[ラグランジュポイント](g049.md)への投入軌道設計にもスイングバイが組み合わされることがあり、燃料効率の高い深宇宙探査を支える核心技術となっている。
