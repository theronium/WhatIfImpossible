# スピントロニクス（Spintronics）

**Spintronics**  物理学 / 物性物理学・量子情報工学 / g456
読み: すぴんとろにくす　関連: [wiim_111](../../docs/quantum/wiim_111.md)

[電子](g064.md)の「電荷」ではなく「スピン（[量子力学](g253.md)的な角運動量）」を情報の担体として利用する技術・研究分野。Spin と Electronics の合成語。

スピンは上向き・下向きの2状態を持ち、古典的なビット（0/1）に対応する。電荷を動かさずにスピン状態だけを制御できれば、発熱が少なく高速な情報処理が[原理](g306.md)的に可能になる。ハードディスクの巨大磁気抵抗（GMR）効果による読み取りヘッドは実用化されたスピントロニクス技術の代表例で、2007年のノーベル物理学賞対象となった。

量子生物学との接点では、鳥類の磁気感覚を担う[ラジカル対](g457.md)機構がスピントロニクスと本質的に同じ物理——スピン状態の量子的重ね合わせと磁場応答——を利用している。地球磁場がラジカル対のスピンコヒーレンスに影響を与えるこの現象は生体スピントロニクスと呼ばれる。[菌糸](g112.md)ネットワークにラジカル対を介した量子シグナルが存在するなら、[菌類量子コヒーレンス](g455.md)とスピントロニクスは直接接続する可能性がある。
