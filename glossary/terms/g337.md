# ルーローの三角形（Reuleaux Triangle）

**Reuleaux Triangle**  数学・論理・幾何学 / 幾何学 / g337
読み: るーろーのさんかっけい　関連: —

正三角形の各頂点を中心とし、対辺の長さを半径とする3つの円弧によって囲まれた曲線図形。19世紀ドイツの機械工学者フランツ・ルーロー（Franz Reuleaux）にちなんで命名された。

最大の特徴は「定幅性（定幅図形）」であり、任意の方向から測った幅が常に一定の値になる。この性質から、正方形の穴を掘るドリルビットの断面やヴァンケル型回転エンジンのローター形状、英国の50ペンスコインなど実用的な設計にも採用されている。

「丸くないのに転がせる」という直感に反した形状は、思考実験の文脈では「幾何学的不可能の偽命題」の典型例として引用されることがある。円以外の定幅図形が存在するという事実は、形状と機能の自明な対応関係を問い直す契機となる。
