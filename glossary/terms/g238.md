# 特異点光学（Singular Optics）

**Singular Optics**  物理学 / 光学・波動物理学 / g238
読み: とくいてんこうがく　関連: —

光の[位相](g242.md)が不定となる点（位相[特異点](g043.md)）を中心に扱う光学の分野。位相特異点では光の振幅がゼロになり、周囲の位相が螺旋状に巻きつく構造が生まれる。この螺旋構造を持つ光を[光渦](g239.md)（g239）と呼ぶ。

位相特異点は自然光・レーザー光・回折光のいずれにも発生し得る。工学的には回折格子・空間光変調器・液晶素子を用いて意図的に生成される。

応用は[光ピンセット](g313.md)（光渦による粒子の回転捕捉）、光通信の軌道角運動量多重（OAM多重）、超解像顕微鏡（STED法）、量子情報処理など広い。WIIM世界観では[菌糸](g112.md)[リングレーザー](g244.md)や[アンキロン](g128.md)光路制御（g237）との文脈で関連する。
