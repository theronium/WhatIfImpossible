# 胞子（Spore）

**Spore**  生物学・進化 / 生物学・菌類学・宇宙生物学 / g111
読み: ほうし　関連: [wiim_017](../../docs/biology/wiim_017.md), [wiim_018](../../docs/biology/wiim_018.md), [wiim_083](../../docs/biology/wiim_083.md)

**別名**: スポア

菌類・植物・細菌などが無性生殖または有性生殖の一環として形成する、単独で新個体を生じうる細胞または細胞群。種子と異なり受精を必要とせず、極めて少ないエネルギーで大量に生産・散布できる。

胞子の最大の特徴は環境耐性の高さだ。菌類の胞子は乾燥・紫外線・高温・低温に強く、宇宙空間でも一定期間生存できるものがある。[極限環境微生物](g030.md)の胞子（特に芽胞形成細菌）は真空・放射線環境下でも数年単位で生存が確認されており、[パンスペルミア](g007.md)仮説において生命の[惑星](g123.md)間輸送手段として候補に挙がる。

散布手段も多様で、風・水・昆虫・爆発的な自己射出など生物ごとに異なる。[菌根ネットワーク](g031.md)を形成する菌類はネットワーク拡大のために胞子を広域に散布し、生態系の回復・拡張に寄与する。胞子雨による惑星[テラフォーミング](g042.md)（[wiim_017](../../docs/biology/wiim_017.md)）や、金星・タイタンへの生物気候工学（[wiim_018](../../docs/biology/wiim_018.md)）では、胞子の耐性と散布効率を宇宙規模で応用するアイデアが展開されている。
