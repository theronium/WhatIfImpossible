# メビウス環（Möbius strip）

**Möbius strip**  数学・論理・幾何学 / 位相幾何学・トポロジー / g302
読み: めびうすかん　関連: [クラインの壺](g303.md)

帯状の紙を一回ねじって両端を貼り合わせた曲面。表と裏が一続きになっており、面を一周すると出発点の裏側に戻る「一面体」である。1858年にアウグスト・メビウスとヨハン・リスティングが独立に発見した。

[トポロジー](g158.md)（g158）的な特徴として、メビウス環は「向きづけ不可能な曲面」の最も単純な例である。向きづけ可能な曲面（球・トーラスなど）は内側と外側を一貫して区別できるが、メビウス環にはその区別がない。境界は1本の閉曲線のみで、中央を切ると一本の大きな輪になる。高次元版に当たる[クラインの壺](g303.md)（g303）は境界を持たない向きづけ不可能な閉曲面であり、メビウス環2枚を貼り合わせた構造として理解できる。
