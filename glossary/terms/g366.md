# 角速度（Angular Velocity）

**Angular Velocity**  物理学 / 力学 / g366
読み: かくそくど　関連: —

回転運動において、単位時間あたりに回転する角度（ラジアン）を表す量。記号はω（オメガ）。1周（2πラジアン）を周期Tで割ることで求められ、ω = 2π/T と表される。

角速度と周期の関係は、ケプラーの第3法則とニュートンの万有引力を組み合わせて[静止軌道](g077.md)半径を導出する際の核心にある。重力と遠心力のつりあい式にω = 2π/Tを代入することで、「周期の2乗は軌道半径の3乗に比例し、比例定数は4π²/GM」という形が導かれる。

線速度v（m/s）との関係はv = rω（rは回転半径）。角速度が一定の等速円運動では、線速度の方向は変化し続けるが大きさは一定になる。
