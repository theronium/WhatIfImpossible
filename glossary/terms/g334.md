# ルーネベルクレンズ（Luneburg Lens）

**Luneburg Lens**  物理学 / 電波工学・光学・生物視覚 / g334
読み: るーねべるくれんず　関連: —

球内の屈折率が中心から表面に向かって連続的に変化し、球面上の一点から入射した波を反対側の球面上の一点に完全収束させる球形グラジェントインデックス（GRIN）レンズ。1944年にRudolf Luneburgが理論的に導出した。

工学ではレーダー反射率校正体・広角[衛星](g123.md)通信アンテナ・全方位カメラとして利用される。自然界では一部の甲殻類の眼の反射層や、クモ目の眼の内部構造が類似の屈折率分布を持つことが示唆されており、方向を問わず光を受容する全方向視覚に適応した独立進化と考えられる。
