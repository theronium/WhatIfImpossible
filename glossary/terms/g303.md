# クラインの壺（Klein bottle）

**Klein bottle**  数学・論理・幾何学 / 位相幾何学・トポロジー / g303
読み: くらいんのつぼ　関連: [メビウス環](g302.md)

1882年にフェリックス・クラインが考案した、内側と外側の区別がない閉曲面。[メビウス環](g302.md)（g302）を2枚、境界同士を合わせて貼り合わせた構造と等価であり、向きづけ不可能な閉曲面の代表例である。

3次元空間内では自己交差なしに実現できず、完全な埋め込みには4次元以上の空間が必要になる。日常的に見る「壺の首を底に差し込んだ形」は3次元への射影（自己交差あり）にすぎない。[トポロジー](g158.md)（g158）的性質として、[オイラー](g229.md)標数（g230）は0、境界は存在しない。表面を一周すると鏡像に戻るという向きの反転は、メビウス環と共通する特性である。SFや宇宙論では宇宙の大域的トポロジーの候補として言及されることがある。
