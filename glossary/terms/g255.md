# 余剰次元（Extra Dimensions）

**Extra Dimensions**  物理学 / 高エネルギー物理学・弦理論 / g255
読み: よじょうじげん　関連: —

我々が知覚する3次元空間（＋時間の4次元時空）の外側に存在すると予言される追加の空間次元。[超弦理論](g093.md)（g093）では理論的整合性のために6〜7個の余剰次元が必要とされ、[プランクスケール](g014.md)（約10⁻³⁵m）に「丸まった（コンパクト化された）」状態にあると考えられている。

WIIMでは余剰次元が複数の技術の基盤概念として機能する。[コーラ粒子](g127.md)（g127）の空間バイパスは余剰次元を経由するとされ、量子永久機関（[wiim_039](../../docs/quantum/wiim_039.md)）は[エントロピー](g008.md)を余剰次元に転嫁することで「[余剰次元バンク](g262.md)」を利用する。[グラビトーペイク](g129.md)（g129）がエネルギーを転嫁する先も余剰次元とされる。
