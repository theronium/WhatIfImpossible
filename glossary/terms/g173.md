# デコヒーレンス（Decoherence）

**Decoherence**  物理学 / 量子力学 / g173
読み: でこひーれんす　関連: [wiim_037](../../docs/physics/wiim_037.md), [wiim_039](../../docs/quantum/wiim_039.md)

**別名**: 量子デコヒーレンス

量子系が外部環境と相互作用することで、量子的重ね合わせが失われ古典的な確率分布に移行する過程。量子力学と古典力学の橋渡しとなる現象だ。

孤立した量子系は重ね合わせ状態を保てるが、現実の系は周囲の熱雑音・電磁場・空気分子と絶え間なく相互作用する。この環境との絡み合いにより、量子的な[位相](g242.md)情報が瞬時に「散逸」して古典的な振る舞いに変わる。マクロな物体が量子的重ね合わせを示さない理由はデコヒーレンスが極めて速く起きるためだ。

量子コンピュータの実現を難しくしている最大の障壁であり、量子永久機関（[wiim_039](../../docs/quantum/wiim_039.md)）の実現を阻む技術的限界でもある——[コーラ粒子](g127.md)板の量子状態を維持しようとしても、デコヒーレンスが瞬時に板を「通常の導体」として振る舞わせる。[レトロン](g163.md)（[wiim_037](../../docs/physics/wiim_037.md)）が透明な観察者を実現しようとする際も、デコヒーレンスは観測後の絡み合いを残し続ける。
