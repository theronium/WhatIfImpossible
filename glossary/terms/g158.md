# トポロジー（Topology）

**Topology**  物理学 / 数学・位相幾何学・凝縮系物理学 / g158
読み: とぽろじー　関連: [wiim_021](../../docs/physics/wiim_021.md)

**別名**: 位相幾何学 / トポロジカル

図形や空間の「連続的な変形で変わらない性質」を研究する数学の一分野。距離や角度ではなく、穴の数・繋がり方・内外の区別といった大域的な構造を扱う。有名な例として、コーヒーカップとドーナツは穴が一つという点でトポロジー的に同じ（同相）だが、球とドーナツは異なる。

物理学では「トポロジカル保護」という概念が重要だ。ある物理量がトポロジー的な性質に由来する場合、局所的な摂動（不純物・温度揺らぎなど）によって変化しない頑健性を持つ。整数量子ホール効果の伝導度が精密に整数になるのはこの保護による。また[エニオン](g157.md)（g157）やトポロジカル量子コンピュータはこの性質を誤り耐性の根拠として利用する。

宇宙論ではトポロジーが宇宙全体の「形」を決める。宇宙が有限か無限か、トーラス状か球状かはトポロジーの問題だ。[wiim_021](../../docs/physics/wiim_021.md)（[切れないエネルギー紐](g131.md)）ではトポロジカル保護によって不変距離を定義する概念が論じられている。
