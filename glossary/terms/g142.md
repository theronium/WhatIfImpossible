# パラレルワールド（Parallel World）

**Parallel World**  仮説・未観測の粒子 / 量子力学・宇宙論・SF / g142
読み: ぱられるわーるど　関連: [wiim_037](../../docs/physics/wiim_037.md)

**別名**: 平行世界 / 並行宇宙 / 多世界 / 多元宇宙

私たちの宇宙と並行して存在するとされる別の宇宙・世界の総称。理論的背景により複数の意味を持つ。

**エヴェレットの[多世界解釈](g162.md)**：[量子力学](g253.md)の観測問題に対する解釈の一つ。量子系が観測されるたびに宇宙が分岐し、すべての可能性が異なる世界で実現するとする。[シュレーディンガーの猫](g083.md)が「生きている世界」と「死んでいる世界」に分かれるのがその例だ。コペンハーゲン解釈（波束の収縮）と並ぶ主要解釈だが、実験的に区別する方法はない。

**インフレーション宇宙論の[多元宇宙](g143.md)**：永久インフレーション理論では無数の[ビッグバン](g082.md)が起き、それぞれ異なる物理定数を持つ宇宙が泡のように生じるとされる（バブル宇宙）。私たちの宇宙はその一つに過ぎない。

**[ひも理論](g093.md)のランドスケープ**：ひも理論が許容する解の数は10⁵⁰⁰以上とされ、それぞれ異なる物理法則を持つ宇宙に対応するとの見方がある。

いずれも現時点では観測的に検証する手段がなく、思考実験と科学の境界に位置する概念だ。
