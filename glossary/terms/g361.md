# ゆえに記号（therefore sign）

**therefore sign**  数学・論理・幾何学 / 数学・記号論 / g361
読み: ゆえにきごう　関連: —

**別名**: ∴

論理的な結論を導く際に使われる記号「∴」。3つの点を逆三角形に配置した形で、「よって」「ゆえに」を意味する。17世紀ごろから数学・論理学の文書で使われはじめ、現代では証明の結論部に ∴ を置いて前提から帰結を明示する慣習が定着している。

証明の終端を示す[ハルモス記号](g360.md)（g360）と対をなす存在で、∴ が「ここから結論が導かれる」を示すのに対し、∎ は「証明がここで完結した」を宣言する。WhatIfImpossibleの補遺ノートでは「∴ 〜は矛盾しない。∎」という形でセットで用いる。対になる記号として ∵（なぜならば）があり、∵ が根拠を示し、∴ が帰結を示す。
