# パンサイキズム（Panpsychism）

**Panpsychism**  哲学・存在論 / 心の哲学・形而上学 / g168
読み: ぱんさいきずむ　関連: —

意識や主観的体験は宇宙の根本的な性質であり、物質の基本的な構成要素にも何らかの形で宿っているという立場。[電子](g064.md)一個には人間の意識はないが、何らかの「原意識的」な性質があり、それが複雑に組み合わさると人間の意識が生まれるという考え方だ。

[意識のハードプロブレム](g169.md)——なぜ物理過程が主観体験を生むのか——に対する一つの回答として近年注目が高まっている。物理から意識が「[創発](g175.md)」するのではなく、意識は最初から物理の一部として存在するという逆転の発想だ。

[コズミックマイス](g134.md)（[wiim_008](../../docs/biology/wiim_008.md)）が本当に知性を持つかという問いに、パンサイキズムは独自の答えを示す。[菌糸](g112.md)ネットワークが宇宙規模に広がるとき、そこに宿る「原意識」の総和は、何か新しい意識形態を生む可能性がある。
