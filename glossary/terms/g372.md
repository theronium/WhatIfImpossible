# スカイフック（Skyhook）

**Skyhook**  物理学 / 宇宙工学・軌道力学 / g372
読み: すかいふっく　関連: [wiim_091](../../docs/physics/wiim_091.md)

軌道上を周回する長大なテザー（紐状構造）を利用して、地上から宇宙へ物体を輸送する構想。[静止軌道](g077.md)エレベーターとは異なり、軌道速度で回転するテザーの下端が定期的に大気上層や地表付近に近づく動作を利用する。下端が低速で接触し、上端が軌道速度以上となることで、ロケットなしに物体を軌道速度まで加速できる可能性がある。

回転型と非回転型の2種類があり、[テザー衛星](g373.md)の技術的延長として構想されている。現在の材料強度では実現困難だが、カーボンナノチューブなどの超高強度素材が実用化されれば[原理](g306.md)的には可能とされ、打ち上げコストを大幅に削減できる構想として研究されている。
