# エラトステネス（Eratosthenes）

**Eratosthenes**  天文学・宇宙論 / 天文学・古代ギリシャ / g327
読み: えらとすてねす　関連: —

古代ギリシャの学者（紀元前276年頃〜195年頃）。地球の円周を初めて定量的に算出した人物として知られる。アレクサンドリアとシエネ（現アスワン）の距離と、夏至の正午に両地点で生じる影の角度差（約7.2°）を用いて、地球の円周を約4万kmと推定した。これは現代の測定値（約4万75km）ときわめて近い値である。

数学では素数の列挙法「エラトステネスの篩」の考案者でもあり、地理・詩・歴史にも精通した博学者だった。アレクサンドリア図書館の館長を務め、当時の知の拠点を主導した。地球の大きさを観察と幾何学だけで導いたその手法は、現代の天文測量と同じく「観測量から物理量を逆算する」思考の原型とも言える。
