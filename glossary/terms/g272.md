# 核変換（Nuclear Transmutation）

**Nuclear Transmutation**  物理学 / 核物理学・放射化学 / g272
読み: かくへんかん　関連: [wiim_024](../../docs/biology/wiim_024.md), [wiim_069](../../docs/physics/wiim_069.md)

**別名**: 元素変換 / 核種変換

[原子核](g114.md)の陽子数・中性子数を変化させることで、ある元素を別の元素へ転換する現象または技術の総称。核分裂・核融合・中性子捕獲・放射性崩壊などがすべて核変換の一形態だ。

歴史的には「[錬金術](g117.md)（g117）の科学的実現」とも呼ばれ、1919年にラザフォードが窒素に高速アルファ粒子を照射して酸素と水素を生成したのが人工核変換の最初の成功例とされる。現代では加速器で重イオンを衝突させて超重元素を合成したり、原子炉内の中性子照射で放射性廃棄物の半減期を短縮したりする研究が進んでいる。

WIIM世界では、[マイコプラズマギカ](g130.md)（g130）が生物的に核変換を行う設定として登場するほか、架空粒子を利用した安価な核変換の実現可能性も思考実験の対象となっている（[wiim_069](../../docs/physics/wiim_069.md)）。
