# 重力波検出器（Gravitational Wave Detector）

**Gravitational Wave Detector**  天文学・宇宙論 / 重力波天文学・観測技術 / g073
読み: じゅうりょくはけんしゅつき　関連: [グレーザー](g066.md), [wiim_020](../../docs/physics/wiim_020.md)

**別名**: LIGO / LISA / KAGRA / Virgo / 干渉計

[重力波](g004.md)（時空の歪みの波）を検出するための観測施設。現在主流なのはレーザー干渉計方式で、L字型のトンネル内を往復するレーザー光の経路長の微小な変化（[陽子](g114.md)直径の1000分の1以下）を精密に測定する。2015年9月、LIGOが13億[光年](g096.md)彼方のブラックホール合体による重力波を人類史上初めて検出し、2017年のノーベル物理学賞につながった。

複数の検出器が世界各地に設置されている。複数拠点が同時に信号を捉えることで重力波の到来方向を三角測量的に特定でき、電磁波望遠鏡との同時観測（マルチメッセンジャー[天文学](g124.md)）が可能になる。

| 施設 | 場所 | 特徴 |
|------|------|------|
| LIGO | アメリカ（2拠点） | アーム長4km、2015年初検出 |
| Virgo | イタリア | アーム長3km、欧州代表 |
| KAGRA | 日本・神岡 | 地下設置・低温鏡で熱雑音を抑制 |
| [GEO](g077.md)600 | ドイツ | アーム長600m、技術実証機 |
| LISA | 宇宙空間（2030年代計画） | ESA、3機の宇宙機をアーム長250万kmで結ぶ |
| Einstein Telescope | 欧州（計画中） | 地下・三角形配置の次世代型 |

LISAが実現すれば地上では遮断される低周波重力波（超大質量ブラックホール合体など）を観測でき、宇宙論的スケールの現象への窓口となる。[グレーザー](g066.md)（g066）が「重力波を送る」装置とすれば、重力波検出器は「受ける」装置という対の関係にある。
