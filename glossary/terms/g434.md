# バンクス軌道（Banks Orbital）

**Banks Orbital**  SF概念 / SF・メガストラクチャー・宇宙居住 / g434
読み: ばんくすきどう　関連: —

イアン・M・バンクスのSF小説シリーズ「カルチャー」に登場する巨大環状居住構造物。幅約数百万km、直径数百万kmのリング状で、内側の表面が居住域となる。ラリー・ニーヴンの[リングワールド](g041.md)と外見は似るが、バンクス軌道はリングワールドより小規模で自転速度も緩く、端に高い壁を設けて大気を保持する点が異なる。

[恒星](g084.md)を軌道の中心に置かず、やや外れた位置に配置することで昼夜サイクルを生み出す。居住可能面積は地球の数千倍に達し、生態系ごとに異なる環境ゾーンを設けることもできるとされる。

リングワールドと同様に回転による遠心力を擬似重力として使う。端壁の設計や素材強度の問題はリングワールドと共通しており、建設には[カルダシェフスケール](g002.md)のタイプII文明相当の資源と技術が必要と考えられる。
