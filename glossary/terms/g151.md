# ホログラフィック原理（Holographic Principle）

**Holographic Principle**  量子物理学 / 理論物理学・量子重力・情報理論 / g151
読み: ほろぐらふぃっくげんり　関連: [wiim_020](../../docs/physics/wiim_020.md)

ある空間領域に含まれる情報量の上限は、その領域の体積ではなく境界面の面積に比例するという[原理](g306.md)。ゲラルト・トホーフトとレオナルド・サスキンドが1990年代に定式化した。名称はホログラム（2次元フィルムに3次元情報を記録する技術）から取られており、「3次元の世界は2次元の境界面に書き込まれた情報の投影である可能性がある」という直観を表す。

起源はブラックホールの[エントロピー](g008.md)にある。ヤコブ・ベッケンシュタインとスティーブン・ホーキングは、ブラックホールのエントロピーが[事象の地平線](g006.md)の面積に比例することを示した。これは「情報が体積でなく面積に収まる」ことを示唆しており、ホログラフィック原理の出発点となった。後にフアン・マルダセナがAdS/CFT対応（反ド・ジッター空間と共形場理論の双対性）として数学的に定式化し、現代の[弦理論](g093.md)・量子重力研究の中心的な枠組みとなっている。

[エントロピー重力](g150.md)（g150）はこの原理を重力の起源の説明に応用したものだ。また[アカシックレコード](g086.md)を重力場による情報網として論じた[wiim_020](../../docs/physics/wiim_020.md)とも概念的に接続する。
