# エキゾチック物質（Exotic Matter）

**Exotic Matter**  仮説・未観測の粒子 / 理論物理学・量子場理論 / g068
読み: えきぞちっくぶっしつ　関連: [ワームホール](g036.md), [wiim_065](../../docs/physics/wiim_065.md), [wiim_074](../../docs/physics/wiim_074.md)

負のエネルギー密度・[負の質量](g072.md)・負の圧力など、通常の物質が持たない極端な性質を持つ仮説上の物質の総称。自然界での単独存在は未確認だが、理論物理の複数の文脈で必要とされる。

**種類と主な粒子**
- **負の質量**（g072）：重力に対して反発し、力を加えると逆方向に動く。通常物質と対を作る「ランニングペア」解がある
- **負のエネルギー密度**：[カシミール効果](g009.md)（g009）で2枚の金属板間に局所的に実現する。[量子揺らぎ](g061.md)（g061）が制限されることで生じる
- **[タキオン](g069.md)**（g069）：質量の二乗が負（虚数質量）の仮説上の粒子。光速未満には減速できない
- **[ストレンジ物質](g071.md)**（g071）：アップ・ダウン・ストレンジ[クォーク](g064.md)等量の塊。通常物質より安定な可能性がある
- **幻のエネルギー（ファントムエネルギー）**：[ダークエネルギー](g218.md)の一種で状態方程式 w < −1 を持つ。宇宙をビッグリップで終わらせる可能性がある

**[反物質](g067.md)（g067）との違い**
反物質は通常物質と電荷・量子数が逆なだけで、質量は正・エネルギー密度も正だ。エキゾチック物質とは根本的に異なる。陽電子や反[陽子](g114.md)は[LHC](g074.md)（g074）で生成可能だが、エキゾチック物質の生成手段は理論的にも不明な段階が多い。

[ワームホール](g036.md)（g036）の安定維持・[アルクビエレドライブ](g035.md)の実現にはいずれも大量のエキゾチック物質が必要とされる。エネルギー条件（弱い・強い・支配的エネルギー条件）を破るため多くの物理学者は懐疑的だが、量子重力理論の完成が判断の鍵とされている。

**仮想粒子との関係**
量子場理論では仮想粒子（量子揺らぎで一時的に現れる粒子）がエキゾチック物質的な状態を局所的・瞬間的に生み出す可能性が議論されている。カシミール効果はその最も具体的な例だ。しかし物理学者ローレンス・フォードとトーマス・ローマンが導いた「量子不等式（フォード＝ローマン不等式）」は、負のエネルギーを集中させられる量と時間に厳しい上限を設けており、ワームホール維持に必要な規模には宇宙的スケールで不足するとされる。ホーキング輻射でも[事象の地平線](g006.md)付近で仮想粒子ペアが生成され、片方が負エネルギーを持ってブラックホールに吸収される描像があり、エキゾチック物質的な負エネルギーの例として引用される。また運動項の符号が逆のスカラー場（ファントム場・ゴースト場）も候補とされるが、真空が不安定になる問題を抱える。結論として「仮想粒子はエキゾチック物質を局所的には生み出せるが、量子不等式により実用規模の生成は不可能」というのが現在の主流見解だ。
