# タキオン（Tachyon）

**Tachyon**  仮説・未観測の粒子 / 理論物理学・相対性理論 / g069
読み: たきおん　関連: [ローレンツ不変性](g063.md)

**別名**: 超光速粒子

光速より速く移動するとされる仮説上の粒子。1967年にアメリカの物理学者ジェラルド・ファインバーグが命名した。通常の粒子は加速するほどエネルギーが増し光速に近づくにつれて無限大のエネルギーが必要になるため光速の壁を越えられないが、タキオンは最初から光速以上の速度でしか存在できないという対称的な存在として構想された。

[特殊相対性理論](g108.md)の枠組みでタキオンを認めると深刻な問題が生じる。[因果律](g017.md)の破綻——タキオンを使えば「原因より前に結果を送る」情報伝達が可能になり、タキオン電話の[パラドックス](g113.md)が生まれる。また[ローレンツ不変性](g063.md)（g063）を破らずにタキオンを記述すると虚数の静止質量が必要になり、物理的実体としての解釈が困難になる。

現代の量子場理論では「タキオン場」は粒子ではなく不安定な真空状態のシグナルとして再解釈されており、ヒッグス機構の記述にも登場する。SF定番素材として光速超過通信・タイムトラベルの手段として繰り返し描かれるが、現在までに実験的な証拠は得られていない。なお「仮想粒子」は[量子揺らぎ](g061.md)で一時的に現れる別の概念であり、タキオンと混同しないよう注意が必要だ。
