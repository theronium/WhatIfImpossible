# AI記事生成モード

Claude Code の `/write-article` スキルを使って、タイトルと概要だけで記事を自動生成できます。

---

## 前提

- このリポジトリを開いた状態の Claude Code で実行する
- `CLAUDE.md` がプロジェクト文脈として自動ロードされる

---

## 使い方

Claude Code のチャットで以下のように入力します：

```
/write-article タイトル: （記事タイトル）
概要: （1〜3行で何を考察するか）
```

### 例

```
/write-article タイトル: 無限のエネルギーを持つ電池が存在したら
概要: 熱力学第二法則を無視した永久機関が実現した場合、
      エネルギー保存則はどう解釈されるべきかを考察する。
```

---

## 生成されるもの

スキルは以下を自動で行います：

1. `docs/` フォルダを確認して次の通し番号（`wiim_NNN`）を決定
2. カテゴリを判断してフォルダを選択（なければ作成）
3. frontmatter 付きの記事を執筆
4. `docs/<category>/<id>.md` に保存

---

## 生成ルール（CLAUDE.md より）

| 項目 | ルール |
|------|--------|
| 言語 | 日本語 |
| 数式 | 極力使わない。使う場合は1記事1つまで |
| 図 | Mermaid で論理フローを図示する |
| トーン | 断定せず「〜と考えられる」などの表現を使う |
| 構成 | 概要 → 根拠 → 設定 → 考察 → 図解 → 関連記事 |

> ホーキング博士の言葉にならい、読みやすさを数式より優先します。

---

## スキルが使えない場合

Claude Code を**再起動**すると `.claude/commands/` が読み込まれ、スキルが有効になります。

再起動後に `/write-article` と入力するとコマンド候補に表示されます。
