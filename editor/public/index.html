Git 連携
無効時は Git パネルを非表示にし、内部ストレージのフォルダを .gitignore に自動追記します。内部コレクション（小説等）は無効推奨。
再度有効にする場合は、リポジトリルートの .gitignore から対象フォルダの行を手動で削除してください。
保存時の用語自動リンク
有効時: 保存のたびに用語名を [用語名](glossary/terms/gXXX.md) に変換。既存リンク・コードブロックは保護。小説など用語リンク不要なコレクションは無効に。
